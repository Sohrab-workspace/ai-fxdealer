# Key Shortcuts

Keyboard shortcuts (also referred to as hotkeys) are keys and their combinations that allow to execute various commands fast and without using menus or toolbars. Keyboard shortcuts can be assigned for calling any element of the [Navigator](/en/docs/mt5/client/interface) window, except for elements of the [Accounts](/en/docs/mt5/client/start_advanced/account_manage) group. In order to assign a combination of keys to an element, the "![Set hotkey](/en/docs/mt5/client/img/set_hotkey_icon.png "Set hotkey") Set hotkey" command of its context menu should be executed.

<table class="attentable" cellspacing="0" cellpadding="8" border="1"><tbody><tr class="attentable"><td class="attentable"><p class="p_tableatten"><span class="f_tableatten">Assigned hotkeys are of higher priorities than predefined ones. For example, initially the "Ctrl+O" combination calls the <a href="/en/docs/mt5/client/settings" class="topiclink">platform setup</a> window. If call of the <a href="/en/docs/mt5/client/indicators/volume_indicators/obv" class="topiclink">"On Balance Volume"</a> indicator is set to the same combination, "Ctrl+O" will not call the platform setup window any more.</span></p></td></tr></tbody></table>

Predefined keyboard shortcuts execute various functions depending on what platform interface window is currently active. Some hotkeys execute certain functions independent of focus.
