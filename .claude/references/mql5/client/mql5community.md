# MQL5.com Algo Trading community

[MQL5 Algo Community](https://www.mql5.com "MQL5 Algo Community") for algorithmic traders offers a plethora of invaluable services, from automated copy trading to the possibility of purchasing ready-made trading robots from the Market and running them 24/7 on a virtual platform.

![MQL5.community](/en/docs/mt5/client/img/mql5.png "MQL5.community")

Here you will find:

-   The Market — the platform allows [purchasing](/en/docs/mt5/client/market "Purchase the product at the Market tab") any ready-made application from the [MQL5 application store](https://www.mql5.com/en/market "Market on MQL5.community website"). Before purchasing, you can download a trial version and test it in the [strategy tester](/en/docs/mt5/client/testing).
-   MQL5 Cloud Network [is a powerful distributed computing network](https://cloud.mql5.com/en "MQL5 Distributed Cloud Network") available for testing and optimization of Expert Advisors in the [Strategy Tester](/en/docs/mt5/client/testing). Thousands of optimization sessions can now be performed in minutes. In addition to using the network, you can provide your own computing capacities and [earn](/en/docs/mt5/client/mql5cloud "Take part in MQL5 Cloud Network") profit.
-   MQL5 Storage — personal storage of source codes integrated into the MetaEditor. Keep your code safe and access it from anywhere in the world. The MQL5 Storage features will be expanded soon to allow multiple users to remotely work on one project.
-   Freelance — if you cannot find the desired application in the Code Base or Market, order one from a professional developer in the [Freelance section](https://www.mql5.com/en/job "Freelance on MQL5.community").
-   Code Base — [download](/en/docs/mt5/client/algotrading_get_ea_indicator#codebase "Download an application at the Code Base tab") any code published in the [Code Base](https://www.mql5.com/en/code "Code Base on MQL5.Community") section. The code is automatically placed in the correct directory and compiled. You only need to run the application from the [Navigator](/en/docs/mt5/client/interface) window.
-   MQL5 Charts — a special service allowing to [post screenshots from the trading platform online](/en/docs/mt5/client/mql5_charts) and share them in popular social networks.
-   Signals — subscribe to the [trading signals](/en/docs/mt5/client/signals) of professional traders and receive them straight in the platform.

## Built-in Chat feature [#](mql5community#chat)

The trading platform features the integrated [MQL5 Algo Trading](https://www.mql5.com/en/forum) chat. You may communicate with fellow traders from the community while monitoring the market and controlling your trading operations. Specify your MQL5 account in the [platform settings](/en/docs/mt5/client/settings#community). After that you may continue communication, which was started on the site, since the entire history of correspondence can be viewed in the platform.

![Built-in MQL5.community chat](/en/docs/mt5/client/img/chat.png "Built-in MQL5.community chat")

Chat for algo traders are also integrated into [mobile platforms](/en/docs/mt5/client/mobile_trading). Whatever device you choose, you can conveniently communicates with your fellow traders.
