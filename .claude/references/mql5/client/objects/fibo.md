# Fibonacci Tools

Fibonacci tools can be applied to a price or indicator chart using the "Objects — Fibonacci" items of the ["Insert"](/en/docs/mt5/client/interface) menu or the ["Line Studies"](/en/docs/mt5/client/interface) toolbar. The following types of Fibonacci tools are available in the platform:
