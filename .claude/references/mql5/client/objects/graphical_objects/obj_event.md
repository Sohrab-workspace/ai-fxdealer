# Event

This object is designed for displaying various events on a chart, like the events of the [economic calendar](/en/docs/mt5/client/fundamental). An object is anchored to the time scale and is moved together with the chart scroll. The object is shown on the time scale and is does not move vertically.

![The "Event" object](/en/docs/mt5/client/img/obj_event.png "The "Event" object")

## Controls

An object can be moved using an anchor point located on the time scale.

## Parameters

The object only has [common parameters](/en/docs/mt5/client/objects#draw_settings).
