# Elliott Tools

Elliott tools can be applied to a price or indicator chart using the "Objects — Elliott" items of the ["Insert"](/en/docs/mt5/client/interface) menu or the ["Line Studies"](/en/docs/mt5/client/interface) toolbar.

Two types of Elliott tools for displaying waves are available in the platform:
