# Terminal Settings

The window of the administrator terminal settings can be opened by pressing the "![Options](/en/docs/mt5/platform/img/options_icon.png "Options") Options" button in the ["Services"](/en/docs/mt5/platform/administrator/interface/main_menu/menu_services) menu or using the "Ctrl+O" key combination. All the settings are grouped in three tabs:

-   [Common](/en/docs/mt5/platform/administrator/settings/settings_common)  
    General settings of the administrator terminal.
-   [Support](/en/docs/mt5/platform/administrator/settings/settings_support)  
    The authorization details for contacting the technical support team.
-   [Events](/en/docs/mt5/platform/administrator/settings/settings_events)  
    Setup of notifications about different events in the terminal.
-   [Confirmations](/en/docs/mt5/platform/administrator/settings/settings_confirmations)  
    Configuring confirmation requests for dangerous administration actions.
