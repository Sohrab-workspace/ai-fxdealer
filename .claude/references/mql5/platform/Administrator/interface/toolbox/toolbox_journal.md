# Journal

The "Journal" tab contains information about the registered actions of the administrator terminal for the current session of connection to the server. The information about the terminal start and events that occur during its working is registered in the journal. Only the last messages are displayed in the window. In order to view the folder that contains all the log files it is necessary to execute the "![Open](/en/docs/mt5/platform/img/open_data_folder_button.png "Open") Open" command in the context menu.

![Journal](/en/docs/mt5/platform/img/toolbox_journal.png "Journal")

The journal entries are marked with the corresponding icons depending on their types:

-   ![Information](/en/docs/mt5/platform/img/journal_info_icon.png "Information") — information message;
-   ![Warning](/en/docs/mt5/platform/img/journal_warning_icon.png "Warning") — warning;
-   ![Error](/en/docs/mt5/platform/img/journal_error_icon.png "Error") — error message.

## Context Menu [#](toolbox_journal#context)

Context menu of the "Journal" tab contains the following commands:

-   ![Open](/en/docs/mt5/platform/img/open_data_folder_button.png "Open") Open — open the folder that contains the text files of the journal entries;
-   ![Copy](/en/docs/mt5/platform/img/copy_button.png "Copy") Copy — copy a selected entry of the journal to clipboard. You can copy the selected record using the "Ctrl+C" key combination;
-   Auto Arrange — if this option is enabled, the size of columns is selected automatically;
-   Grid — this option shows/hides grid to separate the fields of the journal entries table.
