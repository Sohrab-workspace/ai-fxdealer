# Search

This part of the toolbar allows to perform the global search of elements through the administrator terminal.

![Search Field](/en/docs/mt5/platform/img/search_field.png "Search Field")

Before starting the search one can specify sections to search in. To do it press the ![Search Settings](/en/docs/mt5/platform/img/search_configure_button.png "Search Settings") button, then the list of available sections will be opened. Specify the necessary sections by clicking on the corresponding items. Besides it, using the "Select All" and "Select None" buttons, one can  select or unselect all the sections respectively.

Once the necessary settings are made, it is necessary to type the desired word or phrase in the field to the left and press the ![Find](/en/docs/mt5/platform/img/search_button.png "Find") button. Also, using the ![Select the previous request](/en/docs/mt5/platform/img/downarrow_button.png "Select the previous request") button, one can choose one of the previous search requests.
