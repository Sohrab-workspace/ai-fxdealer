# Main Menu

Main menu is located below the title of the program. It contains different commands and functions that can be executed in the terminal. Main menu consists of:

-   [File](/en/docs/mt5/platform/administrator/interface/main_menu/menu_file)  
    Adding and deleting [servers](/en/docs/mt5/platform/administrator/getting_started/server_add_delete) to administer in the terminal, connecting and disconnecting from them.
-   [Edit](/en/docs/mt5/platform/administrator/interface/main_menu/menu_edit)  
    Commands for changing parameters in the different sections of [administration](/en/docs/mt5/platform/administration) and the [search](/en/docs/mt5/platform/administrator/interface/search) system.
-   [View](/en/docs/mt5/platform/administrator/interface/main_menu/menu_view)  
    Managing elements of the terminal ([toolbars](/en/docs/mt5/platform/administrator/interface/toolbar), [status bar](/en/docs/mt5/platform/administrator/interface/status_bar), ["Toolbox"](/en/docs/mt5/platform/administrator/interface/toolbox) window) and choosing the interface language.
-   [Services](/en/docs/mt5/platform/administrator/interface/main_menu/menu_services)  
    Different service commands.
-   [Help](/en/docs/mt5/platform/administrator/interface/main_menu/menu_help)  
    User guide, information about the program and useful links.
