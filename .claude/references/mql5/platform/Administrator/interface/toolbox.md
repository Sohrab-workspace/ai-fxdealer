# Toolbox

The "Toolbox" window is intended for working with news, mails and journal of the administrator terminal. This window can be hidden using the "![Toolbox](/en/docs/mt5/platform/img/toolbox_button.png "Toolbox") Toolbox" command of the [View](/en/docs/mt5/platform/administrator/interface/main_menu/menu_view) menu or similar command of the [toolbar](/en/docs/mt5/platform/administrator/interface/toolbar/toolbar_standard). If you want to move the Toolbox window, click on its title and, holding it, drag the window to the necessary part of the terminal.

The Toolbox window contains the following tabs:

-   [News](/en/docs/mt5/platform/administrator/interface/toolbox/toolbox_news)  
    This tab contains all the news messages that came to the terminal.
-   [Economic Calendar](/en/docs/mt5/platform/administrator/interface/toolbox/toolbox_calendar)  
    The calendar features publications of over 600 macroeconomic indicators concerning 15 largest global economies, including USA, European Union, Japan, UK, Canada, Australia, China, etc.
-   [Search](/en/docs/mt5/platform/administrator/interface/toolbox/toolbox_search)  
    This tab displays the results of the global search through the terminal performed via the [toolbar](/en/docs/mt5/platform/administrator/interface/toolbar/toolbar_search).
-   [Journal](/en/docs/mt5/platform/administrator/interface/toolbox/toolbox_journal)  
    Information about terminal launch and events during its operation is stored in the journal.
