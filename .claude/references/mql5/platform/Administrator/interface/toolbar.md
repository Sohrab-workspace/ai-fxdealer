# Toolbar

Toolbar is a part of the administrator terminal interface. It allows to organize working on the administration of the trading platform in a convenient way. The toolbar is divided into two parts:

-   [Standard](/en/docs/mt5/platform/administrator/interface/toolbar/toolbar_standard)  
    In this part of the toolbar key commands of the [main menu](/en/docs/mt5/platform/administrator/interface/main_menu) are duplicated.
-   [Search](/en/docs/mt5/platform/administrator/interface/toolbar/toolbar_search)  
    This part of the toolbar allows to perform a global search of elements through the administrator terminal.
