# Install Terminal

In order to install the administrator terminal, it is necessary to [download its distribution](https://support.metaquotes.net/en/download/mt5) and launch it.

<table class="attentable" cellspacing="0" cellpadding="8" border="1"><tbody><tr class="attentable"><td class="attentable"><ul><li class="p_tableatten"><span class="f_tableatten">The installation program is a web installer; this means that the major portion of the distributive will be downloaded from the Internet during installation.</span></li></ul><ul><li class="p_tableatten"><span class="f_tableatten">The administrator terminal installer determines type of an operating system and installs the corresponding version of the terminal.</span></li><li class="p_tableatten"><span class="f_tableatten">The terminal can operate under Microsoft Windows 10/11 or Windows Server 2008R2/2012/2012R2/2016. A processor with SSE2 support (Pentium 4/Athlon 64 or higher) is required for operating as well. Other hardware requirements are limited with software ones.</span></li></ul><ul><li class="p_tableatten"><span class="f_tableatten">The terminal can run on devices with ARM64 (for examples, on Microsoft Surface tablets), but in this case, the minimum OS version is Windows 11.</span></li></ul></td></tr></tbody></table>

![Administrator Terminal Installation](/en/docs/mt5/platform/img/setup_wizard.png "Administrator Terminal Installation")

At the first step of installation, read the program description and the copyright notice. To get acquainted with the full text of the End User License Agreement, press the "Read License Agreement" button. If you agree with the license terms without seeing, press the "Next" button.

![License Agreement](/en/docs/mt5/platform/img/setup_wizard_agreement.png "License Agreement")

If you agree with all terms of the agreement, press the "Next" button. If you do not agree, exit the terminal installation program.

![Installation Parameters](/en/docs/mt5/platform/img/setup_wizard_settings.png "Installation Parameters")

The following parameters should be selected at this installation stage:

-   Installation folder — specify a folder to install the terminal to. Another folder can be selected by writing a path to it manually or using the "Browse" button.
-   Program group — the name of the program group that will be created in the "Start" menu is indicated in this field.
-   Create a desktop shortcut — if this field is checked, a shortcut for launching the terminal will be additionally created on the desktop.
-   Launch program after successful installation — if this field is checked, the terminal will be automatically launched after the successful installation.

<table class="attentable" cellspacing="0" cellpadding="8" border="1"><tbody><tr class="attentable"><td class="attentable"><p class="p_tableatten"><span class="f_tableatten">Terminal can be installed over an installed version of it. At that, all terminal settings remain the same.</span></p></td></tr></tbody></table>

After "Next" is pressed, the terminal downloading and installation process will start.

![Installation Completion](/en/docs/mt5/platform/img/setup_wizard_finish.png "Installation Completion")

To complete the installation process, press "Done". After that you can [launch](/en/docs/mt5/platform/administrator/getting_started/launch) the terminal.
