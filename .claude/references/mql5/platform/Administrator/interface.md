# User Interface

Window of the program is divided into several parts:

-   [Main Menu](/en/docs/mt5/platform/administrator/interface/main_menu)  
    The main menu contains different commands for administering servers and setting up the administrator terminal.
-   [Toolbar](/en/docs/mt5/platform/administrator/interface/toolbar)  
    The toolbar is implemented in the administrator terminal, it is divided into two parts: ["Standard"](/en/docs/mt5/platform/administrator/interface/toolbar/toolbar_standard) and ["Search"](/en/docs/mt5/platform/administrator/interface/toolbar/toolbar_search). Commands of the main menu used for administering servers are duplicated in the standard part of the toolbar. It can be [customized](/en/docs/mt5/platform/administrator/interface/toolbar/toolbar_settings). The search part of the toolbar allows to perform a global search through the administrator terminal.
-   [Status Bar](/en/docs/mt5/platform/administrator/interface/status_bar)  
    The information line that displays the status of connection to the server and tips for different interface elements when the mouse cursor is put over them.
-   [Toolbox](/en/docs/mt5/platform/administrator/interface/toolbox)  
    "Toolbox" service window is intended for working with [news](/en/docs/mt5/platform/administrator/interface/toolbox/toolbox_news), [mail](/en/docs/mt5/platform/administration/mail), [search system](/en/docs/mt5/platform/administrator/interface/toolbox/toolbox_search) and [journal](/en/docs/mt5/platform/administrator/interface/toolbox/toolbox_journal) of the terminal.

The title of the terminal window contains the name of the server that is currently administered. The main part of the administrator terminal is intended for managing servers. The description of administering is given at the [separate section](/en/docs/mt5/platform/administration).
