# BulkSMS

To set up the provider, [create an SMS gateway configuration](/en/docs/mt5/platform/administration/integration/integration_sms) and specify the following parameters in it:

-   Provider address — address for connecting to the service. The default value is https://api.bulksms.com/v1. In most cases, there is no need to change it.
-   Token ID — token for sending messages. It can be obtained at [https://www2.bulksms.com/home/advanced\_features/api\_tokens/](https://www2.bulksms.com/home/advanced_features/api_tokens/) after you authorize using your BulkSMS account.
-   Token secret — token secret for sending messages.
-   Sender — the name of the sender which will be displayed in the message received by the recipient. The fields can be left blank or it can be filled with the [Sender ID](https://www.bulksms.com/features/sender-id) value.

The token ID and token secret will be available to you at [https://www2.bulksms.com/home/advanced\_features/api\_tokens/](https://www2.bulksms.com/home/advanced_features/api_tokens/) after you log in using your BulkSMS account.

![Receiving settings for sending messages via BulkSMS](/en/docs/mt5/platform/img/messengers_bulksms.png "Receiving settings for sending messages via BulkSMS")

Sender ID is available in your account:

![Sender ID for the "Sender" fields is available in the user account](/en/docs/mt5/platform/img/messengers_bulksms_sender.png "Sender ID for the "Sender" fields is available in the user account")

## Other Settings

All other settings are standard and no different from those of other SMS gateways. You can manage the provider's availability for [countries](/en/docs/mt5/platform/administration/integration/integration_sms#country) and [groups](/en/docs/mt5/platform/administration/integration/integration_sms#group), as well as customize [message templates](/en/docs/mt5/platform/administration/integration/integration_sms#template).
