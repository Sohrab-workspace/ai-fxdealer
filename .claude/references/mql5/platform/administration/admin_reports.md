# Reports

Reports allow analyzing clients' trading activity. Report settings are managed in the administrator terminal, because they are generated exactly on the trade server side. Reports are viewed in a manager terminal. A manager requests a report with certain parameters, the server generates it and sends back ready HTML files.

![Report](/en/docs/mt5/platform/img/reports.png "Report")

For easy access and efficiency, all reports are displayed as a tree list, in which reports are grouped by servers, for which they are configured. You can also [create subdirectories](/en/docs/mt5/platform/administration/admin_reports#name) to separate reports by purpose: "Accounts", "Daily Reports", "Gateways", etc.

## Adding and Editing Reports [#](admin_reports#add_edit)

To add or edit a report, click![Add](/en/docs/mt5/platform/img/add_button.png "Add")or![Edit](/en/docs/mt5/platform/img/edit_button.png "Edit") in the [Edit](/en/docs/mt5/platform/administrator/interface/main_menu/menu_edit) menu or in the [Standard](/en/docs/mt5/platform/administrator/interface/toolbar/toolbar_standard) toolbar.

It is possible to edit multiple configurations. Select multiple reports using the Ctrl or Ctrl+Shift keys and proceed to editing. Using this functionality, you can enable and disable reports in bulk.

### Common [#](admin_reports#common)

![Report editing](/en/docs/mt5/platform/img/reports_edit.png "Report editing")

The following parameters should be specified on this tab:

-   Enable — enable/disable the report. If the report is disabled, it will not be available in the manager terminal. Disabled reports are marked with icon ![Disabled](/en/docs/mt5/platform/img/reports_disabled_icon.png "Disabled") in the list.
-   Name — the name of the report configuration and path to it.  
       
    Specify paths to configurations to create a hierarchy of reports by arranging them into directories according to their purpose. A well-created structure can assist in setting individual [report access permissions](/en/docs/mt5/platform/administration/admin_managers#reports) for manager accounts. For example, you can create a separate directory with dealer transaction reports, a separate directory for marketers who are responsible for database analysis and customer acquisition, etc.:  
       
    Trades\\Daily Reports  
    Trades\\Deals History  
    Marketing\\First Time Deposit  
    Marketing\\Lifetime Value  
     
-   Type — all reports available on the server are displayed here. The list is compiled based on the DLL report files which are located under the /report directory of the [main trading server](/en/docs/mt5/platform/components/trade_server/trade_server_structure). The platform features a variety of built-in [standard reports](/en/docs/mt5/platform/administration/admin_reports#standard), while additional reports can be created using the MetaTrader 5 Report API.
-   Trader Server — to the right of the Type field, select a trade server the report is configured for. The report will be generated using the data on the specified server; and it will be available only when a [manager](/en/docs/mt5/platform/administration/admin_managers) connects to the specified server.

Parameters

The block of additional report parameters is available in the bottom part of the window. The additional parameters are available if they are implemented by the report developer.

The following commands are used for managing additional parameters:

-   Add — add a new parameter. A line appears upon pressing this button. Specify the parameter name and value in it. String type parameters are created by default. To select another type (integer or fractional) click the arrow on the "Add" button.
-   Edit — edit a selected parameter. The same action can be performed by a double click on the required field.
-   Delete — delete a selected parameter.

### Information [#](admin_reports#info)

![Information](/en/docs/mt5/platform/img/report_information.png "Information")

This tab displays various information on the report module: description, copyright, author, version of the report module and version of MetaTrader 5 Report API used for developing this report.

Furthermore, it displays information on available formats (a report can be generated in the form of a HTML page or in the form of a table), snapshots of databases required for the report generation. A database snapshot is a fixation of a database state on the server for generating the report (for example, the account database).

## Standard Reports [#](admin_reports#standard)

The standard delivery package of the trading platform includes several ready-made reports:

-   [Accounts Groups](/en/docs/mt5/platform/administration/admin_reports/accounts_groups_report) — statistical report on accounts in groups on a server.
-   [Accounts Grow](/en/docs/mt5/platform/administration/admin_reports/accounts_growth_report) — report on the growth of the number of accounts in groups on a server.

-   [Accounts Lifetime by Countries](/en/docs/mt5/platform/administration/admin_reports/accounts_lifetime_report) — lifetime of accounts with a distribution by countries.

-   [Agents](/en/docs/mt5/platform/administration/admin_reports/agents_report) — report on all transactions connected with agent commissions.
-   [Agents Detailed](/en/docs/mt5/platform/administration/admin_reports/agents_detailed_report) — detailed report on agent commissions.
-   [Credit Facility](/en/docs/mt5/platform/administration/admin_reports/credit_facility_report) — credit operations report for the selected period.
-   [Daily](/en/docs/mt5/platform/administration/admin_reports/daily_report) — report on the financial state of the requested accounts as of the end of the day.
-   [Daily Detailed](/en/docs/mt5/platform/administration/admin_reports/daily_detailed_report) — detailed report on the financial state of the one selected account as of the end of the day.
-   [Daily Server Logs](/en/docs/mt5/platform/administration/admin_reports/daily_server_report) — statistical report on the operation of the platform for the specified day.
-   [Daily Dealing](/en/docs/mt5/platform/administration/admin_reports/daily_dealing_report) — daily report on the activity of dealers.
-   [Daily Trades](/en/docs/mt5/platform/administration/admin_reports/daily_trade_report) — report on trade operations for the specified day.
-   [Daily Orders](/en/docs/mt5/platform/administration/admin_reports/daily_orders_report) — report on client orders open at the end of the selected day.
-   [Daily Positions](/en/docs/mt5/platform/administration/admin_reports/daily_positions_report) — report on client orders open at the end of the selected day.
-   [Daily Expert Advisors](/en/docs/mt5/platform/administration/admin_reports/daily_expert_report) — report on trading operations performed by Expert Advisors of clients.
-   [Deals History](/en/docs/mt5/platform/administration/admin_reports/deals_history_report) — summary report on the deals for the selected period.
-   [Deals Profit](/en/docs/mt5/platform/administration/admin_reports/deals_profit_report) — detailed report on deals for the selected period.
-   [Deals Initiators](/en/docs/mt5/platform/administration/admin_reports/deals_initiators_report) — deal initiators (reasons) report.
-   [Deals Geography](/en/docs/mt5/platform/administration/admin_reports/deals_geography_report) — trading activity distribution by country.
-   [Deals Weekly](/en/docs/mt5/platform/administration/admin_reports/deals_weekly_report) — trader activity report.
-   [Deposit and Withdrawal](/en/docs/mt5/platform/administration/admin_reports/deposit_withdrawal_report) — report on deposit and withdrawal operations for the selected period.
-   [Equity](/en/docs/mt5/platform/administration/admin_reports/equity_report) — report on the financial state of the requested accounts as at the end of the day. The accounts are grouped according to their "Comment" field value. This report allows to generate reports on clients' accounts grouping them by custom criteria.
-   [Margin Calls](/en/docs/mt5/platform/administration/admin_reports/margin_call_report) — report on the state of margin call or stop out accounts.
-   [Orders History](/en/docs/mt5/platform/administration/admin_reports/orders_history_report) — summary report on the orders for the selected period.
-   [Positions History](/en/docs/mt5/platform/administration/admin_reports/positions_history_report) — summary report on positions for the selected period.
-   [Segregated](/en/docs/mt5/platform/administration/admin_reports/segregated_report) — summary report on the change of financial state of the requested accounts for the specified period.
-   [Summary](/en/docs/mt5/platform/administration/admin_reports/summary_report) — summary motion of funds and funds cycles for the selected period.
-   [Gateways Turnover](/en/docs/mt5/platform/administration/admin_reports/gateways_turnover_report) — report on volumes of the deals handled by gateways.
-   [Gateways Profit](/en/docs/mt5/platform/administration/admin_reports/gateways_profit_report) — report on volumes of the deals handled by gateways, as well as the profit earned by a broker when handling these deals.
-   [Gateways White Label](/en/docs/mt5/platform/administration/admin_reports/gateways_wl_report) — report on volumes of the deals performed on a trading server by external brokers via MetaTrader 5 Gateway (standard or White Label version).
-   [Execution Type](/en/docs/mt5/platform/administration/admin_reports/execution_type_report) — report on the number, volume and type of preformed trades.
-   [Trade Accounts](/en/docs/mt5/platform/administration/admin_reports/trade_accounts_report) — report on the current trading statuses of client accounts.
-   [Trade Transactions](/en/docs/mt5/platform/administration/admin_reports/trade_transaction_report) — provides detailed data on trading operations on MetaTrader 5 servers.
-   [Trade Modifications](/en/docs/mt5/platform/administration/admin_reports/trade_modification_report) — report on trade operations manually changed by an administrator, manager or API.

-   [Trade Performance Summary](/en/docs/mt5/platform/administration/admin_reports/performance_summary_report) — detailed trading statistics of trading accounts.
-   [Trade Group Statistics](/en/docs/mt5/platform/administration/admin_reports/group_statistics_report) —  detailed statistics of groups: the number of accounts, commission amounts, share of trades by type, and more.

-   [StopOut Compensations](/en/docs/mt5/platform/administration/admin_reports/stopouts_compensation_report) — report on operations associated with the compensation of a negative balance in the situation of Stop out. All trades with the "so compensation" type are included into this report.
-   [Money Flow Daily](/en/docs/mt5/platform/administration/admin_reports/money_flow_daily_report) — a group of reports presenting the movement of funds on client accounts.
-   [Money Flow Weekly](/en/docs/mt5/platform/administration/admin_reports/money_flow_weekly_report) — a group of reports representing the weekly movement of funds on client accounts.
-   [Lifetime Value](/en/docs/mt5/platform/administration/admin_reports/lifetime_value_report) — a group of reports concerning client LTVs.
-   [First Time Deposit](/en/docs/mt5/platform/administration/admin_reports/first_time_deposit_report) — a group of reports on first-time deposits on client accounts.
-   [Retention of Clients](/en/docs/mt5/platform/administration/admin_reports/retention_of_clients_report) — new client retention report.
-   [Retention of Trading Accounts](/en/docs/mt5/platform/administration/admin_reports/retention_of_trading_accounts_report) — new client retention report by individual accounts.
-   [NFA](/en/docs/mt5/platform/administration/admin_reports/nfa_report) — a group of reports to be submitted to NFA regulator.
-   [EMIR](/en/docs/mt5/platform/administration/admin_reports/emir_report) — a report to be submitted to a regulator according to EMIR.
-   [Fund Overview](/en/docs/mt5/platform/administration/admin_reports/fund_overview_report) — a report reflecting the key characteristics of a hedge fund.
-   [Risk Appetite](/en/docs/mt5/platform/administration/admin_reports/risk_appetite_report) — a report reflecting the risk level accepted by traders.
-   [Fast Profit Deals](/en/docs/mt5/platform/administration/admin_reports/fast_profit_deals_report) — this report is designed to detect traders exploiting arbitrage opportunities through quote delays.

-   [Margin Call Statistics](/en/docs/mt5/platform/administration/admin_reports/margin_call_statistics_report) — accumulated statistics of account margin calls by country.
-   [Accounts Stop Out Statistics](/en/docs/mt5/platform/administration/admin_reports/accounts_stopout_statistics_report) — accounts in the margin call or stop-out state along with detailed information.
-   [Clients Statistics](/en/docs/mt5/platform/administration/admin_reports/clients_statistics_report) — comprehensive client statistics, including type, registration status, source of income, etc.
-   [Accounts Assets and Liabilities](/en/docs/mt5/platform/administration/admin_reports/accounts_assets_liabilities_report) — distribution of client account assets and liabilities by country.
-   [Traders Subscriptions](/en/docs/mt5/platform/administration/admin_reports/traders_subscriptions_report) — statistics on the popularity of subscriptions to broker services.
-   [Autotrading Statistics](/en/docs/mt5/platform/administration/admin_reports/autotrading_statistics_report) — provides insight into user engagement with trading robots and signals.
-   [Trading Frequency](/en/docs/mt5/platform/administration/admin_reports/trading_frequency_report) — allows you to analyze the number of deals per user over time.
-   [Clients Activity](/en/docs/mt5/platform/administration/admin_reports/clients_activity_report) — comparative statistics on active and inactive clients.
-   [A-Book & B-Book Exposure](/en/docs/mt5/platform/administration/admin_reports/abook_bbook_exposure_report) — total volume of clients' current positions routed to external liquidity providers (A-Book) and those executed internally by the broker (B-Book).
-   [Execution Speed](/en/docs/mt5/platform/administration/admin_reports/execution_speed_report) — frequency of order placement on client accounts.

The availability of these reports is determined by an administrator when setting up the platform.

## Additional reports [#](admin_reports#additional)

A wide variety of additional reports is available in the [App Store](https://support.metaquotes.net/en/market/mt5/reporting) on the technical support website. The App Store showcase is featured directly in Manager Terminals, allowing managers to view and order additional products without leaving their workspace:

![Additional reports in the App Store](/en/docs/mt5/platform/img/report_appstore.png "Additional reports in the App Store")

Furthermore, you can develop your own reports using [MetaTrader Report API](https://support.metaquotes.net/en/docs/mt5/api/reportapi). The package contains all the necessary tools, alongside the source code examples of standard reports. You can modify the source codes to create your own reports.

## Context Menu [#](admin_reports#context)

The context menu of the "Reports" section contains the following commands:

-   ![Add](/en/docs/mt5/platform/img/add_button.png "Add") Add — add a new report;
-   ![Edit](/en/docs/mt5/platform/img/edit_button.png "Edit") Edit — edit a selected report;
-   ![Delete](/en/docs/mt5/platform/img/delete_button.png "Delete") Delete — delete a selected report;
-   ![Move Up](/en/docs/mt5/platform/img/move_up_button.png "Move Up") Move Up — move a selected report up relative to others;
-   ![Move Down](/en/docs/mt5/platform/img/move_down_button.png "Move Down") Move Down — move a selected report down relative to others;
-   ![Sort Alphabetically](/en/docs/mt5/platform/img/sort_symbols_icon.png "Sort Alphabetically") Sort Alphabetically — sort configurations alphabetically. Please note that sorting is done on the server, not in the local terminal.
-   ![Enable](/en/docs/mt5/platform/img/enable_configuration_icon.png "Enable") Enable — enable the selected configuration.
-   ![Disable](/en/docs/mt5/platform/img/disable_configuration_icon.png "Disable") Disable — disable the selected configuration.
-   ![Export](/en/docs/mt5/platform/img/export_button.png "Export") Export to File — [export](/en/docs/mt5/platform/administration/common_info/import_export_config) the settings of reports to a file.
-   ![Import](/en/docs/mt5/platform/img/import_button.png "Import") Import from File — [import](/en/docs/mt5/platform/administration/common_info/import_export_config#import) the settings of reports to a file.
-   ![Journal](/en/docs/mt5/platform/img/journal_icon.png "Journal") Journal — request [logs](/en/docs/mt5/platform/administration/admin_network/network_journal) according to the selected configuration. This will open the trade server logs section, with the name of the selected configuration automatically specified in the query field. You will only need to press the request button.
-   ![Find](/en/docs/mt5/platform/img/find_button.png "Find") Find — open a [search](/en/docs/mt5/platform/administrator/interface/search) window;
-   Auto Arrange — if this option is enabled the size of columns is selected automatically;
-   Grid — this option shows/hides field separators in the table with the reports.
