# Integrations

The "Integrations" section allows configuring platform integration with external service:

-   [The Finteza business analytics system](/en/docs/mt5/platform/administration/integration/integration_finteza) which provides audience analysis and new client attraction features
-   [Sponsored VPS](/en/docs/mt5/platform/administration/integration/integration_vps) for attracting new customers and increasing trading volumes on existing accounts
-   [Mail servers](/en/docs/mt5/platform/administration/integration/integration_mail) for sending trading reports and notifications
-   [SMS Gateways](/en/docs/mt5/platform/administration/integration/integration_sms) for phone confirmation during registration and for notifications
-   [Instant Messengers](/en/docs/mt5/platform/administration/integration/integration_messenger) for phone confirmation during registration and for notifications
-   [KYC systems](/en/docs/mt5/platform/administration/integration/integration_kyc) for automated client data and document verification
-   [Web Services](/en/docs/mt5/platform/administration/integration/integration_web_services) for integration with websites and for creating trader rooms

Integrations enable automation of various processes. You can reduce manual operations and improve the quality of services provided to traders.

## Service overview

The home section contains stats and graphs revealing key data on using the services:

-   Number of user action events in the platform you have tracked via Finteza
-   Number of hosting spots you have offered to your traders
-   Number of emails and SMS messages you have sent to your clients
-   Number of documents automatically checked via the KYC provider

Here you can also see if any of the services has not been configured yet meaning you are not using all the platform features.
