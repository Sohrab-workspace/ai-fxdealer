# Margin Calculation

Margin is charged to provide collateral for traders' open positions and orders. It serves as a guarantee that the conditions of the concluded deal will be fulfilled. The trading platform provides different risk management models, which define the type of pre-trade control. The model can be defined separately for each client group on the ["Margin" tab](/en/docs/mt5/platform/administration/admin_groups/admin_groups_symbols/group_symbols_margin) in the "Risk Management" field.

![Margin](/en/docs/mt5/platform/img/groups_margin.png "Margin")

The following models are currently available:

-   [for Retail Forex, CFD, Futures](/en/docs/mt5/platform/administration/admin_symbols/admin_symbols_settings/symbol_settings_trade/margin_calculation/margin_forex) — used for the OTC market. Margin calculation is based on the type of instrument, as well as group settings. [Netting position accounting system](/en/docs/mt5/platform/administration/admin_groups/group_position#netting) is used.
-   [for Retail Forex, CFD, Futures with hedging](/en/docs/mt5/platform/administration/admin_symbols/admin_symbols_settings/symbol_settings_trade/margin_calculation/margin_forex_hedge) — used for the OTC market. Margin calculation is based on the type of instrument, as well as group settings. [Hedging position accounting system](/en/docs/mt5/platform/administration/admin_groups/group_position#hedging) is used.
-   [for Stock Exchange, based on margin discount rates](/en/docs/mt5/platform/administration/admin_symbols/admin_symbols_settings/symbol_settings_trade/margin_calculation/margin_exchange) — used for the exchange market. Margin calculation is based on the discounts specified in [symbol settings](/en/docs/mt5/platform/administration/admin_symbols/admin_symbols_settings/symbol_settings_margin). Discounts are set by the broker, however they cannot be lower than the exchange set values.
