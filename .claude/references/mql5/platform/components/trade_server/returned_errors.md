# Return Errors

When an attempt is made to perform invalid operations, a trade server returns errors to terminals. They are displayed on the ["Journal"](/en/docs/mt5/platform/administrator/interface/toolbox/toolbox_journal) tab of the "Toolbox" window, as well as in [journals of servers](/en/docs/mt5/platform/administration/admin_network/network_journal).
