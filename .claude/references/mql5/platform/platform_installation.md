# Platform Installation

The choice and preparation of hardware, as well as the correct installation of the MetaTrader 5 trading platform are very important steps, on which the reliability of the platform operation and the quality of your customer service will depend. This section of the user manual provides information on trading platform installation and preparation:

-   [System Requirements](/en/docs/mt5/platform/platform_installation/requirements) — recommendations on what hardware to use for the platform servers.
-   [System Preparations](/en/docs/mt5/platform/platform_installation/installation_preparations) — recommendations on how to prepare the operating system before platform installation.
-   [Installation](/en/docs/mt5/platform/platform_installation/installation) — initial installation and recommendations on further platform extension.
-   [Activation](/en/docs/mt5/platform/platform_installation/activation) — before you start using the platform, it needs to be activated in order to remove all functional limitations.
-   [White Label](/en/docs/mt5/platform/platform_installation/white_label) — how to order a White Label, and what data is needed for that.
-   [Fast Deployment](/en/docs/mt5/platform/platform_installation/deployer) — instructions on fast installation of additional platform components.
-   [Console Setup](/en/docs/mt5/platform/platform_installation/console_setup) — instructions on platform configuration using the command line.
-   [Platform Migration](/en/docs/mt5/platform/platform_installation/platform_move) — how to move the platform to new hardware.
-   [Platform Uninstall](/en/docs/mt5/platform/platform_installation/platform_uninstall) — if necessary, the trading platform can be uninstalled from the system.
