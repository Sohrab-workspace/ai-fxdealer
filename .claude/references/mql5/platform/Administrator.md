# MetaTrader 5 Administrator — User Guide

MetaTrader 5 Administrator is a software program for the remote administration of the MetaTrader 5 trading platform. It enables control over servers and other components of the platform, management of user [group](/en/docs/mt5/platform/administration/admin_groups) settings, [manager](/en/docs/mt5/platform/administration/admin_managers) configurations, [financial symbol](/en/docs/mt5/platform/administration/admin_symbols) and [data feed](/en/docs/mt5/platform/administration/admin_feeds) parameters. Also, the Administrator Terminal allows managing databases of [quotes](/en/docs/mt5/platform/administration/admin_charts) and [tick data](/en/docs/mt5/platform/administration/admin_ticks), configuring [trading schedule](/en/docs/mt5/platform/administration/admin_time) and adjusting [backup](/en/docs/mt5/platform/administration/admin_network/network_add_edit/network_backup_server) parameters.

The trading platform is provided with enhanced configuration options. All platform settings are grouped into categories, which are displayed as a tree in the left part of the Administrator Terminal:
