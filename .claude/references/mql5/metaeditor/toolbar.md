# Toolbars

MetaEditor features toolbars to make your work more convenient and place the necessary commands and data at your fingertips:

-   [Standard](/en/docs/mt5/metaeditor/toolbar_standard) — add the most frequently used commands from the [main menu](/en/docs/mt5/metaeditor/main_menu) here to execute them in a single click.
-   [Search](/en/docs/mt5/metaeditor/toolbar_search) — look for necessary data in a program text, files and MQL5.community website directly in MetaEditor.
