# Toolbox

The "Toolbox" service window is intended for viewing results of different actions performed in the MetaEditor. It can be shown or hidden using the "![Toolbox](/en/docs/mt5/metaeditor/img/toolbox_icon.png "Toolbox") Toolbox" command in the ["View"](/en/docs/mt5/metaeditor/interface/main_menu/main_menu_view#toolbox) menu, the ["Standard"](/en/docs/mt5/metaeditor/interface/toolbar/toolbar_standard#toolbox) toolbar or by pressing the "Ctrl+T" key combination. It contains the following tabs:

-   [Errors](/en/docs/mt5/metaeditor/interface/toolbox/toolbox_errors)  
    In this tab the [compilation](/en/docs/mt5/metaeditor/development/compile) errors and the line and column numbers where they appeared are displayed in the form of a list.
-   [Search](/en/docs/mt5/metaeditor/interface/toolbox/toolbox_find)  
    Results of searching through [files](/en/docs/mt5/metaeditor/development/source_code/source_code_find#find_in_files), [articles and "Code Base"](/en/docs/mt5/metaeditor/interface/toolbar/toolbar_search) are shown at this tab.
-   [Profiler](/en/docs/mt5/metaeditor/interface/toolbox/toolbox_profiler)  
    This tab displays the source code [profiling](/en/docs/mt5/metaeditor/development/profiling) results.
-   [Debug](/en/docs/mt5/metaeditor/interface/toolbox/toolbox_debug)  
    The "Debug" tab displays the program [debugging](/en/docs/mt5/metaeditor/development/debug) results. This tab appears only if the debugging process is started.
-   [Articles](/en/docs/mt5/metaeditor/interface/toolbox/toolbox_articles)  
    This tab displays the list of articles published at [MQL4.community](https://www.mql4.com "MQL4.community") or [MQL5.community](https://www.mql5.com "MQL5.Community").
-   [Code Base](/en/docs/mt5/metaeditor/interface/toolbox/toolbox_codebase)  
    This tab displays the list of MQL4/MQL5 programs published in the "Code Base" section of the [MQL4.community](https://www.mql4.com "MQL4.community") or [MQL5.community](https://www.mql5.com "MQL5.community") website.
-   [Journal](/en/docs/mt5/metaeditor/interface/toolbox/toolbox_journal)  
    The journal contains messages about actions performed in MetaEditor. For example, messages about working with [MQL5 Storage](/en/docs/mt5/metaeditor/mql5storage).
