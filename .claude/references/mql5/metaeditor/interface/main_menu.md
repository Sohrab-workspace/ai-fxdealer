# Main Menu

The MetaEditor main menu is located below the program header and is a set of lower-level menus. Executable commands and functions are gathered here. The main menu consists of the following components:

-   [File](/en/docs/mt5/metaeditor/interface/main_menu/main_menu_file)  
    Working with files: creation, opening, compilation and printing of MQL4/MQL5 programs.
-   [Edit](/en/docs/mt5/metaeditor/interface/main_menu/main_menu_edit)  
    Working with the source code of a program: copying, pasting, search, replacement, use of bookmarks, etc.
-   [View](/en/docs/mt5/metaeditor/interface/main_menu/main_menu_view)  
    Setup of [toolbars](/en/docs/mt5/metaeditor/interface/toolbar), management of windows (["Navigator"](/en/docs/mt5/metaeditor/interface/navigator), ["Tools"](/en/docs/mt5/metaeditor/interface/toolbox)) and the language of the program interface.
-   [Debug](/en/docs/mt5/metaeditor/interface/main_menu/main_menu_debug)  
    Debugging of programs.
-   [Tools](/en/docs/mt5/metaeditor/interface/main_menu/main_menu_tools)  
    Switching to the client terminal, code stylization and [MetaEditor setup](/en/docs/mt5/metaeditor/settings).
-   [Window](/en/docs/mt5/metaeditor/interface/main_menu/main_menu_window)  
    List of open windows of editable files and their location.
-   [Help](/en/docs/mt5/metaeditor/interface/main_menu/main_menu_help)  
    Help files and information about the program.
