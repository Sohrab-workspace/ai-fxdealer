# Managing Windows

Windows of program files edited are located at the working area of the MetaEditor. On default they are shown as different tabs named in compliance with file names.

![Tabs](/en/docs/mt5/metaeditor/img/tabs.png "Tabs")

In order to switch to working with one of the files you should choose the corresponding tab using the mouse. The context menu of the tabs allows to execute the following commands:

Positions of the file windows can be changed using the commands of the ["Window"](/en/docs/mt5/metaeditor/interface/main_menu/main_menu_window) or the ["Standard"](/en/docs/mt5/metaeditor/interface/toolbar/toolbar_standard#cascade) toolbar. A tab can also be minimized to a window by a double-click on its title. In order to maximize the tab you should double-click with the mouse on its title again.

The commands of the ["Edit"](/en/docs/mt5/metaeditor/interface/main_menu/main_menu_edit) menu and the ["Standard"](/en/docs/mt5/metaeditor/interface/toolbar/toolbar_standard) toolbar are repeated in the context menu of the window for working with source code:
