# Main menu

MetaEditor main menu is located under the program header. It features almost all commands and functions that can be executed in MetaEditor.

-   [File](/en/docs/mt5/metaeditor/main_menu_file) — working  with files: creating new projects and opening existing ones, compiling programs and moving to the data directory.
-   [Edit](/en/docs/mt5/metaeditor/main_menu_edit) — working with a program source code: copy, paste, use bookmarks, etc.
-   [Search](/en/docs/mt5/metaeditor/main_menu_search) — search for a text in one or several files.
-   [View](/en/docs/mt5/metaeditor/main_menu_view) — configuring [toolbars](/en/docs/mt5/metaeditor/toolbar), managing service windows ([Navigator](/en/docs/mt5/metaeditor/navigator), [Toolbox](/en/docs/mt5/metaeditor/toolbox)) and a program interface language.
-   [Debug](/en/docs/mt5/metaeditor/main_menu_debug) — searching for errors in programs and profiling.
-   [Build](/en/docs/mt5/metaeditor/main_menu_build) — compiling the source code into an executable file.
-   [Tools](/en/docs/mt5/metaeditor/main_menu_tools) — switching to the trading platform, code stylizing and [MetaEditor options](/en/docs/mt5/metaeditor/settings).
-   [Window](/en/docs/mt5/metaeditor/main_menu_window) — list of opened files and their location.
-   [Help](/en/docs/mt5/metaeditor/main_menu_help) — user guide and other reference materials.
