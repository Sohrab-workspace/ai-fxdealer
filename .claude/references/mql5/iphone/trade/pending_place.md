<table class="container" cellspacing="0" cellpadding="0" border="0"><tbody><tr class="container"><td class="container" style="width:50%;"><h1>Placing of Pending Orders</h1><p class="p_txt"><span class="f_txt">To place a pending order, execute one of the following actions:</span></p><ul><li class="p_ol"><span class="f_ol">Select the required symbol on the <a href="/en/docs/mt5/iphone/quotes" class="topiclink">Quotes</a> screen and execute the 'Trade' command of the context menu.</span></li><li class="p_ol"><span class="f_ol">If there are no open positions or orders, select 'Open position' on the <a href="/en/docs/mt5/iphone/trade" class="topiclink">Trade</a> tab.</span></li><li class="p_ol"><span class="f_ol">If there are open positions or orders, use commands of the context menu commands on the <a href="/en/docs/mt5/iphone/trade#context" class="topiclink">Trade</a> tab.</span></li></ul><p class="p_txt"><span class="f_txt">The following parameters should be specified here:</span></p><ul><li class="p_ol"><span class="f_ol" style="font-weight: bold;">Symbol</span><span class="f_ol"> — the symbol for which you place a pending order.</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;">Trade operation type</span><span class="f_ol"> — select the <a href="/en/docs/mt5/iphone/trade/general_concept/order_types#pending_order" class="topiclink">type of a pending order</a>. In this field you can also switch to placing a <a href="/en/docs/mt5/iphone/trade/open_positions" class="topiclink">market order</a>.</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;">Volume</span><span class="f_ol"> — volume of an order in lots. To change the volume, use the arrow buttons or tap on the box to indicate the volume using the keyboard.</span></li></ul></td><td class="container" style="width:50%;"><p class="p_Image"><img class="help" alt="Pending order" title="Pending order" width="324" height="661" src="/en/docs/mt5/iphone/img/pending_place.png"></p></td></tr><tr class="container"><td class="container" colspan="2" style="width:50%;"><ul><li class="p_ol"><a name="stop_price" class="hmanchor"></a><span class="f_ol" style="font-weight: bold;">Price</span><span class="f_ol"> — price at which the pending order should trigger. For stop and limit orders this is the price, at which they will be placed. For Stop Limit orders, this is the price of triggering and setting of limit orders at the level specified in 'Stop Limit Price'.</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;">Stop Limit Price</span><span class="f_ol"> — this field is active only for Stop Limit orders. When the Stop Limit order triggers, a limit order will be placed at the price specified in it.</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;">Stop Loss</span><span class="f_ol"> — the level of <a href="/en/docs/mt5/iphone/trade/general_concept/order_types#stop_loss" class="topiclink">Stop Loss</a>. If you leave this field empty, this type of order will not be set.</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;">Take Profit</span><span class="f_ol"> - Level <a href="/en/docs/mt5/iphone/trade/general_concept/order_types#take_profit" class="topiclink">Take Profit</a>. If you leave this field empty, this type of order will not be set.</span></li><li class="p_ol"><a name="fill_policy" class="hmanchor"></a><span class="f_ol" style="font-weight: bold;">Fill Policy</span><span class="f_ol"> — additional <a href="/en/docs/mt5/iphone/trade/general_concept/fill_policy" class="topiclink">rules of filling</a> for <a href="/en/docs/mt5/iphone/trade/general_concept/order_types" class="topiclink">limit and stop-limit orders</a>: Fill or Kill, Immediate or Cancel, Book or Cancel, or Return. If this box is unavailable, then the option is locked on the server;</span></li><li class="p_ol"><a name="expiration" class="hmanchor"></a><span class="f_ol" style="font-weight: bold;">Expiration</span><span class="f_ol"> — conditions of the order expiration:</span></li></ul><ul><li class="p_ol_2lev"><span class="f_ol_2lev" style="font-weight: bold;">Good Till Canceled (GTC)</span><span class="f_ol_2lev"> </span><span class="f_ol">— the order will stay in the queue until it is manually canceled.</span></li><li class="p_ol_2lev"><span class="f_ol_2lev" style="font-weight: bold;">Today</span><span class="f_ol_2lev"> </span><span class="f_ol">— the order will be valid only during the current trading day.</span></li><li class="p_ol_2lev"><span class="f_ol_2lev" style="font-weight: bold;">Date and Time</span><span class="f_ol_2lev"> </span><span class="f_ol">— the order will be valid until the specified date. When selecting this option, you will automatically switch to the date and time specification. To return to choosing the expiration type, remove the expiration date.</span></li><li class="p_ol_2lev"><span class="f_ol" style="font-weight: bold;">Date</span><span class="f_ol"> — the order will be valid until 00:00 of the specified day. If this time is not within a trading session, expiration will occur in the nearest trading time.</span></li></ul><div><div class="parent-table"><table class="attentable" cellspacing="0" cellpadding="8" border="1"><tbody><tr class="attentable"><td class="attentable" style="vertical-align:middle;"><ul><li class="p_tableatten"><span class="f_tableatten">The 'Place' button is inactive if any of the parameters is incorrect.</span></li><li class="p_tableatten"><span class="f_tableatten">Stop Loss and Take Profit orders only trigger for open positions, not for pending orders.</span></li><li class="p_tableatten"><span class="f_tableatten">If the 'Execution' and 'Expiration' fields are inactive, it means the possibility to change them is disabled on the server.</span></li></ul></td></tr></tbody></table></div></div></td></tr><tr class="container"><td class="container" style="width:25%;"><p class="p_Image"><img class="help" alt="Advanced mode for placing a pending order" title="Advanced mode for placing a pending order" width="324" height="661" src="/en/docs/mt5/iphone/img/pending_order_extended.png"></p></td><td class="container" style="width:25%;"><h2><a name="extended" class="hmanchor"></a>Advanced Mode <a class="anchor" href="pending_place#extended">#</a></h2><p class="p_txt"><span class="f_txt">This mode is used when placing pending orders directly from the <a href="/en/docs/mt5/iphone/chart#pending_chart" class="topiclink">chart</a>. It provides additional tools for calculating volume and risk.</span></p><h3>Working with the Dialog</h3><p class="p_txt"><span class="f_txt">The trading dialog can be minimized, expanded, or stretched to full screen by dragging it up or down, or by using the buttons<img class="help" alt="Expand" title="Expand" width="20" height="6" src="/en/docs/mt5/iphone/img/trade_dialog_maximize.png"> / <img class="help" alt="Resize" title="Resize" width="20" height="2" src="/en/docs/mt5/iphone/img/trade_dialog_resize.png">in the header. Tapping once on an empty area of the chart minimizes the dialog, and tapping again hides it completely.</span></p><h3>Instrument and Prices</h3><p class="p_txt"><span class="f_txt">The top section displays the selected trading instrument and its current prices. To switch instruments, click the current instrument name.</span></p><h3>Order Type</h3><p class="p_txt"><span class="f_txt">Use the menu to switch between <a href="/en/docs/mt5/iphone/trade/general_concept/order_types#pending_order" class="topiclink">pending order types</a>. In this field, you can also switch to placing a <a href="/en/docs/mt5/iphone/trade/open_positions" class="topiclink">market order</a>.</span></p><h3>Volume</h3><p class="p_txt"><span class="f_txt">You can specify order volume in two modes:</span></p><ul><li class="p_ol"><span class="f_ol">Lots — one lot represents a specific number of units of the base asset, defined by the 'Contract size' parameter in the instrument <a href="/en/docs/mt5/iphone/quotes#specification" class="topiclink">specification</a>. For example, if the contract size is 100,000 units, a volume of 0.01 lots equals 1,000 units of the asset.</span></li><li class="p_ol"><span class="f_ol">Margin — in this mode, volume is specified as the approximate amount of margin that will be reserved in your account if the pending order is triggered and executed. In fact, this is the amount of margin for a future market position.</span></li></ul></td></tr><tr class="container"><td class="container" colspan="2" style="width:25%;"><p class="p_txt"><span class="f_txt">For each mode, the application displays the corresponding value in the alternative mode. For example, if you set the volume in lots, the approximate required margin is shown immediately.</span></p><p class="p_txt"><span class="f_txt">Volume can be specified in several ways:</span></p><ul><li class="p_ol"><span class="f_ol">Using the slider below the field. Minimum and maximum allowable values are calculated automatically.</span></li><li class="p_ol"><span class="f_ol">Manually via keyboard — tap the current value and enter a new one.</span></li><li class="p_ol"><span class="f_ol">Using the adjustment panel opened by the arrow button. Suggested increment steps adapt dynamically depending on the current value.</span></li></ul><div><div class="parent-table"><table class="attentable" cellspacing="0" cellpadding="8" border="1"><tbody><tr class="attentable"><td class="attentable"><ul><li class="p_tableatten"><span class="f_tableatten">Volume changes are synchronized with the <a href="/en/docs/mt5/iphone/chart#one_click" class="topiclink">quick trading panel</a>. If you change the volume in the trading dialog, it updates in the panel and vice versa.</span></li><li class="p_tableatten"><span class="f_tableatten">The last used volume is saved after order submission. When creating a new order, the saved values are automatically loaded into the dialog. Only the most recent value is stored; volume values are not saved separately for each symbol.</span></li></ul></td></tr></tbody></table></div></div><h3>Price</h3><p class="p_txt"><span class="f_txt">Set the trigger price for the pending order. For stop and limit orders, this is the price at which the orders will be placed. For Stop Limit orders, this is the price at which limit orders will be triggered and placed at the level specified in 'Stop Limit Price'.</span></p><h3>Take Profit and Stop Loss</h3><p class="p_txt"><span class="f_txt"><a href="/en/docs/mt5/iphone/trade/general_concept/order_types#take_profit" class="topiclink">Take Profit</a> and <a href="/en/docs/mt5/iphone/trade/general_concept/order_types#stop_loss" class="topiclink">Stop Loss</a> levels can be set in several modes:</span></p><ul><li class="p_ol"><span class="f_ol">Price — the instrument price at which the order should execute.</span></li><li class="p_ol"><span class="f_ol">Pips — the distance from the order price in pips.</span></li><li class="p_ol"><span class="f_ol">Currency — an approximate monetary amount in the deposit currency that you will gain or lose when the level is triggered. When entering a desired value, the actual calculated amount is displayed below. This may not exactly match the entered value due to price increments.</span></li></ul><p class="p_txt"><span class="f_txt">For each mode, the application displays the corresponding value in the alternative mode. For example, if you set levels in pips, you will immediately see the corresponding price and the approximate amount you will gain or lose if the level is triggered (if nonzero).</span></p><p class="p_txt"><a name="lock_sl_tp" class="hmanchor"></a><span class="f_txt">When setting levels in pips, you can lock the Take Profit to Stop Loss ratio. To do this, set both levels, then tap <img class="help" alt="Fix the Take Profit and Stop Loss ratio" title="Fix the Take Profit and Stop Loss ratio" width="14" height="20" src="/en/docs/mt5/iphone/img/lock_sl_tp_button.png">. Now, adjusting one level will automatically update the other according to the specified ratio, including when moving levels on the <a href="/en/docs/mt5/iphone/chart#manage_orders" class="topiclink">chart</a>. This setting is retained for future orders until it is changed or disabled.</span></p><p class="p_Image"><img class="help" alt="Take Profit and Stop Loss levels change proportionally" title="Take Profit and Stop Loss levels change proportionally" width="660" height="661" src="/en/docs/mt5/iphone/img/lock_sl_tp.png"></p><p class="p_txt"><span class="f_txt">Levels can be set in multiple ways:</span></p><ul><li class="p_ol"><span class="f_ol">Directly on the chart — for this, you should temporarily minimize the trading panel to adjust levels. Moving levels near chart edges will <a href="/en/docs/mt5/iphone/chart#scale_levels" class="topiclink">automatically rescale</a> the chart, providing access to a wider price range.</span></li><li class="p_ol"><span class="f_ol">If stop levels are set in pips or currency, they will automatically move by the same amount when the order price changes. This also respects the <a href="/en/docs/mt5/iphone/trade/pending_place#lock_sl_tp" class="topiclink">locked ratio of levels</a> if enabled.</span></li><li class="p_ol"><span class="f_ol">Manually via keyboard — tap the current value and enter a new one.</span></li><li class="p_ol"><span class="f_ol">Using the adjustment panel that opens when you tap on the field with a checkmark. Suggested increment steps adapt dynamically depending on the current value.</span></li></ul><p class="p_txt"><span class="f_txt">To remove a level, either clear the value in the corresponding field or tap on the checkmark and then the icon <img class="help" alt="delete_all_icon" width="16" height="20" src="/en/docs/mt5/iphone/img/delete_all_icon.png"> in the panel that appears.</span></p><div><div class="parent-table"><table class="attentable" cellspacing="0" cellpadding="8" border="1"><tbody><tr class="attentable"><td class="attentable"><ul><li class="p_tableatten"><span class="f_tableatten">The entered Take Profit and Stop Loss values are saved after submitting an order. When placing a new order, previously used values will be automatically loaded into the dialog. Only the most recent value for each level is stored; values are not saved separately for each symbol. Saved values are stored in pips and converted to prices when applied, ensuring levels remain accurate even if the instrument price changes.</span></li><li class="p_tableatten"><span class="f_tableatten">When switching instruments, previously saved levels may be too far from the current price and may not appear on the chart. In this case, adjust the levels manually or remove and re-add them. </span><span class="f_ol">By default, levels are placed within visible chart range.</span></li></ul></td></tr></tbody></table></div></div><h3>Additional Parameters</h3><p class="p_txt"><span class="f_txt">At the bottom of the dialog, you can specify additional parameters:</span></p><ul><li class="p_ol"><span class="f_ol">Order expiration time.</span></li><li class="p_ol"><span class="f_ol">Order execution mode, if supported by the broker.</span></li><li class="p_ol"><span class="f_ol">A text comment to the order. It will be displayed in the <a href="/en/docs/mt5/iphone/trade" class="topiclink">trade</a> section. The maximum comment length is 31 symbols. The broker or server may modify the comment.</span></li></ul><h3>Tick Chart and Order Placement</h3><p class="p_txt"><span class="f_txt">For additional price control when placing an order, a tick chart of the instrument is displayed at the bottom of the dialog.</span></p><p class="p_txt"><span class="f_txt">After setting all the required parameters, place the order using the corresponding button. If any value in the dialog is incorrect, the corresponding field is highlighted in red. Also, if the price or stop levels are incorrect, the order placement button becomes inactive.</span></p></td></tr></tbody></table>
