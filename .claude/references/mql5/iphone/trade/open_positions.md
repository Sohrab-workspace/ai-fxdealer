[MetaTrader 5](/en/docs/mt5)[iPhone/iPad](/en/docs/mt5/iphone)[Trade](/en/docs/mt5/iphone/trade)Opening and Closing Positions

<table class="container" cellspacing="0" cellpadding="0" border="0"><tbody><tr class="container"><td class="container" style="width:50%;"><h1>Opening and Closing Positions</h1><p class="p_txt"><span class="f_txt">Opening a <a href="/en/docs/mt5/iphone/trade/general_concept#position" class="topiclink">position</a> (entering the market) is the initial purchase or sale of a specified volume of a financial instrument. To do this, you must place a <a href="/en/docs/mt5/iphone/trade/general_concept/order_types#market_order" class="topiclink">market order</a>, which results in the execution of a <a href="/en/docs/mt5/iphone/trade/general_concept#deal" class="topiclink">deal</a>. A position may also be opened when a <a href="/en/docs/mt5/iphone/trade/general_concept/order_types#pending_order" class="topiclink">pending order</a> is triggered.</span></p><p class="p_txt"><span class="f_txt">An order can be placed in several ways:</span></p><ul><li class="p_ol"><span class="f_ol">In the <a href="/en/docs/mt5/iphone/quotes" class="topiclink">Quotes</a> window, select the desired instrument and choose Trade from the context menu.</span></li><li class="p_ol"><span class="f_ol">Use the <a href="/en/docs/mt5/iphone/chart#one_click" class="topiclink">One-Click Trading panel</a> or the advanced trading dialog on the chart.</span></li><li class="p_ol"><span class="f_ol">If there are no open positions or orders, select 'Open position' in the <a href="/en/docs/mt5/iphone/trade" class="topiclink">Trade</a> tab.</span></li><li class="p_ol"><span class="f_ol">If there are open positions or orders, use the context menu commands in the <a href="/en/docs/mt5/iphone/trade#context" class="topiclink">Trade</a> tab.</span></li></ul><h2>Placing an Order and General Parameters</h2><p class="p_txt"><span class="f_txt">After opening the new order dialog, specify the following parameters:</span></p></td><td class="container" style="width:50%;"><p class="p_Image"><img class="help" alt="Placing an Order" title="Placing an Order" width="324" height="661" src="/en/docs/mt5/iphone/img/open_market.png"></p></td></tr><tr class="container"><td class="container" colspan="2" style="width:50%;"><ul><li class="p_ol"><span class="f_ol" style="font-weight: bold;">Symbol</span><span class="f_ol"> — the symbol for which you want to perform a trading operation.</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;">Order type</span><span class="f_ol"> — if one of the execution modes is selected in this field, a market operation will be performed for the selected instrument. Execution modes are described below. You can also switch to placing a <a href="/en/docs/mt5/iphone/trade/pending_place" class="topiclink">pending order</a> here.</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;">Volume</span><span class="f_ol"> — the trade size in lots. Adjust the volume using the adaptive buttons on either side of the field or click the field to enter a value via the keyboard.</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;">Stop Loss</span><span class="f_ol"> — the <a href="/en/docs/mt5/iphone/trade/general_concept/order_types#stop_loss" class="topiclink">Stop Loss</a> level in prices. The level is set to limit the potential losses on the position. If set to zero, this level will not be attached.</span></li></ul><ul><li class="p_ol"><span class="f_ol" style="font-weight: bold;">Take Profit</span><span class="f_ol"> — the <a href="/en/docs/mt5/iphone/trade/general_concept/order_types#take_profit" class="topiclink">Take Profit</a> level in prices. The level is set to lock in profit on the position. If set to zero, this level will not be attached.</span></li></ul><p class="p_txt"><span class="f_txt">Depending on the instrument's <a href="/en/docs/mt5/iphone/trade/general_concept/execution_types" class="topiclink">execution type</a>, additional parameters may be available in the order placing window. Execution types are described below.</span></p><p class="p_txt"><span class="f_txt">Tap Buy to place a buy order or Sell to place a sell order.</span></p><p class="p_txt"><span class="f_txt">Once an order is sent, its execution results will appear in the window: either confirmation of a successful trade or a rejection with the reason specified.</span></p><p class="p_txt"><span class="f_txt">Once the order is executed, the relevant open position record will appear on the <a href="/en/docs/mt5/iphone/trade" class="topiclink">Trade</a> tab, as well as the order and the resulting deal(s) will appear in the <a href="/en/docs/mt5/iphone/history" class="topiclink">History</a> tab.</span></p></td></tr><tr class="container"><td class="container" style="width:25%;"><p class="p_Image"><img class="help" alt="Advanced position opening mode" title="Advanced position opening mode" width="324" height="661" src="/en/docs/mt5/iphone/img/open_market_extended.png"></p></td><td class="container" style="width:25%;"><h2><a name="extended" class="hmanchor"></a>Advanced Mode <a class="anchor" href="open_positions#extended">#</a></h2><p class="p_txt"><span class="f_txt">This mode is used when opening positions from the <a href="/en/docs/mt5/iphone/chart#manage_positions" class="topiclink">chart</a>. It provides additional tools for calculating volume and risk.</span></p><h3>Working with the Dialog</h3><p class="p_txt"><span class="f_txt">The trading dialog can be minimized, expanded, or stretched to full screen by dragging it up or down, or by using the buttons<img class="help" alt="Expand" title="Expand" width="20" height="6" src="/en/docs/mt5/iphone/img/trade_dialog_maximize.png"> / <img class="help" alt="Resize" title="Resize" width="20" height="2" src="/en/docs/mt5/iphone/img/trade_dialog_resize.png">in the header. Tapping once on an empty area of the chart minimizes the dialog, and tapping again hides it completely.</span></p><h3>Instrument and Prices</h3><p class="p_txt"><span class="f_txt">The top section displays the selected trading instrument and its current prices. To switch instruments, click the current instrument name.</span></p><h3>Order Type</h3><p class="p_txt"><span class="f_txt">Use the menu to switch between placing a market order and <a href="/en/docs/mt5/iphone/trade/pending_place" class="topiclink">creating a pending order</a>.</span></p><h3>Volume</h3><p class="p_txt"><span class="f_txt">You can specify order volume in two modes:</span></p><ul><li class="p_ol"><span class="f_ol">Lots — one lot represents a specific number of units of the base asset, defined by the 'Contract size' parameter in the instrument <a href="/en/docs/mt5/iphone/quotes#specification" class="topiclink">specification</a>. For example, if the contract size is 100,000 units, a volume of 0.01 lots equals 1,000 units of the asset.</span></li><li class="p_ol"><span class="f_ol">Margin — in this mode, volume is specified as the approximate amount of margin that will be reserved in your account to open the position.</span></li></ul></td></tr><tr class="container"><td class="container" colspan="2" style="width:25%;"><p class="p_txt"><span class="f_txt">For each mode, the application displays the corresponding value in the alternative mode. For example, if you set the volume in lots, the approximate required margin is shown immediately.</span></p><p class="p_txt"><span class="f_txt">Volume can be specified in several ways:</span></p><ul><li class="p_ol"><span class="f_ol">Using the slider below the field. Minimum and maximum allowable values are calculated automatically.</span></li><li class="p_ol"><span class="f_ol">Manually via keyboard — tap the current value and enter a new one.</span></li><li class="p_ol"><span class="f_ol">Using the adjustment panel opened by the arrow button. Suggested increment steps adapt dynamically depending on the current value.</span></li></ul><div><div class="parent-table"><table class="attentable" cellspacing="0" cellpadding="8" border="1"><tbody><tr class="attentable"><td class="attentable"><ul><li class="p_tableatten"><span class="f_tableatten">Volume changes are synchronized with the <a href="/en/docs/mt5/iphone/chart#one_click" class="topiclink">quick trading panel</a>. If you change the volume in the trading dialog, it updates in the panel and vice versa.</span></li><li class="p_tableatten"><span class="f_tableatten">The last used volume is saved after order submission. When creating a new order, the saved values are automatically loaded into the dialog. Only the most recent value is stored; volume values are not saved separately for each symbol.</span></li></ul></td></tr></tbody></table></div></div><h3>Take Profit and Stop Loss</h3><p class="p_txt"><span class="f_txt"><a href="/en/docs/mt5/iphone/trade/general_concept/order_types#take_profit" class="topiclink">Take Profit</a> and <a href="/en/docs/mt5/iphone/trade/general_concept/order_types#stop_loss" class="topiclink">Stop Loss</a> levels can be set in several modes:</span></p><ul><li class="p_ol"><span class="f_ol">Price — the instrument price at which the order should execute.</span></li><li class="p_ol"><span class="f_ol">Pips — the distance from the order price in pips.</span></li><li class="p_ol"><span class="f_ol">Currency — an approximate monetary amount in the deposit currency that you will gain or lose when the level is triggered. When entering a desired value, the actual calculated amount is displayed below. This may not exactly match the entered value due to price increments.</span></li></ul><p class="p_txt"><span class="f_txt">For each mode, the application displays the corresponding value in the alternative mode. For example, if you set levels in pips, you will immediately see the corresponding price and the approximate amount you will gain or lose if the level is triggered (if nonzero).</span></p><p class="p_txt"><a name="lock_sl_tp" class="hmanchor"></a><span class="f_txt">When setting levels in pips, you can lock the Take Profit to Stop Loss ratio. To do this, set both levels, then tap <img class="help" alt="Fix the Take Profit and Stop Loss ratio" title="Fix the Take Profit and Stop Loss ratio" width="14" height="20" src="/en/docs/mt5/iphone/img/lock_sl_tp_button.png">. Now, adjusting one level will automatically update the other according to the specified ratio, including when moving levels on the <a href="/en/docs/mt5/iphone/chart#manage_orders" class="topiclink">chart</a>. This setting is retained for future orders until it is changed or disabled.</span></p><p class="p_Image"><img class="help" alt="Take Profit and Stop Loss levels change proportionally" title="Take Profit and Stop Loss levels change proportionally" width="660" height="661" src="/en/docs/mt5/iphone/img/position_lock_sl_tp.png"></p><p class="p_txt"><span class="f_txt">Levels can be set in multiple ways:</span></p><ul><li class="p_ol"><span class="f_ol">Directly on the chart — for this, you should temporarily minimize the trading panel to adjust levels. Moving levels near chart edges will <a href="/en/docs/mt5/iphone/chart#scale_levels" class="topiclink">automatically rescale</a> the chart, providing access to a wider price range.</span></li><li class="p_ol"><span class="f_ol">If stop levels are set in pips or currency, they will automatically move by the same amount when the order price changes. This also respects the <a href="/en/docs/mt5/iphone/trade/open_positions#lock_sl_tp" class="topiclink">locked ratio of levels</a> if enabled.</span></li><li class="p_ol"><span class="f_ol">Manually via keyboard — tap the current value and enter a new one.</span></li><li class="p_ol"><span class="f_ol">Using the adjustment panel that opens when you tap on the field with a checkmark. Suggested increment steps adapt dynamically depending on the current value.</span></li></ul><p class="p_txt"><span class="f_txt">To remove a level, either clear the value in the corresponding field or tap the checkmark and then the icon <img class="help" alt="delete_all_icon" width="16" height="20" src="/en/docs/mt5/iphone/img/delete_all_icon.png"> in the panel that appears.</span></p><div><div class="parent-table"><table class="attentable" cellspacing="0" cellpadding="8" border="1"><tbody><tr class="attentable"><td class="attentable"><ul><li class="p_tableatten"><span class="f_tableatten">The entered Take Profit and Stop Loss values are saved after submitting an order. When placing a new order, previously used values will be automatically loaded into the dialog. Only the most recent value for each level is stored; values are not saved separately for each symbol. Saved values are stored in pips and converted to prices when applied, ensuring levels remain accurate even if the instrument price changes.</span></li><li class="p_tableatten"><span class="f_tableatten">When switching instruments, previously saved levels may be too far from the current price and may not appear on the chart. In this case, adjust the levels manually or remove and re-add them. </span><span class="f_ol">By default, levels are placed within visible chart range.</span></li></ul></td></tr></tbody></table></div></div><h3>Additional Parameters</h3><p class="p_txt"><span class="f_txt">At the bottom of the dialog, you can specify additional parameters:</span></p><ul><li class="p_ol"><span class="f_ol">In the 'Fill Policy', additional order <a href="/en/docs/mt5/iphone/trade/general_concept/fill_policy" class="topiclink">filling rules</a> can be specified: "Fill or Kill" of "Immediate or Cancel". If the field is inactive, the selection is restricted by the server.</span></li><li class="p_ol"><span class="f_ol">A text comment to the order. It will be displayed in the <a href="/en/docs/mt5/iphone/trade" class="topiclink">trade</a> and <a href="/en/docs/mt5/iphone/history" class="topiclink">history</a> section. The maximum comment length is 31 symbols. The broker or server may modify the comment. For example, if the position is closed by Stop Loss or Take Profit, the comment may reflect this.</span></li></ul><h3>Tick Chart and Position Opening</h3><p class="p_txt"><span class="f_txt">For additional price control when opening a position, a tick chart of the instrument is displayed at the bottom of the dialog.</span></p><p class="p_txt"><span class="f_txt">To send a buy order click Buy, to send a sell order click Sell. If any value in the dialog is incorrect, the corresponding field is highlighted in red. Trading buttons become inactive if the volume or stop levels are invalid.</span></p><h2><a name="instant" class="hmanchor"></a>Instant Execution <a class="anchor" href="open_positions#instant">#</a></h2><p class="p_txt"><span class="f_txt">In this mode, a market order is executed at the price quoted by the broker. When sending an execution request, the platform automatically inserts the current price into the order. If the broker confirms the price, the order is executed.</span></p><p class="p_txt"><a name="deviation" class="hmanchor"></a><span class="f_txt">n this mode, the trader may specify an additional parameter — Deviation. This defines the maximum acceptable difference between the requested price and the execution price. The larger the allowed deviation, the lower the likelihood of receiving a requote. If the price deviation is within the specified limit, the order is executed at the new price without additional confirmation.</span></p><p class="p_Image"><img class="help" alt="Position opening in instant execution mode" title="Position opening in instant execution mode" width="660" height="661" src="/en/docs/mt5/iphone/img/open_instant.png"></p></td></tr><tr class="container"><td class="container" style="width:25%;"><p class="p_txt"><a name="requote" class="hmanchor"></a><span class="f_txt">If, during order processing, the instrument price changes by more than the value specified in the Deviation field, the dealer (server) may reject the order and provide new execution prices. In this case, a corresponding message will appear in the order window.</span></p><p class="p_txt"><span class="f_txt">If you agree with the new prices, tap Accept, and the order will be executed at the updated price. If you do not accept the new price, tap Reject.</span></p><p class="p_txt"><span class="f_txt">The new prices remain valid for only a few seconds. If you do not make a decision within this time, the message "Price changed" will appear. After clicking OK, you will return to the original order placement window.</span></p></td><td class="container" style="width:25%;"><p class="p_Image"><img class="help" alt="Requote" title="Requote" width="324" height="661" src="/en/docs/mt5/iphone/img/requote.png"></p></td></tr><tr class="container"><td class="container" colspan="2" style="width:25%;"><h2><a name="request" class="hmanchor"></a>Request Execution <a class="anchor" href="open_positions#request">#</a></h2><p class="p_txt"><span class="f_txt">In this mode, a market order is executed at a price previously received from the broker. Before submitting a market order, the platform requests execution prices from the broker. After receiving them, the trader may either confirm or reject execution at the quoted price.</span></p><p class="p_txt"><span class="f_txt">Order parameters can be modified only before requesting prices. Once a price request has been sent, the trader can only place the order using the predefined parameters.</span></p><p class="p_txt"><span class="f_txt">To request prices, tap Request. After that, the Buy and Sell buttons will appear in the dialog. The quoted prices are valid for only a few seconds. If no decision is made within that time, the Buy and Sell buttons will disappear again.</span></p><p class="p_Image"><img class="help" alt="An order in the Request Execution mode" title="An order in the Request Execution mode" width="660" height="661" src="/en/docs/mt5/iphone/img/open_request.png"></p><h2><a name="market" class="hmanchor"></a>Market Execution <a class="anchor" href="open_positions#market">#</a></h2><p class="p_txt"><span class="f_txt">In this mode, the broker determines the execution price without additional confirmation from the trader. Sending a market order in this mode implies prior acceptance of the execution price.</span></p><p class="p_txt"><span class="f_txt">In the 'Fill Policy', additional order <a href="/en/docs/mt5/iphone/trade/general_concept/fill_policy" class="topiclink">filling rules</a> can be specified: Fill or Kill, Immediate or Cancel. If the field is inactive, the selection is restricted by the server.</span></p><p class="p_txt"><span class="f_txt">By tapping on 'Sell by Market' or 'Buy by Market', an instruction is sent to the broker to execute the respective sell or buy transaction at the price determined by the broker.</span></p><p class="p_Image"><img class="help" alt="Position opening in the Market Execution mode" title="Position opening in the Market Execution mode" width="660" height="661" src="/en/docs/mt5/iphone/img/market_execution.png"></p><h2><a name="exchange" class="hmanchor"></a>Exchange Execution <a class="anchor" href="open_positions#exchange">#</a></h2><p class="p_txt"><span class="f_txt">In Exchange Execution mode, trading operations are routed to an external trading system (exchange).</span></p><p class="p_txt"><a name="fill_policy" class="hmanchor"></a><span class="f_txt">In the 'Fill Policy', additional order <a href="/en/docs/mt5/iphone/trade/general_concept/fill_policy" class="topiclink">filling rules</a> can be specified: Fill or Kill, Immediate or Cancel. If the field is inactive, the selection is restricted by the server.</span></p><p class="p_txt"><span class="f_txt">By tapping on 'Sell by Market' or 'Buy by Market', an instruction is sent to the broker to execute the respective sell or buy transaction at the price determined by the broker.</span></p><p class="p_Image"><img class="help" alt="Position opening in exchange execution mode" title="Position opening in exchange execution mode" width="660" height="661" src="/en/docs/mt5/iphone/img/open_exchange.png"></p><h2><a name="close" class="hmanchor"></a>Closing a Position <a class="anchor" href="open_positions#close">#</a></h2><p class="p_txt"><span class="f_txt">In order to obtain profit from exchange rate differences, a position must be closed. Closing a position involves executing a trade opposite to the original one. For example, if the initial trade was the purchase of one lot of GOLD, closing the position requires selling one lot of the same instrument.</span></p><p class="p_txt"><span class="f_txt">To fully close a position, select it in the <a href="/en/docs/mt5/iphone/trade#context" class="topiclink">Trade</a> tab and open the context menu. You can also select it on the chart and use the corresponding command in the trading dialog.</span></p><p class="p_Image"><img class="help" alt="Closing a Position" title="Closing a Position" width="660" height="661" src="/en/docs/mt5/iphone/img/close_position.png"></p><p class="p_txt"><span class="f_txt">In <a href="/en/docs/mt5/iphone/trade/open_positions#request" class="topiclink">Request Execution</a> mode, you must first request a price from the broker before closing the position. After receiving the quoted price, the close button will be available for a few seconds, during which the trader must make a decision.</span></p><div><div class="parent-table"><table class="attentable" cellspacing="0" cellpadding="8" border="1"><tbody><tr class="attentable"><td class="attentable"><p class="p_tableatten"><span class="f_tableatten">A position can be closed in full or in part, depending on the volume of the opposite operation.</span></p></td></tr></tbody></table></div></div></td></tr></tbody></table>
