<table class="container" cellspacing="0" cellpadding="0" border="0"><tbody><tr class="container"><td class="container" style="width:50%;"><h1>Modifying a Position</h1><p class="p_txt"><span class="f_txt">Modifying a position involves changing its <a href="/en/docs/mt5/iphone/trade/general_concept/order_types#stop_loss" class="topiclink">Stop Loss</a> and <a href="/en/docs/mt5/iphone/trade/general_concept/order_types#take_profit" class="topiclink">Take Profit</a> levels. To do this, select the position in the <a href="/en/docs/mt5/iphone/trade#context" class="topiclink">Trade</a> tab and choose 'Modify Position' from the context menu.</span></p><p class="p_txt"><span class="f_txt">To change a value, tap on - or + buttons next to the field, or tap directly on the field to enter a value using the keyboard</span><span class="f_ol">.</span></p><p class="p_txt"><span class="f_ol">Once you have set the desired values, click 'Modify'.</span></p><div><div class="parent-table"><table class="attentable" cellspacing="0" cellpadding="8" border="1"><tbody><tr class="attentable"><td class="attentable" style="vertical-align:middle;"><p class="p_tableatten"><span class="f_tableatten">The "Modify..." button remains disabled until valid values for <a href="/en/docs/mt5/iphone/trade/general_concept/order_types#stop_loss" class="topiclink">Stop Loss</a> and <a href="/en/docs/mt5/iphone/trade/general_concept/order_types#take_profit" class="topiclink">Take Profit</a> are entered. Stop level requirements are determined by the broker and specified in the <a href="/en/docs/mt5/iphone/quotes#specification" class="topiclink">symbol properties</a> (contract specifications).</span></p></td></tr></tbody></table></div></div></td><td class="container" style="width:50%;"><p class="p_Image"><img class="help" alt="Modifying a Position" title="Modifying a Position" width="324" height="661" src="/en/docs/mt5/iphone/img/position_modify.png"></p></td></tr><tr class="container"><td class="container" colspan="2" style="width:50%;"><h2><a name="extended" class="hmanchor"></a>Advanced Mode <a class="anchor" href="change_positions#extended">#</a></h2><p class="p_txt"><span class="f_txt">This mode is available when modifying positions directly from the <a href="/en/docs/mt5/iphone/chart#manage_orders" class="topiclink">chart</a> and provides additional tools for risk calculation.</span></p><h3>Working with the Dialog</h3><p class="p_txt"><span class="f_txt">The trading dialog can be minimized, expanded, or stretched to full screen by dragging it up or down, or by using the buttons <img class="help" alt="Expand" title="Expand" width="20" height="6" src="/en/docs/mt5/iphone/img/trade_dialog_maximize.png"> / <img class="help" alt="Resize" title="Resize" width="20" height="2" src="/en/docs/mt5/iphone/img/trade_dialog_resize.png"> in the header. Tapping once on an empty area of the chart minimizes the dialog, and tapping again hides it completely.</span></p><h3>Take Profit and Stop Loss</h3><p class="p_txt"><span class="f_txt"><a href="/en/docs/mt5/iphone/trade/general_concept/order_types#take_profit" class="topiclink">Take Profit</a> and <a href="/en/docs/mt5/iphone/trade/general_concept/order_types#stop_loss" class="topiclink">Stop Loss</a> levels can be set in several modes:</span></p><ul><li class="p_ol"><span class="f_ol">Price — the instrument price at which the order should execute.</span></li><li class="p_ol"><span class="f_ol">Pips — the distance from the order price in pips.</span></li><li class="p_ol"><span class="f_ol">Currency — an approximate monetary amount in the deposit currency that you will gain or lose when the level is triggered. When entering a desired value, the actual calculated amount is displayed below. This may not exactly match the entered value due to price increments.</span></li></ul><p class="p_txt"><span class="f_txt">For each mode, the application displays the corresponding value in the alternative mode. For example, if you set levels in pips, you will immediately see the corresponding price and the approximate amount you will gain or lose if the level is triggered (if nonzero).</span></p><p class="p_txt"><a name="lock_sl_tp" class="hmanchor"></a><span class="f_txt">When setting levels in pips, you can lock the Take Profit to Stop Loss ratio. To do this, set both levels, then tap <img class="help" alt="Fix the Take Profit and Stop Loss ratio" title="Fix the Take Profit and Stop Loss ratio" width="14" height="20" src="/en/docs/mt5/iphone/img/lock_sl_tp_button.png">. Now, adjusting one level will automatically update the other according to the specified ratio, including when moving levels on the <a href="/en/docs/mt5/iphone/chart#manage_orders" class="topiclink">chart</a>. This setting is retained for future orders until it is changed or disabled.</span></p><p class="p_Image"><img class="help" alt="Setting Take Profit and Stop Loss Levels in advanced mode" title="Setting Take Profit and Stop Loss Levels in advanced mode" width="660" height="661" src="/en/docs/mt5/iphone/img/position_modify_extended.png"></p><p class="p_txt"><span class="f_txt">Levels can be set in multiple ways:</span></p><ul><li class="p_ol"><span class="f_ol">Directly on the chart — for this, you should temporarily minimize the trading panel to adjust levels. Moving levels near chart edges will <a href="/en/docs/mt5/iphone/chart#scale_levels" class="topiclink">automatically rescale</a> the chart, providing access to a wider price range.</span></li><li class="p_ol"><span class="f_ol">If stop levels are set in pips or currency, they will automatically move by the same amount when the order price changes. This also respects the <a href="/en/docs/mt5/iphone/trade/change_positions#lock_sl_tp" class="topiclink">locked ratio of levels</a> if enabled.</span></li><li class="p_ol"><span class="f_ol">Manually via keyboard — tap the current value and enter a new one.</span></li><li class="p_ol"><span class="f_ol">Using the adjustment panel that opens when you tap on the field with a checkmark. Suggested increment steps adapt dynamically depending on the current value.</span></li></ul><p class="p_txt"><span class="f_txt">To remove a level, either clear the value in the corresponding field or tap on the checkmark and then the icon <img class="help" alt="delete_all_icon" width="16" height="20" src="/en/docs/mt5/iphone/img/delete_all_icon.png"> in the panel that appears.</span></p><p class="p_txt"><span class="f_txt">Once you have set the desired values, click 'Modify'.</span></p><div><div class="parent-table"><table class="attentable" cellspacing="0" cellpadding="8" border="1"><tbody><tr class="attentable"><td class="attentable"><ul><li class="p_tableatten"><span class="f_tableatten">The entered Take Profit and Stop Loss values are saved after submitting an order. When placing a new order, previously used values will be automatically loaded into the dialog. Only the most recent value for each level is stored; values are not saved separately for each symbol. Saved values are stored in pips and converted to prices when applied, ensuring levels remain accurate even if the instrument price changes.</span></li><li class="p_tableatten"><span class="f_tableatten">When switching instruments, previously saved levels may be too far from the current price and may not appear on the chart. In this case, adjust the levels manually or remove and re-add them. </span><span class="f_ol">By default, levels are placed within visible chart range.</span></li></ul></td></tr></tbody></table></div></div></td></tr></tbody></table>
