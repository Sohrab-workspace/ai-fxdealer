<table class="container" cellspacing="0" cellpadding="0" border="0"><tbody><tr class="container"><td class="container" style="width:50%;"><h1>Managing Pending Orders</h1><p class="p_txt"><span class="f_txt">When trading, you may need to <a href="/en/docs/mt5/iphone/trade/pending_change#delete" class="topiclink">delete</a> or modify a <a href="/en/docs/mt5/iphone/trade/general_concept/order_types#pending_order" class="topiclink">pending order</a> — for example, to set a new trigger price, adjust Stop Loss or Take Profit levels, or change the expiration time.</span></p><h2><a name="modify" class="hmanchor"></a>Modifying a Pending Order <a class="anchor" href="pending_change#modify">#</a></h2><p class="p_txt"><span class="f_txt">You can modify a pending order in several ways:</span></p><ul><li class="p_ol"><span class="f_ol">Open its context menu in the <a href="/en/docs/mt5/iphone/trade#context" class="topiclink">Trade</a> tab and tap Modify.</span></li><li class="p_ol"><span class="f_ol">Select the order on the chart, then <a href="/en/docs/mt5/iphone/chart#manage_orders" class="topiclink">drag its levels</a> to the desired prices. After that, fine-tune the parameters in the <a href="/en/docs/mt5/iphone/trade/pending_place#extended" class="topiclink">advanced trading dialog</a>.</span></li></ul><p class="p_txt"><span class="f_txt">Almost all parameters of a pending order can be modified, just as when <a href="/en/docs/mt5/iphone/trade/pending_place" class="topiclink">placing</a> it, except for volume. After setting the required parameters, tap Modify.</span></p><p class="p_Image"><img class="help" alt="Modifying a pending order" title="Modifying a pending order" width="660" height="661" src="/en/docs/mt5/iphone/img/pending_modify.png"></p><div><div class="parent-table"><table class="attentable" cellspacing="0" cellpadding="8" border="1"><tbody><tr class="attentable"><td class="attentable" style="vertical-align:middle;"><p class="p_tableatten"><span class="f_tableatten">If the order parameters are incorrect, the "Modify" button becomes inactive.</span></p></td></tr></tbody></table></div></div><h2><a name="delete" class="hmanchor"></a>Deleting a Pending Orders <a class="anchor" href="pending_change#delete">#</a></h2><p class="p_txt"><span class="f_txt">To delete a pending order, tun the Delete command in its context menu on the <a href="/en/docs/mt5/iphone/trade#context" class="topiclink">Trade</a> tab.</span></p><p class="p_txt"><span class="f_txt">Pending orders are deleted automatically at the time specified in the <a href="/en/docs/mt5/iphone/trade/pending_place#expiration" class="topiclink">Expiration</a> field. In the <a href="/en/docs/mt5/iphone/history" class="topiclink">History</a> tab, a deleted pending order is marked as 'Canceled'.</span></p><p class="p_txt"><span class="f_txt">To delete an order directly form the chart, select it, expand the <a href="/en/docs/mt5/iphone/trade/pending_place#extended" class="topiclink">lower trading menu</a>, and tap 'Delete'.</span></p><p class="p_Image"><img class="help" alt="Deleting a pending order" title="Deleting a pending order" width="660" height="661" src="/en/docs/mt5/iphone/img/pending_delete.png"></p></td></tr></tbody></table>
