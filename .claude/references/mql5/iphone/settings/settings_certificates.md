<table class="container" cellspacing="0" cellpadding="0" border="0"><tbody><tr class="container"><td class="container" style="width:50%;"><h1>Certificates</h1><p class="p_txt"><span class="f_txt">The list of installed certificates for the <a href="/en/docs/mt5/iphone/settings_accounts/extended_authorization" class="topiclink">extended authentication</a> is displayed in this section. The extended authentication improves trading account security. If it is enabled on a trade server, users will need to have a special security certificate in addition to a login (account number) and a password. Certificate is a unique access key for a trading account.</span></p><p class="p_txt"><span class="f_txt">You can open this section by selecting Certificates in the <a href="/en/docs/mt5/iphone/settings" class="topiclink">platform settings</a>.</span></p><p class="p_txt"><span class="f_txt">To remove a previously installed certificate, tap on its line and select the relevant command. Optionally, you can swipe left on the certificate line, after which the "Delete" button will appear.</span></p><p class="p_txt"><span class="f_txt">From this section, you can also export the certificate to a PFX file and use it on another device. Press on the certificate line and select "Export". Next, enter the master password for the account for which the certificate is used. The certificate file will be protected by this password. So, when installing the certificate on a new device, you will need to enter this password.</span></p><p class="p_txt"><span class="f_txt">&nbsp;</span></p></td><td class="container" style="width:50%;"><p class="p_Image"><img class="help" alt="Certificates" title="Certificates" width="324" height="578" src="/en/docs/mt5/iphone/img/certificates.png"></p></td></tr><tr class="container"><td class="container" colspan="2" style="width:50%;"><div><div class="parent-table"><table class="attentable" cellspacing="0" cellpadding="8" border="1"><tbody><tr class="attentable"><td class="attentable"><ul><li class="p_tableatten"><span class="f_tableatten">If you delete the certificate necessary for connecting to a trading account, you will not be able to enter the account unless <a href="/en/docs/mt5/iphone/settings_accounts/extended_authorization" class="topiclink">the certificate is reinstalled</a>.</span></li><li class="p_tableatten"><span class="f_tableatten">If MetaTrader 5 iPhone contains no certificates, "Certificates" section is not displayed.</span></li></ul></td></tr></tbody></table></div></div></td></tr></tbody></table>
