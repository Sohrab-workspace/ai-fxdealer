Tap <img class="help" alt="Rate" title="Rate" width="61" height="12" src="/en/docs/mt5/iphone/img/rate_button.png"> to add a review.</span></p></td></tr></tbody></table></div></div></td><td class="container" style="width:50%;"><p class="p_Image"><img class="help" alt="About" title="About" width="322" height="646" src="/en/docs/mt5/iphone/img/about.png"></p></td></tr><tr class="container"><td class="container" colspan="2" style="width:50%;"><h2>Lock Screen</h2><p class="p_txt"><span class="f_txt">You can set a PIN or use Face ID/Touch ID to prevent other users from accessing the app. This function will keep your trading account secure even if your iPhone or iPad falls into the wrong hands.</span></p><p class="p_txt"><span class="f_txt">To activate the function, enable the Screen Lock option in the Settings/About window and set a four-digit PIN. Now, if the application is hidden for more than a minute, you will need to enter the PIN or unlock the app using Face ID/Touch ID.</span></p><p class="p_txt"><span class="f_txt">The app uses the same PIN for the screen lock feature and for access to the <a href="/en/docs/mt5/iphone/settings_accounts/otp" class="topiclink">one-time password generator</a>.</span></p><p class="p_Image"><img class="help" alt="Turn on screen lock for increased security" title="Turn on screen lock for increased security" width="650" height="646" src="/en/docs/mt5/iphone/img/lock_screen.png"></p></td></tr><tr class="container"><td class="container" style="width:25%;"><p class="p_Image"><img class="help" alt="App data management" title="App data management" width="324" height="636" src="/en/docs/mt5/iphone/img/storage.png"></p></td><td class="container" style="width:25%;"><h2><a name="storage" class="hmanchor"></a>Storage <a class="anchor" href="settings_about#storage">#</a></h2><p class="p_txt"><span class="f_txt">The running application uses various data downloaded from the broker's server: account information and trading history, prices for symbol charts and other data.</span></p><p class="p_txt"><span class="f_txt">Over time, the amount of accumulated data can become quite large.</span></p><p class="p_txt"><span class="f_txt">If your device does not have enough free space, you can delete some of this data. This data is not used in trading.</span></li><li class="p_txt"><span class="f_txt">Clean all caches — this command deletes all data stored in the app, including trading and price history. To delete current account data, the account will be reconnected to the server.</span></li></ul><p class="p_txt"><span class="f_txt">Also, you can clean the data associated only with a specific trading server. This can be done by tapping on the relevant server line in the list.</span></p><p class="p_txt"><span class="f_txt">To clean the data of a specific trading account, tap on this account in the list.</span></p></td></tr></tbody></table>
