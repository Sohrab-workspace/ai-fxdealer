<table class="container" cellspacing="0" cellpadding="0" border="0"><tbody><tr class="container"><td class="container" style="width:50%;"><h1>Managing Passwords</h1><p class="p_txt"><span class="f_txt">MetaTrader 5 iPhone allows changing trading account passwords, as well as deleting previously saved ones. If you wish to change a password, tap "Change Password" in the <a href="/en/docs/mt5/iphone/settings_accounts#current_account" class="topiclink">current account management</a> section. The list of available actions will appear:</span></p><ul><li class="p_ol"><span class="f_ol">Change Master Password providing full access to the trading account;</span></li><li class="p_ol"><span class="f_ol">Change Investor Password providing access to the trading account in "read-only" mode (no trading operations).</span></li><li class="p_ol"><span class="f_ol">Clear previously saved password. If "Save password" option is enabled <a href="/en/docs/mt5/iphone/settings_accounts/account_connect" class="topiclink">when connecting to the account</a>, the application will save the password, so that you will not have to enter it each time you connect. "Clear Saved Password" deletes such password. Thus, you will have to enter it again when connecting next time.</span></li></ul><div><div class="parent-table"><table class="attentable" cellspacing="0" cellpadding="8" border="1"><tbody><tr class="attentable"><td class="attentable"><p class="p_tableatten"><span class="f_tableatten">All performed actions influence the passwords of the current account (the account you are currently connected to).</span></p></td></tr></tbody></table></div></div></td><td class="container" style="width:50%;"><p class="p_Image"><img class="help" alt="Managing Passwords" title="Managing Passwords" width="324" height="646" src="/en/docs/mt5/iphone/img/manage_passwords.png"></p></td></tr><tr class="container"><td class="container" style="width:50%;"><p class="p_Image"><img class="help" alt="Password change" title="Password change" width="322" height="554" src="/en/docs/mt5/iphone/img/change_password.png"></p></td><td class="container" style="width:50%;"><h2><a name="change" class="hmanchor"></a>Changing Master and Investor Passwords <a class="anchor" href="account_passwords#change">#</a></h2><p class="p_txt"><span class="f_txt">Master and investor passwords are changed in similar ways. After selecting an appropriate option in the above dialog, you proceed to password change. The following data should be specified here:</span></p><ul><li class="p_ol"><span class="f_ol" style="font-weight: bold;">Current</span><span class="f_ol"> — specify the current master password;</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;">New</span><span class="f_ol"> — specify the new password;</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;">Confirm</span><span class="f_ol"> — specify the new password once again to prevent errors;</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;">Save password</span><span class="f_ol"> — enable to save the password. You will not have to enter saved passwords when connecting to your trading account.</span></li></ul><p class="p_txt"><span class="f_txt">The new password must meet the following requirements:</span></p><ul><li class="p_ol"><span class="f_ol">It must be no shorter than the length requirement displayed in the password change dialog.</span></li><li class="p_ol"><span class="f_ol">It must contain four character types: lowercase letters, uppercase letters, numbers, and special <a href="https://learn.microsoft.com/en-us/style-guide/a-z-word-list-term-collections/term-collections/special-characters" target="_blank" class="weblink">characters</a> (#, @, ! etc.). For example, 1Ar#pqkj.</span></li><li class="p_ol"><span class="f_ol">It must not match the previous password.</span></li></ul></td></tr><tr class="container"><td class="container" style="width:50%;"><h2><a name="clear" class="hmanchor"></a>Deleting Saved Passwords <a class="anchor" href="account_passwords#clear">#</a></h2><p class="p_txt"><span class="f_txt">If "Save password" option is enabled when <a href="/en/docs/mt5/iphone/settings_accounts/account_connect" class="topiclink">connecting to the account</a>, the application saves the password, so that you do not have to enter it each time you connect. "Clear Saved Password" deletes such password. Thus, you will have to enter it again during the next connection.</span></p><p class="p_txt"><span class="f_txt">&nbsp;</span></p></td><td class="container" style="width:50%;"><p class="p_Image"><img class="help" alt="Saved password has been cleared" title="Saved password has been cleared" width="324" height="596" src="/en/docs/mt5/iphone/img/clear_passwords.png"></p></td></tr></tbody></table>
