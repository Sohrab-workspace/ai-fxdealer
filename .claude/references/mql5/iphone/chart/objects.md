<table class="container" cellspacing="0" cellpadding="0" border="0"><tbody><tr class="container"><td class="container" colspan="2" style="width:50%;"><h1>Objects</h1><p class="p_txt"><span class="f_txt">Objects in the platform mean objects that are added manually to a chart. These objects are used for analytical purposes. This group of functions includes:</span></p><ul><li class="p_ol"><span class="f_ol" style="font-weight: bold;"><a href="/en/docs/mt5/iphone/chart/objects/lines" class="topiclink">Lines</a></span><span class="f_ol"> — various lines (trend lines, horizontal, cyclic lines, etc.) added to price charts and indicators.</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;"><a href="/en/docs/mt5/iphone/chart/objects/channels" class="topiclink">Channels</a></span><span class="f_ol"> — various channels (equidistant, regression channel, etc.) added to price charts and indicators.</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;"><a href="/en/docs/mt5/iphone/chart/objects/gann" class="topiclink">Gann Tools</a></span><span class="f_ol"> — a set of technical analysis tools developed by W. D. Gann.</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;"><a href="/en/docs/mt5/iphone/chart/objects/fibo" class="topiclink">Fibonacci Tools</a></span><span class="f_ol"> — a set of technical analysis tools created on the bases of the numerical sequence by L. Fibonacci.</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;"><a href="/en/docs/mt5/iphone/chart/objects/elliott" class="topiclink">Elliott Tools</a></span><span class="f_ol"> — a set of tools technical analysis based on the wave theory of R. N. Elliott.</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;"><a href="/en/docs/mt5/iphone/chart/objects/shapes" class="topiclink">Shapes</a></span><span class="f_ol"> — geometric shapes (square, triangle, and ellipse) that allow to mark various areas in the price chart.</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;"><a href="/en/docs/mt5/iphone/chart/objects/text" class="topiclink">Text</a></span><span class="f_ol"> — an object to add text labels to the chart.</span></li></ul><h2>How to Add an Object</h2><p class="p_txt"><span class="f_txt">To add an object, tap <img class="help" alt="Objects" title="Objects" width="21" height="20" src="/en/docs/mt5/iphone/img/objects_icon.png"> at the top of the <a href="/en/docs/mt5/iphone/chart" class="topiclink">chart</a>. Then tap "Add object" or select one of the recently used objects below.</span></p><p class="p_Image"><img class="help" alt="Adding an object" title="Adding an object" width="322" height="654" src="/en/docs/mt5/iphone/img/add_object.png"><span class="f_Image"> <img class="help" alt="Selecting an object" title="Selecting an object" width="322" height="654" src="/en/docs/mt5/iphone/img/add_object_select.png"></span></p><p class="p_txt"><span class="f_txt">Upon object selection, a chart will open with an icon of the selected object.</span></p><div><div class="parent-table"><table class="attentable" cellspacing="0" cellpadding="8" border="1"><tbody><tr class="attentable"><td class="attentable"><p class="p_tableatten"><span class="f_tableatten">The total number of objects on all charts cannot exceed 128. If you try to add more objects, the oldest objects will be automatically deleted.</span></p></td></tr></tbody></table></div></div></td></tr><tr class="container"><td class="container" style="width:50%;"><h2>How to Drawing an Object</h2><p class="p_txt"><span class="f_txt">Objects may have different numbers of changing and moving points depending on their type. As a rule an object can be moved using its central point. To create most objects, you need to specify one or more initial points and then, holding your finger down, set the direction and size.</span></p><p class="p_txt"><span class="f_txt">When you move the object point, a special subwindow with the zoomed chart will appear in the upper left corner of the chart to allow you to accurately position the object. For more information on object drawing and modifying using points, please read descriptions of <a href="/en/docs/mt5/iphone/chart/objects" class="topiclink">objects</a>.</span></p><p class="p_txt"><span class="f_txt">When you move the point close to one of the bar prices (Open, High, Low or Close), the object point will be automatically snapped to this price level. This option is included for the convenient positioning of objects on a chart.</span></p><p class="p_txt"><span class="f_txt">Created objects can be copied. This enables faster chart markup. Open the object menu with a long press and select "Copy".</span></p><div><div class="parent-table"><table class="attentable" cellspacing="0" cellpadding="8" border="1"><tbody><tr class="attentable"><td class="attentable"><p class="p_tableatten"><span class="f_tableatten">When dragging an object using its moving point (central point) snapping does not work.</span></p></td></tr></tbody></table></div></div></td><td class="container" style="width:50%;"><p class="p_Image"><img class="help" alt="Drawing an object" title="Drawing an object" width="323" height="556" src="/en/docs/mt5/iphone/img/object_plot.png"></p></td></tr><tr class="container"><td class="container" style="width:50%;"><p class="p_Image"><img class="help" alt="Object Properties" title="Object Properties" width="322" height="569" src="/en/docs/mt5/iphone/img/object_properties.png"></p></td><td class="container" style="width:50%;"><h2>How to Modify an Object</h2><p class="p_txt"><span class="f_txt">After an object has been created, it can be moved or modified. To do so, select the object first by tapping on it. The object is considered selected when markers (object points) appear. The markers are used for moving objects and adjusting their drawing parameters.</span></p><p class="p_txt"><span class="f_txt">For example, to change the position of a Fibonacci Fan, hold its central marker and drag it to the desired position. Moving any of its outer markers modifies the drawing parameters of the object.</span></p><div><div class="parent-table"><table class="attentable" cellspacing="0" cellpadding="8" border="1"><tbody><tr class="attentable"><td class="attentable"><p class="p_tableatten"><span class="f_tableatten">To delete an object or open its properties, perform a long tap (about 1 sec) on it directly on the chart.</span></p></td></tr></tbody></table></div></div><p class="p_txt"><span class="f_txt">Once you finish your markup, tap "Lock all" at the top of the object list. This will disable the ability to move objects or access their properties, allowing you to review prices without the risk of accidentally altering your markup.</span></p><p class="p_txt"><span class="f_txt">To quickly view a "clean" chart without deleting existing objects, tap "Hide all" at the top of the object list.</span></p></td></tr><tr class="container"><td class="container" colspan="2" style="width:50%;"><h2><a name="properties" class="hmanchor"></a>Object Properties <a class="anchor" href="objects#properties">#</a></h2><p class="p_txt"><span class="f_txt">Every object has its specific properties. Properties can differ depending on the object. To manage the properties of an object, long-tap on it on the chart and then select "Properties" from the context menu. You can also tap on <img class="help" alt="Objects" title="Objects" width="21" height="20" src="/en/docs/mt5/iphone/img/objects_icon.png"> at the top of the <a href="/en/docs/mt5/iphone/chart" class="topiclink">chart</a> and select one of the added objects.</span></p><p class="p_txt"><span class="f_txt">The following is available here:</span></p><ul><li class="p_ol"><a name="name" class="hmanchor"></a><span class="f_ol" style="font-weight: bold;">Name</span><span class="f_ol"> — the unique name of an object within one chart, it is set automatically. It can be changed if another name is entered in this field. Such names make it easy to find an object among many other objects of the same type.</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;">Ray right/left</span><span class="f_ol"> — displaying trend lines as rays in the specified directions.</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;">Coordinates</span><span class="f_ol"> — one or several snap points along the time and price axes.</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;">Visualization</span><span class="f_ol"> — instruments and timeframes, in which the object will be displayed. If the object should have different settings depending on the chart period, create several copies of the object and enable the display for each of them in the required timeframe.</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;">Style</span><span class="f_ol"> — object lines style. Color, style and width of lines can be set here.</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;">Draw as background</span><span class="f_ol"> — draw the object in the background layer, behind the chart. For shapes and channels (except for the Fibonacci channel), enabling this option sets color filling for the object.</span></li><li class="p_ol"><span class="f_txt" style="font-weight: bold;">Levels</span><span class="f_txt"> </span><span class="f_ol">—</span><span class="f_txt"> specific display settings for the levels of Fibonacci tools and Andrews' Pitchfork.</span></li></ul><p class="p_txt"><span class="f_txt">Some object have specific properties like "Gann Angle", "Pips per bar". etc. Description of these properties in available in corresponding <a href="/en/docs/mt5/iphone/chart/objects" class="topiclink">object</a> sections.</span></p><h2>How to Delete an Object</h2><p class="p_txt"><span class="f_txt">All objects applied to a chart become unnecessary sooner or later, and they can be removed using the object managing menu. Tap <img class="help" alt="Objects" title="Objects" width="21" height="20" src="/en/docs/mt5/iphone/img/objects_icon.png"> at the top of the <a href="/en/docs/mt5/iphone/chart" class="topiclink">chart</a>.</span></p><p class="p_txt"><span class="f_txt">Objects can be deleted as follows:</span></p><ul><li class="p_ol"><span class="f_ol">Tap </span><img class="help" alt="Delete" title="Delete" width="16" height="20" src="/en/docs/mt5/iphone/img/delete_all_icon.png"><span class="f_ImageCaption"> </span><span class="f_ol">in the upper right corner of the screen. Here you can delete all objects from the chart.</span></li><li class="p_ol"><span class="f_ol">Swipe from right to left on the object line. To remove an object, use the "Delete" button that appears.</span></li></ul><p class="p_Image"><img class="help" alt="Deleting all objects" title="Deleting all objects" width="322" height="554" src="/en/docs/mt5/iphone/img/object_delete_all.png"><span class="f_Image"> <img class="help" alt="Deleting a single object" title="Deleting a single object" width="322" height="554" src="/en/docs/mt5/iphone/img/object_delete.png"></span></p></td></tr></tbody></table>
