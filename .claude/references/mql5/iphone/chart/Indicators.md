<table class="container" cellspacing="0" cellpadding="0" border="0"><tbody><tr class="container"><td class="container" style="width:50%;"><h1>Indicators for Technical Analysis</h1><p class="p_txt"><span class="f_txt">A technical indicator is a mathematical manipulation of a security price and/or volumes aimed at forecasting future price changes. Decisions about how and when to open or close positions can be made on the basis of signals from technical indicators.</span></p><p class="p_txt"><span class="f_txt">According to their functional properties, indicators can be divided into two groups: trend indicators and oscillators. Trend indicators help to assess the price direction and detect the turn moments synchronously or with a delay. Oscillators allow to find the turning points in advance or synchronously.</span></p><p class="p_txt"><span class="f_txt">The MetaTrader 5 for iPhone/iPad provides 30 indicators. They are divided into the following groups:</span></p><ul><li class="p_ol"><span class="f_ol"><a href="/en/docs/mt5/iphone/chart/indicators/trend_indicators" class="topiclink">Trend Indicators</a></span></li><li class="p_ol"><span class="f_ol"><a href="/en/docs/mt5/iphone/chart/indicators/oscillators" class="topiclink">Oscillators</a></span></li><li class="p_ol"><span class="f_ol"><a href="/en/docs/mt5/iphone/chart/indicators/volume_indicators" class="topiclink">Volume Indicators</a></span></li><li class="p_ol"><span class="f_ol"><a href="/en/docs/mt5/iphone/chart/indicators/bw_indicators" class="topiclink">Bill Williams' Indicators</a></span></li></ul></td><td class="container" style="width:50%;"><p class="p_Image"><img class="help" alt="Managing Indicators" title="Managing Indicators" width="322" height="554" src="/en/docs/mt5/iphone/img/indicator_management.png"></p></td></tr><tr class="container"><td class="container" colspan="2" style="width:50%;"><p class="p_txt"><span class="f_txt">A technical indicator can be drawn in a separate indicator window with its own vertical scale (for example, <a href="/en/docs/mt5/iphone/chart/indicators/oscillators/macd" class="topiclink">MACD</a>) or applied directly to a price chart (like <a href="/en/docs/mt5/iphone/chart/indicators/trend_indicators/moving_average" class="topiclink">Moving Average</a>). Each indicator has a specific set of <a href="/en/docs/mt5/iphone/chart/indicators#settings" class="topiclink">parameters</a>.</span></p></td></tr><tr class="container"><td class="container" style="width:50%;"><p class="p_Image"><img class="help" alt="List of Indicators" title="List of Indicators" width="322" height="553" src="/en/docs/mt5/iphone/img/indicator_list.png"></p></td><td class="container" style="width:50%;"><h2>Managing Indicators</h2><p class="p_txt"><span class="f_txt">To go to the indicators management section, tap <img class="help" alt="Indicators" title="Indicators" width="15" height="20" src="/en/docs/mt5/iphone/img/indicators_icon.png"> when viewing a chart or execute the "Indicators" command of the <a href="/en/docs/mt5/iphone/chart#context" class="topiclink">context menu</a>.</span></p><h2><a name="add" class="hmanchor"></a>Adding Indicators <a class="anchor" href="indicators#add">#</a></h2><p class="p_txt"><span class="f_txt">To add an indicator, tap on the header of the chart screen, for example, "Main chart".</span></p><div><div class="parent-table"><table class="attentable" cellspacing="0" cellpadding="8" border="1"><tbody><tr class="attentable"><td class="attentable"><p class="p_tableatten"><span class="f_tableatten">If the indicators are applied only on the main chart or not applied at all, the window provides options for adding/removing indicators only for the main chart. If an indicator is drawn in a separate window (for example, <a href="/en/docs/mt5/iphone/chart/indicators/oscillators/macd" class="topiclink">MACD</a>), the window is created automatically. After an additional window has been created, you can add indicators right to it.</span></p></td></tr></tbody></table></div></div><p class="p_txt"><span class="f_txt">A tap on the chart screen header opens a tab, from which an indicator can be selected.</span></p><p class="p_txt"><span class="f_txt">&nbsp;</span></p></td></tr><tr class="container"><td class="container" colspan="2" style="width:50%;"><p class="p_txt"><span class="f_txt">Select one of the indicators listed here. A detailed description of all available indicators is provided in sections <a href="/en/docs/mt5/iphone/chart/indicators/trend_indicators" class="topiclink">"Trend Indicators"</a>, <a href="/en/docs/mt5/iphone/chart/indicators/oscillators" class="topiclink">"Oscillators"</a>, <a href="/en/docs/mt5/iphone/chart/indicators/volume_indicators" class="topiclink">"Volume Indicators"</a> and <a href="/en/docs/mt5/iphone/chart/indicators/bw_indicators" class="topiclink">"Bill Williams' Indicators"</a>.</span></p><p class="p_txt"><span class="f_txt">After you select an indicators, its <a href="/en/docs/mt5/iphone/chart/indicators#settings" class="topiclink">properties</a> will open.</span></p><div><div class="parent-table"><table class="attentable" cellspacing="0" cellpadding="8" border="1"><tbody><tr class="attentable"><td class="attentable"><p class="p_tableatten"><span class="f_tableatten">You can add up to 128 indicators to a chart.</span></p></td></tr></tbody></table></div></div></td></tr><tr class="container"><td class="container" style="width:50%;"><h2><a name="settings" class="hmanchor"></a>Indicator Properties <a class="anchor" href="indicators#settings">#</a></h2><p class="p_txt"><span class="f_txt">Each indicator has a specific set of parameters.</span></p><p class="p_txt"><span class="f_txt">One of the common parameters is "Apply to". Indicators can be plotted based on price data and derivatives thereof (Median Price, Typical Price, Weighted Close), as well as on the basis of other indicators.</span></p><p class="p_txt"><span class="f_txt">For example, a <a href="/en/docs/mt5/iphone/chart/indicators/trend_indicators/moving_average" class="topiclink">Moving Average</a> can be constructed based on <a href="/en/docs/mt5/iphone/chart/indicators/oscillators/macd" class="topiclink">MACD</a> values. To do this, first plot the MACD indicator, and then in its window <a href="/en/docs/mt5/iphone/chart/indicators#add" class="topiclink">add</a> MA, selecting in the "Apply to" field of its settings the "Previous indicator's data" option.</span></p><p class="p_txt"><span class="f_txt">If you select "First indicator's data", MA will be constructed on the basis of the very first indicator, which is not necessarily the MACD. There are nine variants of indicator drawing:</span></p><ul><li class="p_ol"><span class="f_ol" style="font-weight: bold;">Close </span><span class="f_ol">— based on close prices.</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;">Open</span><span class="f_ol"> — based on open prices.</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;">High</span><span class="f_ol"> — based on High prices.</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;">Low</span><span class="f_ol"> — based on Low prices.</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;">Median Price (HL/2)</span><span class="f_ol"> — based on the median price: (High + Low)/2.</span></li></ul></td><td class="container" style="width:50%;"><p class="p_Image"><img class="help" alt="Indicator settings" title="Indicator settings" width="322" height="570" src="/en/docs/mt5/iphone/img/indicator_settings.png"></p></td></tr><tr class="container"><td class="container" colspan="2" style="width:50%;"><ul><li class="p_ol"><span class="f_ol" style="font-weight: bold;">Typical Price (HLC/3)</span><span class="f_ol"> — based on the typical price: (High + Low + Close)/3.</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;">Weighted Close (HLCC/4)</span><span class="f_ol"> — based on the average weighted close price: (High + Low + 2*Close)/4.</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;">First indicator's data</span><span class="f_ol"> — based on the values of the first applied indicator. The option of using the data of the first indicator is only available for indicators in a separate window, because in the main chart window the first indicator is the price.</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;">Previous indicator's data</span><span class="f_ol"> — based on the values of the previous indicator.</span></li></ul><p class="p_txt"><span class="f_txt">For some indicators, additional levels can be enabled. When constructed in the main window, levels are plotted at a specified distance from the main line, thus creating a channel. For indicators plotted in the additional subwindow, the levels are shown as horizontal lines. To construct the levels, select Levels in the indicator properties and click "Add level". Next, enter a value in points and a description (optional). The line color, width and style for the levels can be set up below.</span></p><p class="p_txt"><span class="f_txt">Display settings are available for all indicators. These include the color, type and thickness of each line. To open the settings, click on the line name. You can also manage the display of indicators on different timeframes. If the indicator is not suitable for certain chart periods, it can be hidden to free up space on the screen for other analytical tools.</span></p><p class="p_txt"><span class="f_txt">Please see the description of the relevant indicator for details of how to set up specific parameters.</span></p><h2><a name="delete" class="hmanchor"></a>Removing Indicators <a class="anchor" href="indicators#delete">#</a></h2><p class="p_txt"><span class="f_txt">You can delete indicators as follows:</span></p><ul><li class="p_ol"><span class="f_ol">Tap the Modify button in the upper right corner. To remove an indicator or an additional window, tap <img class="help" alt="Delete" title="Delete" width="22" height="22" src="/en/docs/mt5/iphone/img/delete_icon.png">.</span></li><li class="p_ol"><span class="f_ol">Swipe your finger from left to right on the indicator line. To remove an indicator, use the Delete button that appears.</span></li></ul><p class="p_Image"><img class="help" alt="Removing an indicator" title="Removing an indicator" width="322" height="554" src="/en/docs/mt5/iphone/img/indicators_delete.png"><span class="f_Image"> <img class="help" alt="Removing an indicator" title="Removing an indicator" width="322" height="554" src="/en/docs/mt5/iphone/img/indicators_delete_one.png"></span></p><h2>Modifying Indicators</h2><p class="p_txt"><span class="f_txt">To change the parameters of an indicator, tap on it in the "Indicators" screen.</span></p><h2><a name="subwindow" class="hmanchor"></a>Indicators in the Subwindow <a class="anchor" href="indicators#subwindow">#</a></h2><p class="p_txt"><span class="f_txt">Some indicators, such as <a href="/en/docs/mt5/iphone/chart/indicators/oscillators/macd" class="topiclink">MACD</a> or other <a href="/en/docs/mt5/iphone/chart/indicators/oscillators" class="topiclink">oscillators</a>, are usually displayed not on the price chart but in a separate subwindow with its own vertical scale. When such an indicator is added, the subwindow is created automatically. You can add any other indicators to an existing subwindow by tapping its title in the list.</span></p><p class="p_Image"><img class="help" alt="Indicators in the Subwindow" title="Indicators in the Subwindow" width="750" height="739" src="/en/docs/mt5/iphone/img/indicators_subwindow.png"></p><p class="p_txt"><span class="f_txt">To resize a subwindow with an indicator, perform a long tap (more than one second) on it or select "Resize chart windows" in the indicator list. Then, adjust the subwindow size by dragging its top border. To close the subwindow and remove the indicators it contains, tap </span><img class="help" alt="Close" title="Close" width="20" height="20" src="/en/docs/mt5/iphone/img/indicators_subwindow_close_button.png"><span class="f_txt">. You can also perform this action by swiping the subwindow title left in the indicator list.</span></p><p class="p_txt"><span class="f_txt">To change the order of subwindows, drag them using the icon <img class="help" alt="Move" title="Move" width="19" height="16" src="/en/docs/mt5/iphone/img/move_button.png">.</span></p><p class="p_Image"><img class="help" alt="Changing subwindows" title="Changing subwindows" width="750" height="739" src="/en/docs/mt5/iphone/img/indicators_subwindow_edit.png"></p><p class="p_txt"><span class="f_txt">Tap anywhere on the chart to exit the edit mode.</span></p></td></tr></tbody></table>
