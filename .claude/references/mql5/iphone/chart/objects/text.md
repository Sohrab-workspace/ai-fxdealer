<table class="container" cellspacing="0" cellpadding="0" border="0"><tbody><tr class="container"><td class="container" colspan="2" style="width:50%;"><h1>Text</h1><p class="p_txt"><span class="f_txt">This object is designed to add text labels to the chart. You can mark important reversal points, news events, additional information about levels, and more. The object is anchored to the chart and moves along with it.</span></p><p class="p_txt"><span class="f_txt">Click "Add Text" in the objects menu, then enter the text you want to display on the chart. Next, click on any point on the chart to place the object:</span></p><p class="p_Image"><img class="help" alt="Add text to the chart" title="Add text to the chart" width="650" height="654" src="/en/docs/mt5/iphone/img/text_add.png"></p><p class="p_txt"><span class="f_txt">To change the text direction, move the second point.</span></p></td></tr><tr class="container"><td class="container" style="width:50%;"><p class="p_Image"><img class="help" alt="Text object parameters" title="Text object parameters" width="322" height="654" src="/en/docs/mt5/iphone/img/text_properties.png"></p></td><td class="container" style="width:50%;"><h2>Parameters</h2><p class="p_txt"><span class="f_txt">The following parameters can be set for the text object:</span></p><ul><li class="p_ol"><span class="f_ol" style="font-weight: bold;">Text</span><span class="f_ol"> — the content of the object.</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;">Coordinates</span><span class="f_ol"> — the coordinates of the anchor point (date/price scale value).</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;">Font</span><span class="f_ol"> — the type and size of the font for the text.</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;">Reverse</span><span class="f_ol"> — changes the text direction: right to left or left to right.</span></li></ul><p class="p_txt"><span class="f_txt">General object parameters are described in the <a href="/en/docs/mt5/iphone/chart/objects#properties" class="topiclink">relevant section</a>.</span></p></td></tr></tbody></table>
