<table class="container" cellspacing="0" cellpadding="0" border="0"><tbody><tr class="container"><td class="container" style="width:25%;"><h1>Objects</h1><p class="p_txt"><span class="f_txt">Objects in the platform are objects that are imposed manually to a chart. These objects are used for analytical purposes. This group of functions includes:</span></p><ul><li class="p_ol"><span class="f_ol" style="font-weight: bold;"><a href="/en/docs/mt5/android/chart/objects/lines" class="topiclink">Lines</a></span><span class="f_ol"> — various lines (trend lines, horizontal, cyclic lines, etc.) added to price charts and indicators;</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;"><a href="/en/docs/mt5/android/chart/objects/channels" class="topiclink">Channels</a></span><span class="f_ol"> — various channels (equidistant, regression channel, etc.) added to price charts and indicators;</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;"><a href="/en/docs/mt5/android/chart/objects/gann" class="topiclink">Gann Tools</a></span><span class="f_ol"> — a set of technical analysis tools developed by W. D. Gann;</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;"><a href="/en/docs/mt5/android/chart/objects/fibo" class="topiclink">Fibonacci Tools</a></span><span class="f_ol"> — a set of technical analysis tools created on the bases of the numerical sequence by L. Fibonacci;</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;"><a href="/en/docs/mt5/android/chart/objects/elliott" class="topiclink">Elliott Tools</a></span><span class="f_ol"> — a set of tools technical analysis based on the wave theory of R. N. Elliott;</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;"><a href="/en/docs/mt5/android/chart/objects/shapes" class="topiclink">Shapes</a></span><span class="f_ol"> — geometric shapes (square, triangle, and ellipse) that allow to mark various areas in the price chart;</span></li></ul></td><td class="container" style="width:25%;"><p class="p_Image"><img class="help" alt="Objects" title="Objects" width="322" height="636" src="/en/docs/mt5/android/img/objects.png"></p></td></tr><tr class="container"><td class="container" colspan="2" style="width:50%;"><h2>How to Add an Object</h2><p class="p_txt"><span class="f_txt">To add an object to a chart, open the <a href="/en/docs/mt5/android/chart#circle" class="topiclink">radial menu</a> and tap <img class="help" alt="Objects" title="Objects" width="21" height="20" src="/en/docs/mt5/android/img/objects_icon.png">. Select the desired object from the list of recently used objects or tap <img class="help" alt="Add Object" title="Add Object" width="20" height="20" src="/en/docs/mt5/android/img/add_object_icon.png"> at the top of the window.</span></p><p class="p_Image"><img class="help" alt="Adding an object" title="Adding an object" width="322" height="636" src="/en/docs/mt5/android/img/add_object.png"><span class="f_Image"> <img class="help" alt="Selecting an object" title="Selecting an object" width="322" height="636" src="/en/docs/mt5/android/img/add_object_select.png"></span></p><p class="p_txt"><span class="f_txt">Upon object selection, a chart will open with an icon of the selected object.</span></p><div><div class="parent-table"><table class="attentable" cellspacing="0" cellpadding="8" border="1"><tbody><tr class="attentable"><td class="attentable"><p class="p_tableatten"><span class="f_tableatten">The total number of objects on all charts cannot exceed 512. If you try to add more objects, the oldest objects will be automatically deleted.</span></p></td></tr></tbody></table></div></div></td></tr><tr class="container"><td class="container" style="width:50%;"><p class="p_Image"><img class="help" alt="Drawing an object" title="Drawing an object" width="322" height="636" src="/en/docs/mt5/android/img/object_plot.png"></p></td><td class="container" style="width:50%;"><h2>How to Drawing an Object</h2><p class="p_txt"><span class="f_txt">Depending on the type, objects may have different numbers of changing and moving points. As a rule the point of an object moving is its central point. To create most objects need, you need to specify one or more initial points and then, holding your finger down, set the direction and size.</span></p><p class="p_txt"><span class="f_txt">When you move the object point, a special subwindow with the zoomed chart will appear in the upper left corner of the chart to allow you to accurately position the object. For more information on object drawing and modifying using points, please read descriptions of <a href="/en/docs/mt5/android/chart/objects" class="topiclink">objects</a>.</span></p><p class="p_txt"><span class="f_txt">When you move the point close to one of the bar prices (Open, High, Low or Close), the object point will be automatically snapped to this price level. This option is included for the convenient positioning of objects on a chart.</span></p><p class="p_txt"><span class="f_txt">Created objects can be copied. This enables faster chart markup. Open the object menu with a long press and select <img class="help" alt="Copy" title="Copy" width="17" height="20" src="/en/docs/mt5/android/img/copy_button.png"> on the top panel.</span></p></td></tr><tr class="container"><td class="container" colspan="2" style="width:50%;"><div><div class="parent-table"><table class="attentable" cellspacing="0" cellpadding="8" border="1"><tbody><tr class="attentable"><td class="attentable"><p class="p_tableatten"><span class="f_tableatten">When dragging an object using its moving point (central point) snapping does not work.</span></p></td></tr></tbody></table></div></div><h2><a name="modify" class="hmanchor"></a>How to Modify an Object <a class="anchor" href="objects#modify">#</a></h2><p class="p_txt"><span class="f_txt">After an object has been created, it can be moved or modified. To do so, select the object by tapping on it. The object is considered selected when markers (object points) appear. The markers are used for moving objects and adjusting their drawing parameters.</span></p><p class="p_txt"><span class="f_txt">For example, to change the position of a Fibonacci Fan, hold its central marker and drag it to the desired position. Moving any of its outer markers modifies the drawing parameters of the object.</span></p><p class="p_txt"><span class="f_txt">To open a list of all objects, tap <img class="help" alt="Objects" title="Objects" width="21" height="20" src="/en/docs/mt5/android/img/objects_icon.png">. The objects are grouped according to the charts on which they are used. The object for which the <a href="/en/docs/mt5/android/chart/objects#symbol" class="topiclink">display on all symbols is enabled</a> are shown under the separate "Independent charts" category.</span></p><div><div class="parent-table"><table class="attentable" cellspacing="0" cellpadding="8" border="1"><tbody><tr class="attentable"><td class="attentable"><p class="p_tableatten"><span class="f_tableatten">To delete an object or open its properties, perform a long tap (about 1 sec) on it directly on the chart.</span></p></td></tr></tbody></table></div></div><p class="p_txt"><span class="f_txt">After adjusting the desired objects and their parameters, tap </span><img class="help" alt="Lock All" title="Lock All" width="18" height="20" src="/en/docs/mt5/android/img/lock__objects_button.png"><span class="f_ImageCaption"> </span><span class="f_txt">at the top of the object list. This will disable the ability to move objects or access their properties, allowing you to review prices without the risk of accidentally altering your markup.</span></p><p class="p_txt"><span class="f_txt">To quickly view a "clean" chart without deleting the previously added objects, tap </span><img class="help" alt="Hide All" title="Hide All" width="30" height="20" src="/en/docs/mt5/android/img/hide_button.png"><span class="f_ImageCaption"> </span><span class="f_txt">at the top of the object list. You can also hide individual objects by tapping the similar buttons next to their name.</span></p><h2>How to Delete an Object</h2><p class="p_txt"><span class="f_txt">All objects applied to a chart become unnecessary sooner or later, and they can be removed using the object managing menu. Tap <img class="help" alt="Objects" title="Objects" width="21" height="20" src="/en/docs/mt5/android/img/objects_icon.png"> in the radial menu. Tap <img class="help" alt="Delete" title="Delete" width="15" height="20" src="/en/docs/mt5/android/img/delete_button.png"> at the top of the window, select the objects to delete, and then tap <img class="help" alt="Delete" title="Delete" width="15" height="20" src="/en/docs/mt5/android/img/delete_button.png">.</span></p><p class="p_Image"><img class="help" alt="Deleting a single object" title="Deleting a single object" width="650" height="636" src="/en/docs/mt5/android/img/object_delete.png"></p><h2><a name="properties" class="hmanchor"></a>Object Properties <a class="anchor" href="objects#properties">#</a></h2><p class="p_txt"><span class="f_txt">Every object has its specific properties. Properties can differ depending on the object. In order to control the properties of an object, tap on the object to select it, and then long-tap on it. A menu will be opened in the upper part of the window, where you should tap <img class="help" alt="Edit Object" title="Edit Object" width="20" height="20" src="/en/docs/mt5/android/img/object_edit_button.png">.</span></p><p class="p_Image"><img class="help" alt="Object Properties" title="Object Properties" width="650" height="636" src="/en/docs/mt5/android/img/object_properties.png"></p><p class="p_txt"><span class="f_txt">The following is available here:</span></p><ul><li class="p_ol"><a name="name" class="hmanchor"></a><span class="f_ol" style="font-weight: bold;">Name</span><span class="f_ol"> — the unique name of an object within one chart, it is set automatically. It can be changed if another name is entered in this field. Such names make it easy to find an object among many other objects of the same type;</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;">Ray right/left</span><span class="f_ol"> — displaying trend lines as rays in the specified directions;</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;">Point</span><span class="f_ol"> — one or several snap points along the time and price axes.</span></li><li class="p_ol"><a name="symbol" class="hmanchor"></a><span class="f_ol" style="font-weight: bold;">Symbol</span><span class="f_ol"> — selecting a symbol, the object will be displayed on the charts of this symbol. If you select "All symbols", then the object will be displayed on any charts. However, please note that most of the objects are linked to prices, while prices of different symbols can vary greatly. Therefore, an object built on one symbol chart can be far beyond the visibility area after you switch to another symbol. Objects with the "all symbols" display type, are shown under a separate category "Independent charts" in the <a href="/en/docs/mt5/android/chart/objects#modify" class="topiclink">common list of objects</a>.</span></li><li class="p_ol"><a name="timeframe" class="hmanchor"></a><span class="f_ol" style="font-weight: bold;">Timeframe</span><span class="f_ol"> — selecting timeframes, on which the object will be displayed.</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;">Style</span><span class="f_ol"> — object lines style. Color, style and width of lines can be set here.</span></li></ul><ul><li class="p_ol"><span class="f_ol" style="font-weight: bold;">Draw as background</span><span class="f_ol"> — draw the object in the background layer, behind the chart.</span></li></ul><ul><li class="p_ol"><span class="f_txt" style="font-weight: bold;">Levels</span><span class="f_txt"> </span><span class="f_ol">—</span><span class="f_txt"> specific display settings for the levels of Fibonacci tools and Andrews' Pitchfork.</span></li></ul><p class="p_txt"><span class="f_txt">Some object have specific properties like "Gann Angle", "Pips per bar". etc. Description of these properties in available in corresponding <a href="/en/docs/mt5/android/chart/objects" class="topiclink">object</a> sections.</span></p></td></tr></tbody></table>
