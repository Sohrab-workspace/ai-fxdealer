<table class="container" cellspacing="0" cellpadding="0" border="0"><tbody><tr class="container"><td class="container" colspan="2" style="width:50%;"><h1>Account Connection</h1><p class="p_txt"><span class="f_txt">A login and password are required for connection to a trading account. Two types of account access are available in MetaTrader 5 for Android: Master and Investor.</span></p><p class="p_txt"><span class="f_txt">Being authorized with a master password, a user can access the entire functionality of the platform. When using investor authentication, a user can view the account state and analyze prices, but has no access to performing trading operation. Investor access is a convenient tool for demonstrating the trading process on an account.</span></p><div><div class="parent-table"><table class="attentable" cellspacing="0" cellpadding="8" border="1"><tbody><tr class="attentable"><td class="attentable"><p class="p_tableatten"><span class="f_tableatten">If <a href="/en/docs/mt5/android/settings_accounts/extended_authorization" class="topiclink">extended authentication</a> mode is used for the account, certificate password should additionally be specified during the connection. The certificate must be <a href="/en/docs/mt5/android/settings_accounts/extended_authorization#mobile_side" class="topiclink">installed</a> on the device.</span></p></td></tr></tbody></table></div></div><p class="p_txt"><span class="f_txt">To connect to an account, tap <img class="help" alt="Open an Account" title="Open an Account" width="20" height="20" src="/en/docs/mt5/android/img/account_add_button.png"> on the <a href="/en/docs/mt5/android/settings_accounts" class="topiclink">account management</a> page. Next, select the server (brokerage company) on which you have opened your account.</span></p><p class="p_txt"><span class="f_txt">By default, the list does not display all available servers. To find the desired broker server, start typing its name into the search bar. First search results appear after you type three characters. You may contact your broker to find out the appropriate server name.</span></p><p class="p_txt"><span class="f_txt">After selecting the server, specify the connection data in the "Use Existing Account" block:</span></p><ul><li class="p_ol"><span class="f_ol" style="font-weight: bold;">Login</span><span class="f_ol"> </span><span class="f_txt">— the number of the account used for connection.</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;">Password</span><span class="f_ol"> </span><span class="f_txt">— the master of investor password for the account. If you have forgotten your password or are experiencing other issues trying to connect to your account, please contact your broker. To open your broker's contact details, tap "Forgot your password?" below the connection form.</span></li><li class="p_ol"><span class="f_ol" style="font-weight: bold;">Server</span><span class="f_ol"> </span><span class="f_txt">— the name of the server for connection</span><span class="f_txt">.</span></li></ul><p class="p_Image"><img class="help" alt="Select a Server" title="Select a Server" width="326" height="596" src="/en/docs/mt5/android/img/demo_account_server.png"><span class="f_Image"> <img class="help" alt="New Account" title="New Account" width="326" height="596" src="/en/docs/mt5/android/img/new_account.png"></span></p><p class="p_txt"><span class="f_txt">To log in, tap "Login" at the bottom of the window.</span></p><div><div class="parent-table"><table class="attentable" cellspacing="0" cellpadding="8" border="1"><tbody><tr class="attentable"><td class="attentable"><p class="p_tableatten"><a name="save" class="hmanchor"></a><span class="f_tableatten">Enable the "Save password" option, and the next time you will be authorized automatically without having to enter the password. If you disable it, every time you will need to type in your password again.</span></p></td></tr></tbody></table></div></div></td></tr><tr class="container"><td class="container" style="width:50%;"><p class="p_Image"><img class="help" alt="Quickly move accounts from Desktop to Mobile using QR codes" title="Quickly move accounts from Desktop to Mobile using QR codes" width="326" height="634" src="/en/docs/mt5/android/img/qr_code.png"></p></td><td class="container" style="width:50%;"><h2><a name="qr" class="hmanchor"></a>Sign In with QR Code <a class="anchor" href="account_connect#qr">#</a></h2><p class="p_txt"><span class="f_txt">When you <a href="https://www.metatrader5.com/en/terminal/help/startworking/acc_open" target="_blank" class="weblink">open an account in the desktop platform</a>, a QR code is generated for you. Use this code for quick connection to your account from a mobile application.</span></p><p class="p_txt"><span class="f_txt">Tap <img class="help" alt="QR code" title="QR code" width="20" height="20" src="/en/docs/mt5/android/img/qr_button.png"> on the server list page. Then point your camera at the QR code, and the trading account will be instantly connected, without the need to specify login, password and server values.</span></p></td></tr><tr class="container"><td class="container" colspan="2" style="width:50%;"><h2><a name="contest" class="hmanchor"></a>Contest Accounts <a class="anchor" href="account_connect#contest">#</a></h2><p class="p_txt"><span class="f_txt">The platform offers a special account type, which can be used for hosting various trading contests and competitions.
