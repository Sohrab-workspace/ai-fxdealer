# MTBook/MTBookDiff

<table class="attentable" cellspacing="0" cellpadding="8" border="1"><tbody><tr class="attentable"><td class="attentable"><p class="p_tableatten"><span class="f_tableatten">The MTBook and MTBookDiff structures are identical. The difference is that the former is used in the Server API and Manager API, and the latter is used in the Gateway API. The name emphasizes the fundamental difference in passing the Depth of Market data. In the Server API Manager API, the entire Depth of Market is passed each time, while in the Gateway API only changes are passed.</span></p></td></tr></tbody></table>

The MTBook/MTBookDiff structure describes changes of the [Depth of Market](/en/docs/mt5/api/reference_dom). The structure is defined with the one-byte alignment.
