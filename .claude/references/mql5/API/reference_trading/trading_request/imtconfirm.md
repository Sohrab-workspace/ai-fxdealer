## How to perform request confirmation [#](imtconfirm#fill_confirmation)

When responding to any type of request, the following fields should be filled in IMTConfirm:

-   [IMTConfirm::ID](/en/docs/mt5/api/reference_trading/trading_request/imtconfirm/imtconfirm_id) — the ID of the request, in response to which the confirmation will be formed ([IMTRequest::ID](/en/docs/mt5/api/reference_trading/trading_request/imtrequest/imtrequest_id)).
-   [IMTConfirm::Retcode](/en/docs/mt5/api/reference_trading/trading_request/imtconfirm/imtconfirm_retcode) — response code for the request.
-   [IMTConfirm::Comment](/en/docs/mt5/api/reference_trading/trading_request/imtconfirm/imtconfirm_comment) — an optional field. If a comment is specified, it will be shown to the client in the trading dialog instead of the standard message corresponding to the response code.
