Parameters

index

\[in\]  Position of a deal in an array, starting with 0.

Return Value

If successful, it returns a pointer to the path to the object of a deal at the specified position. Otherwise, it returns NULL.

Note

The lifetime of the returned object is controlled by the current array object. Thus, when deleting an array object, the returned pointer will be invalid.

Removal of the returned object (by calling [IMTDealArray::Release](/en/docs/mt5/api/reference_trading/trading_deal/imtdealarray/imtdealarray_release)) will make the pointer stored within the array object invalid, and its call (including the case of array object deletion) will cause the crash of the plugin and the server.
