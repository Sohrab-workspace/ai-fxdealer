Return Value

If successful, it returns a pointer to a string with the name of the data feed module. Otherwise, it returns NULL.

Note

It returns the name of the module (exe-file) that corresponds to the data feed.

The pointer to the resulting string is valid for the lifetime of the [IMTConFeeder](/en/docs/mt5/api/config_datafeeds/imtconfeeder) object.

To use the string after the object removal (call of the [IMTConFeeder::Release](/en/docs/mt5/api/config_datafeeds/imtconfeeder/imtconfeeder_release) method of this object), a copy of it should be created.
