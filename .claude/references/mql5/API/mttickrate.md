# MTTickRate

This structure describes brief information about a tick. It differs from the [MTTick](/en/docs/mt5/api/mttick) abd [MTTickShort](/en/docs/mt5/api/mttickshort) structures by a reduced number of fields. The structure is defined with the one-byte alignment.
