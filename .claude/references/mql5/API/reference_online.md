# Online Connections

The MetaTrader 5 API provides interfaces for receiving information about current connections to the trade server. All types of connection are considered, including client, manager and API ones with the exception of the cluster components' (platform servers') connections.

The interfaces provide access to the parameters of the current connections to the trade server. The following interfaces are available:

-   [IMTOnline](/en/docs/mt5/api/reference_online/imtonline)  
    An interface that provides access to all the main connection parameters.
-   [IMTOnlineArray](/en/docs/mt5/api/reference_online/imtonlinearray)  
    An interface for working with the arrays of connection records.
