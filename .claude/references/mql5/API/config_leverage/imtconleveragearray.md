## Specifics of Array Operations

Please mind the following points while working with arrays:

-   Arrays store pointers to the appropriate interfaces, not the data. This determines specific operation features when [adding](/en/docs/mt5/api/config_leverage/imtconleveragearray/imtconleveragearray_add), [update](/en/docs/mt5/api/config_leverage/imtconleveragearray/imtconleveragearray_update), and [deleting](/en/docs/mt5/api/config_leverage/imtconleveragearray/imtconleveragearray_delete) array elements.
-   Never add a reference to the same object into an array (when [adding](/en/docs/mt5/api/config_leverage/imtconleveragearray/imtconleveragearray_add) or [updating](/en/docs/mt5/api/config_leverage/imtconleveragearray/imtconleveragearray_update) an element), as this will lead to a crash when freeing the memory.
