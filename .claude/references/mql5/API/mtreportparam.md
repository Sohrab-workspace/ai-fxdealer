# MTReportParam

This structure is used for describing a reports module parameters. It describes both reports request parameters from a manager terminal and module parameters that are set during the configuration in MetaTrader 5 Administrator. Requests parameters are passed through the params field and external parameters are passed through the config field of the [MTReportInfo](/en/docs/mt5/api/mtreportinfo) structure. The MTReportParam structure is defined with the one-byte alignment.
