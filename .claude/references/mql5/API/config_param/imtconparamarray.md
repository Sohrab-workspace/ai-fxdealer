## Working with Arrays

There are a number of specific features for working with arrays:

-   Arrays store pointers to the appropriate interfaces, and not the data. This leads to some operation peculiarities when [adding](/en/docs/mt5/api/config_param/imtconparamarray/imtconparamarray_add), [updating](/en/docs/mt5/api/config_param/imtconparamarray/imtconparamarray_update) and [removing](/en/docs/mt5/api/config_param/imtconparamarray/imtconparamarray_delete) array elements.
-   Never add a link (when [adding](/en/docs/mt5/api/config_param/imtconparamarray/imtconparamarray_add) or [updating](/en/docs/mt5/api/config_param/imtconparamarray/imtconparamarray_update) an element) to one and the same object in an array, because this will lead to a crash during memory release.
