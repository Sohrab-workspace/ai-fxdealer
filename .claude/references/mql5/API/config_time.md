# Time Configuration

The MetaTrader 5 API allows to manage global time settings of a trading platform: adjust working time, retrieve and change the time zone and the address of the server for synchronizing time.

The following time configuration interfaces are available:

-   [IMTConTime](/en/docs/mt5/api/config_time/imtcontime)  
    An interface for getting and setting the time parameters of a platform.
-   [IMTConTimeSink](/en/docs/mt5/api/config_time/imtcontimesink)  
    An interface for handling events associated with change of time settings.

The below figure shows different elements of time configuration in the MetaTrader 5 Administrator, to help you understand the purpose of the interfaces:

![Time settings in the MetaTrader 5 Administrator](/en/docs/mt5/api/img/time.png "Time settings in the MetaTrader 5 Administrator")

The following elements are shown above:

1.  [The time zone of a platform](/en/docs/mt5/api/config_time/imtcontime/imtcontime_timezone).
2.  [The daylight saving time option](/en/docs/mt5/api/config_time/imtcontime/imtcontime_daylight).
3.  [The address of a server for synchronizing time](/en/docs/mt5/api/config_time/imtcontime/imtcontime_timeserver).
4.  [The platform operation schedule](/en/docs/mt5/api/config_time/imtcontime/imtcontime_timetableget).
