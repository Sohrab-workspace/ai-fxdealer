# Manager Interface IMTManagerAPI

The IMTManagerAPI interface provides access to the command of the MetaTrader 5 Manager Terminal. Using this interface in an application, you can process trade requests that arrive at the trade server, work with a database of users and the price stream.

<table class="attentable" cellspacing="0" cellpadding="8" border="1"><tbody><tr class="attentable"><td class="attentable"><p class="p_tableatten"><span class="f_tableatten">To be able to use the IMTManagerAPI interface, a manager account that is used for <a href="/en/docs/mt5/api/imtmanagerapi/imtmanagerapi_connection/imtmanagerapi_connect" class="topiclink">connecting to the server</a> must have the "Connect using MetaTrader 5 Manager" permission (<a href="/en/docs/mt5/api/config_manager/imtconmanager/imtconmanager_enum#enmanagerrights" class="topiclink">IMTConManager::RIGHT_MANAGER</a>).</span></p></td></tr></tbody></table>
