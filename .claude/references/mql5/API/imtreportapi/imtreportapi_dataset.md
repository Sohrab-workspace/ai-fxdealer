# Data set

Data sets are used for preparing source data for the creation of [dashboards](/en/docs/mt5/api/reportapi_dashboards) and [tabular reports](/en/docs/mt5/api/reportapi_tables). Information in a set is stored as a table with a set of columns and rows and a special summary row.
