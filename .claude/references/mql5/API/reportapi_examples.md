# Ready-made Examples

MetaTrader 5 Report API installation packet contains several ready-made examples of reports in source codes. Their analysis allows a developer to quickly learn the details of using Report API and start creating custom reports.

Report examples can be found in the [/Examples](/en/docs/mt5/api/files_folders_structure#exmaples) folder of the directory where MetaTrader 5 Report API is installed.

-   Accounts.Standard.Reports — this example demonstrates possibilities for [HTML reports](/en/docs/mt5/api/reportapi_html) generation using a custom template and also for realization of graphic expression in the form of SVG chart It includes the following reports:

-   [Accounts Groups](/en/docs/mt5/api/reportapi_examples/accounts_groups_report) — statistical report on accounts in groups on a server.
-   [Accounts Grow](/en/docs/mt5/api/reportapi_examples/accounts_growth_report) — report on the growth of the number of accounts in groups on a server.
-   [Accounts Lifetime by Countries](/en/docs/mt5/api/reportapi_examples/accounts_lifetime_report) — lifetime of accounts with a distribution by countries.

-   Capital.Standard.Reports — example of financial reports on clients:

-   [Money Flow Daily](/en/docs/mt5/api/reportapi_examples/money_flow_daily_report) — a group of reports presenting the movement of funds on client accounts.
-   [Money Flow Weekly](/en/docs/mt5/api/reportapi_examples/money_flow_weekly_report) — a group of reports representing the weekly movement of funds on client accounts.
-   [Lifetime Value](/en/docs/mt5/api/reportapi_examples/lifetime_value_report) — a group of reports concerning client LTVs.
-   [First Time Deposit](/en/docs/mt5/api/reportapi_examples/first_time_deposit_report) — a group of reports on first-time deposits on client accounts.
-   [Retention of Clients](/en/docs/mt5/api/reportapi_examples/retention_of_clients_report) — new client retention report.
-   [Retention of Trading Accounts](/en/docs/mt5/api/reportapi_examples/retention_of_trading_accounts_report) — new client retention report by individual accounts.

-   Trades.Standard.Reports — the source code that contains [tabular reports](/en/docs/mt5/api/reportapi_tables) generation examples included into the MetaTrader 5 platform:

-   [Agent](/en/docs/mt5/api/reportapi_examples/agents_report) — report on all transactions connected with charging agent commissions.

-   [Agents Detailed](/en/docs/mt5/api/reportapi_examples/agents_detailed_report) — detailed report on agent commissions.

-   [Credit Facility](/en/docs/mt5/api/reportapi_examples/credit_facility_report) — credit operations report for the selected period.
-   [Daily Report](/en/docs/mt5/api/reportapi_examples/daily_report) — report on the financial state of the requested accounts as of the end of a day.
-   [Daily Detailed](/en/docs/mt5/api/reportapi_examples/daily_detailed_report) — detailed report on the financial state of the one selected account as of the end of the day.
-   [Deals History](/en/docs/mt5/api/reportapi_examples/deals_history_report) — summary report on the deals for the selected period.
-   [Deals Profit](/en/docs/mt5/api/reportapi_examples/deals_profit_report) — detailed report on deals for the selected period.
-   [Deals Initiators](/en/docs/mt5/api/reportapi_examples/deals_initiators_report) — deal initiators (reasons) report.
-   [Deals Geography](/en/docs/mt5/api/reportapi_examples/deals_geography_report) — trading activity distribution by country.
-   [Deals Weekly](/en/docs/mt5/api/reportapi_examples/deals_weekly_report) — trader activity report.
-   [Deposit and Withdrawal](/en/docs/mt5/api/reportapi_examples/deposit_withdrawal_report) — report on deposit and withdrawal operations for the selected period.
-   [Equity](/en/docs/mt5/api/reportapi_examples/equity_report) — report on the financial state of the requested accounts as at the end of the day. The accounts are grouped according to their "Comment" field value. This report allows to generate reports on clients' accounts grouping them by custom criteria.

-   [Execution Types](/en/docs/mt5/api/reportapi_examples/execution_type_report) — report on the number, volume and type of executed trades.

-   [Margin Calls](/en/docs/mt5/api/reportapi_examples/margin_call_report) — report on the state of margin call or stop out accounts.
-   [Orders History](/en/docs/mt5/api/reportapi_examples/orders_history_report) — summary report on the orders for the selected period.

-   [Positions History](/en/docs/mt5/api/reportapi_examples/positions_history_report) — summary report on positions for the selected period.

-   [Risk Appetite](/en/docs/mt5/api/reportapi_examples/risk_appetite_report) — a report reflecting the risk level accepted by traders.
-   [Segregated](/en/docs/mt5/api/reportapi_examples/segregated_report) — summary report on the change of financial state of the requested accounts for the specified period.
-   [Summary](/en/docs/mt5/api/reportapi_examples/summary_report) — summary motion of funds and funds cycles for the selected period.

-   [Trade Accounts](/en/docs/mt5/api/reportapi_examples/trade_accounts_report) — report on the current trading statuses of client accounts.

-   [Trade Modifications](/en/docs/mt5/api/reportapi_examples/trade_modification_report) — report on trade operations manually changed by an administrator, manager or API.
-   [Trade Performance Summary](/en/docs/mt5/api/reportapi_examples/performance_summary_report) — detailed trading statistics of trading accounts.
-   [Trade Group Statistics](/en/docs/mt5/api/reportapi_examples/group_statistics_report) —  detailed statistics of groups: the number of accounts, commission amounts, share of trades by type, and more.

-   [StopOut Compensations](/en/docs/mt5/api/reportapi_examples/stopouts_compensation_report) — report on operations associated with the compensation of a negative balance in the situation of Stop out. All trades with the "so compensation" type are included into this report.
-   [Fast Profit Deals](/en/docs/mt5/api/reportapi_examples/fast_profit_deals_report) — this report is designed to detect traders exploiting arbitrage opportunities through quote delays

-   Daily.Standard.Reports — the source code that contains daily reports generation examples included into the MetaTrader 5 platform:

-   [Daily Dealing](/en/docs/mt5/api/reportapi_examples/daily_dealing_report) — daily report on the activity of dealers.
-   [Daily Server Logs](/en/docs/mt5/api/reportapi_examples/daily_server_report) — statistical report on the operation of the platform for the specified day.
-   [Daily Trades](/en/docs/mt5/api/reportapi_examples/daily_trade_report) — report on trade operations for the specified day.

-   [Daily Orders](/en/docs/mt5/api/reportapi_examples/daily_orders_report) — report on client orders open at the end of the selected day.
-   [Daily Positions](/en/docs/mt5/api/reportapi_examples/daily_positions_report) — report on client positions open at the end of the selected day.

-   [Daily Expert Advisors](/en/docs/mt5/api/reportapi_examples/daily_expert_report) — report on trading operations performed by Expert Advisors of clients.

-   Gateway.Standard.Reports — the source code that contains examples of several reports on gateways:

-   [Gateways Turnover](/en/docs/mt5/api/reportapi_examples/gateways_turnover_report) — report on volumes of the deals handled by gateways.
-   [Gateways Profit](/en/docs/mt5/api/reportapi_examples/gateways_profit_report) — report on volumes of the deals handled by gateways, as well as the profit earned by a broker when handling these deals.
-   [Gateways White Label](/en/docs/mt5/api/reportapi_examples/gateways_wl_report) — report on volumes of the deals performed on a trading server by external brokers via MetaTrader 5 Gateway (standard or White Label version).

-   Trades.Transaction.Reports — the source code that contains an example of the trade transaction report:

-   [Trade Transactions](/en/docs/mt5/api/reportapi_examples/trade_transaction_report) — provides detailed data on trading operations on MetaTrader 5 servers.

-   [NFA](/en/docs/mt5/api/reportapi_examples/nfa_report) — the source code that contains an example of reports to be submitted to NFA regulator.

-   [EMIR](/en/docs/mt5/api/reportapi_examples/emir_report) — the source code that contains an example of a report to be submitted to regulator according to EMIR.

-   [Fund Overview](/en/docs/mt5/api/reportapi_examples/fund_overview_report) — a report reflecting the key characteristics of a hedge fund.
