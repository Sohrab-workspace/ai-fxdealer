# Administrator Interface IMTAdminAPI

The IMTAdminAPI interface provides access to the command of the MetaTrader 5 Administrator Terminal. Using this interface in an application, you can manage settings of a trading platform, work with databases of users, mails, prices and trade operations.

<table class="attentable" cellspacing="0" cellpadding="8" border="1"><tbody><tr class="attentable"><td class="attentable"><p class="p_tableatten"><span class="f_tableatten">To be able to use the IMTAdminAPI interface, a manager account that is used for <a href="/en/docs/mt5/api/imtadminapi/imtadminapi_connection/imtadminapi_connect" class="topiclink">connecting to the server</a> must have the "Connect using MetaTrader 5 Administrator" permission (<a href="/en/docs/mt5/api/config_manager/imtconmanager/imtconmanager_enum#enmanagerrights" class="topiclink">IMTConManager::RIGHT_ADMIN</a>).</span></p></td></tr></tbody></table>
