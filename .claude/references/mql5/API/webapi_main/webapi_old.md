# Outdated Rest API version

Starting with MetaTrader 5 build 2920, all Web API requests (endpoints) have been renamed. The new well-structured form enables convenient use and potential expansion.

The documentation describes the current (new) Web API version. However, the support for old names will be continued for some time to provide compatibility.

The below table shows old and new endpoints. We strongly recommend that you upgrade your servers to the current release version, and then upgrade all of your Web API clients to use only the new request names. Support for old endpoints will be disabled after a while.
