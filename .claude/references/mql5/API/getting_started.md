# Getting Started

This section contains basic information you need to know to get started with the MetaTrader 5 API.

-   [System requirements](/en/docs/mt5/api/system_requirements)  
    To develop your own application using the API, your computer must meet the minimum system requirements.
-   [Installation](/en/docs/mt5/api/installation)  
    This section will consider the process of installing a set of MetaTrader 5 API tools.
-   [The structure of directories and files](/en/docs/mt5/api/files_folders_structure)  
    This section describes the location of directories and files of MetaTrader 5 API tools.
-   [Uninstall](/en/docs/mt5/api/deinstallation)  
    If necessary, the API can be uninstalled from a computer.
