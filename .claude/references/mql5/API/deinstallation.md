# Delete

In order to start the process of uninstallation of MetaTrader 5 API, go to its [installation](/en/docs/mt5/api/installation) directory and run "unins000.exe".

![Delete](/en/docs/mt5/api/img/deinstallation.png "Delete")

To uninstall it, you must confirm your choice.
