Return Value

If successful, it returns a pointer to a string with the full name of the platform. Otherwise, it returns NULL.

Note

The full name of the platform is formed of [IMTConCommon::OwnerID](/en/docs/mt5/api/config_common/imtconcommon/imtconcommon_ownerid) - [IMTConCommon::Name](/en/docs/mt5/api/config_common/imtconcommon/imtconcommon_name).

A pointer to the resulting string is valid for the lifetime of the [IMTConCommon](/en/docs/mt5/api/config_common/imtconcommon) object.

To use the string after the object removal (call of the [IMTConCommon::Release](/en/docs/mt5/api/config_common/imtconcommon/imtconcommon_release) method of this object), a copy of it should be created.

The maximum name length is 64 characters (including the end-of-line character).
