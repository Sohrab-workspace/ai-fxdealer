# IMTConGroupSymbol::VolumeMinDefault

Get the default value of [the minimum volume](/en/docs/mt5/api/config_symbol/imtconsymbol/imtconsymbol_volumemin) of trade operations for a symbol. For a more detailed description, please read the ["Use of Default method"](/en/docs/mt5/api/config_group/imtcongroupsymbol#default) section.
