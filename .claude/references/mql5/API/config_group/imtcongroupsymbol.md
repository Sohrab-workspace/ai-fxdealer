## Use of Default methods [#](imtcongroupsymbol#default)

Settings of trading instruments are defined by properties from [IMTConSymbol](/en/docs/mt5/api/config_symbol/imtconsymbol). They include basic symbol settings used for all clients by default. The platform allows overriding symbol settings for each client group. This can be done using the IMTConGroupSymbol class. In addition to Get and Set method used for working with overridden settings, the class also features Default methods, such as [IMTConGroupSymbol::TradeModeDefault](/en/docs/mt5/api/config_group/imtcongroupsymbol/imtcongroupsymbol_trademodedefault). These methods can be used for the following purposes:

-   Checking if a certain symbol property has a default value for the group
-   Setting symbol properties to default for the group
