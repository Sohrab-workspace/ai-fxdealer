## Specific Array Operations

There are a number of specific features for operations with arrays:

-   Arrays store pointers to the appropriate interfaces, not the data itself. This requires some specific behavior [adding](/en/docs/mt5/api/config_symbol/imtconsymbolarray/imtconsymbolarray_add), [updating](/en/docs/mt5/api/config_symbol/imtconsymbolarray/imtconsymbolarray_update) and [deleting](/en/docs/mt5/api/config_symbol/imtconsymbolarray/imtconsymbolarray_delete) array elements.
-   Please be sure to never add a link to one and the same object within an array (when [adding](/en/docs/mt5/api/config_symbol/imtconsymbolarray/imtconsymbolarray_add) or [updating](/en/docs/mt5/api/config_symbol/imtconsymbolarray/imtconsymbolarray_update) an element), as this will lead to a crash during memory release.
