To ensure proper operation of these functions, the Manager API needs to have access to actual quotes of financial instruments used in calculations. To receive quotes of an instrument, add it to the list of selected symbols using the [IMTManagerAPI::SelectedAdd](/en/docs/mt5/api/imtmanagerapi/imtmanagerapi_selected/imtmanagerapi_selectedadd) or [IMTManagerAPI::SelectedAddAll](/en/docs/mt5/api/imtmanagerapi/imtmanagerapi_selected/imtmanagerapi_selectedaddall) method. Please note that these methods work asynchronously, i.e. they only send a request for a subscription to the symbols. The prices do not become available to the application immediately after the call. To make sure that the application uses actual prices, call auxiliary functions after receiving the [IMTTickSink::OnTick](/en/docs/mt5/api/reference_ticks/imtticksink/imtticksink_ontick) event for the corresponding symbol. To receive such events, you need to subscribe to them using the [IMTManagerAPI::TickSubscribe](/en/docs/mt5/api/imtmanagerapi/imtmanagerapi_tick/imtmanagerapi_ticksubscribe) method.
