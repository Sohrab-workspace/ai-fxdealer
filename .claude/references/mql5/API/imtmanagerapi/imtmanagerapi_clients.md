# Clients

The MetaTrader 5 Manager API allows managing a database of clients on a trade server. Using the API, you can add or remove records, edit data and handle database change events. Use the API to expand the standard functionality of client management system in the platform, or to integrate it with external CRM systems.

A detailed description of operations with clients is provided in the [MetaTrader 5 Administrator documentation](https://support.metaquotes.net/en/docs/mt5/platform/administration/clients).

<table class="attentable" cellspacing="0" cellpadding="8" border="1"><tbody><tr class="attentable"><td class="attentable"><p class="p_tableatten"><span class="f_tableatten">The Manager API only allows managing the clients, which are available to the manager account used by the application to <a href="/en/docs/mt5/api/imtadminapi/imtadminapi_connection/imtadminapi_connect" class="topiclink">connect to the server</a>. A client record is available to the manager if one of the following conditions is met:</span></p><ul><li class="p_tableatten"><span class="f_tableatten">The client is created by this manager</span></li><li class="p_tableatten"><span class="f_tableatten">The client is explicitly assigned to the manager (<a href="/en/docs/mt5/api/reference_client/imtclient/imtclient_assignedmanager" class="topiclink">IMTClient::AssignedManager</a>)</span></li><li class="p_tableatten"><span class="f_tableatten">The preferred trading group (<a href="/en/docs/mt5/api/reference_client/imtclient/imtclient_tradinggroup" class="topiclink">IMTClient::TradingGroup</a>) set for the client is available to the manager (the manager has <a href="/en/docs/mt5/api/config_manager/imtconmanager/imtconmanager_groupadd" class="topiclink">permissions for the group</a>)</span></li><li class="p_tableatten"><span class="f_tableatten">Any of the <a href="/en/docs/mt5/api/imtadminapi/imtadminapi_clients/imtadminapi_clientuseradd" class="topiclink">trading accounts bound to the client</a> is available to the manager (the manager has <a href="/en/docs/mt5/api/config_manager/imtconmanager/imtconmanager_groupadd" class="topiclink">permissions for the group</a> in which the account is located)</span></li></ul></td></tr></tbody></table>
