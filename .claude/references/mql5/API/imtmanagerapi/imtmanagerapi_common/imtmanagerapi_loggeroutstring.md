Fast output method which prints unformatted logs to the Manager API local journal.

Parameters

code

\[in\]  Log code which is passed using the [EnMTLogCode](/en/docs/mt5/api/journal#enmtlogcode) enumerations.

string

\[in\]  Log string with optional arguments. 

Return Value

An indication of a successful execution is the [MT\_RET\_OK](/en/docs/mt5/api/retcodes_successful) response code. Otherwise, an error has occurred, which corresponds to the response code.

Note

Compared to [IMTManagerAPI::LoggerOut](/en/docs/mt5/api/imtmanagerapi/imtmanagerapi_common/imtmanagerapi_loggerout), which formats the output, this method consumes less resources.

Each string is limited to 16KB, not including the standard string header.

Do not use the newline characters \\r and \\n, as this will lead to incorrect calculation of the checksum for the log lines.
