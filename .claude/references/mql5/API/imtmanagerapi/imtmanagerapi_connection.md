# Connecting to the Server

This section contains descriptions of methods for connecting and disconnecting an application from the server. The application is connected to the platform in one of the [pumping mode](/en/docs/mt5/api/imtmanagerapi/imtmanagerapi_connection/imtmanagerapi_enpumpmodes), which indicates what information the application will continuously receive from the server.
