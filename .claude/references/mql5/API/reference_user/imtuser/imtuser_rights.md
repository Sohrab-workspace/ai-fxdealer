Parameters

rights

\[in\]  To pass the user permissions, the [IMTUser::EnUsersRights](/en/docs/mt5/api/reference_user/imtuser/imtuser_enum#enusersrights) enumeration is used.

Return Value

An indication of a successful execution is the [MT\_RET\_OK](/en/docs/mt5/api/retcodes_successful) response code. Otherwise, an error has occurred, which corresponds to the response code.

Note

Rights are verified and set using bit operations & ("AND") and | ("OR").
