## Working with Arrays

There are a number of specific features for working with arrays:

-   Arrays store pointers to the appropriate interfaces, and not the data. This leads to some operation peculiarities when [adding](/en/docs/mt5/api/reference_online/imtonlinearray/imtonlinearray_add), [updating](/en/docs/mt5/api/reference_online/imtonlinearray/imtonlinearray_update) and [removing](/en/docs/mt5/api/reference_online/imtonlinearray/imtonlinearray_delete) array elements.
-   Never add a link (when [adding](/en/docs/mt5/api/reference_online/imtonlinearray/imtonlinearray_add) or [updating](/en/docs/mt5/api/reference_online/imtonlinearray/imtonlinearray_update) an element) to one and the same object in an array, because this will lead to a crash during memory release.
