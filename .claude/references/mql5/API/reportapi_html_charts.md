# Charts

The MetaTrader 5 Report API allows to add SVG charts into [HTML reports](/en/docs/mt5/api/reportapi_html) and [dashboards](/en/docs/mt5/api/reportapi_dashboard/imtreportdashboardhtml). A chart consists of two basic elements:

-   Headers series — a data series that is displayed in the form of labels to the one of the chart's axes;
-   Values series — data series that are displayed directly on the chart.

Note that SVG charts are supported beginnig from the Internet Explorer 9.</span></p><p class="p_tableatten"><span class="f_tableatten">On default, Internet Explorer that is used in the manager terminal works in the 7 version compatibility mode. For the SVG chart to be displayed correctly, force it to use the Internet Explorer 9 compatibility mode. To do it, in the <a href="/en/docs/mt5/api/reportapi_html_charts#html" class="topiclink">heading of the HTML page</a> (within the pair of tags &lt;head&gt;&lt;/head&gt;), include the following line:</span></p><p class="p_tableatten"><span class="f_li">&lt;meta http-equiv=\"X-UA-Compatible\" content=\"IE=9\" /&gt;</span></p></td></tr></tbody></table>

An example of working with an SVG chart. The following actions must be performed to add the charts into the report:

-   Add a new CMyChartTable report class equivalent to the way it is done for a [HTML report](/en/docs/mt5/api/reportapi_html#class_add).
-   Realize the class equivalent to the way it is done for a [HTML report](/en/docs/mt5/api/reportapi_html#class_implement).
-   [Set report parameters](/en/docs/mt5/api/reportapi_html_charts#description) transferred in the MTReportInfo structure.
-   Fill the entry points equivalent to the way it is done for a [HTML report](/en/docs/mt5/api/reportapi_html#exported_functions).
-   [Add the charts realization](/en/docs/mt5/api/reportapi_html_charts#implementation) into the report code.

This section covers only working with charts.
