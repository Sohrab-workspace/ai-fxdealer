## Operations with Arrays

There are a number of specific features for operations with arrays:

-   Arrays store pointers to the appropriate interfaces, and not the data. This sets some specific features when [adding](/en/docs/mt5/api/reference_client/imtdocumentarray/imtdocumentarray_add), [updating](/en/docs/mt5/api/reference_client/imtdocumentarray/imtdocumentarray_update) and [deleting](/en/docs/mt5/api/reference_client/imtdocumentarray/imtdocumentarray_delete) array elements.
-   Never add a link to one and the same object in an array (when [adding](/en/docs/mt5/api/reference_client/imtdocumentarray/imtdocumentarray_add) or [updating](/en/docs/mt5/api/reference_client/imtdocumentarray/imtdocumentarray_update) an element), because this will lead to a crash during memory release.
