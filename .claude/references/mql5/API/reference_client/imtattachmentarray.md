## Working with Arrays

Please note the following specific features when working with arrays:

-   Arrays store pointers to the appropriate interfaces, and not the data. This causes some specific features when [adding](/en/docs/mt5/api/reference_client/imtattachmentarray/imtattachmentarray_add), [updating](/en/docs/mt5/api/reference_client/imtattachmentarray/imtattachmentarray_update) and [deleting](/en/docs/mt5/api/reference_client/imtattachmentarray/imtattachmentarray_delete) array elements.
-   Never add a link to one and the same object in an array (when [adding](/en/docs/mt5/api/reference_client/imtattachmentarray/imtattachmentarray_add) or [updating](/en/docs/mt5/api/reference_client/imtattachmentarray/imtattachmentarray_update) elements), because this will lead to a crash during memory release.
