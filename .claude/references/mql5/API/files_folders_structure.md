# Files and Folders

The [installation](/en/docs/mt5/api/installation) directory contains the following files and folders:

-   \\Docs — documentation CHM files.
-   \\Examples — examples for each API type.
-   \\Include — header files for each API type.
-   \\Libs — [Manager API](/en/docs/mt5/api/managerapi) and [Gateway API](/en/docs/mt5/api/gatewayapi) DLLs, including .NET versions
-   unins000.\* — MetaTrader 5 API uninstall program files.
