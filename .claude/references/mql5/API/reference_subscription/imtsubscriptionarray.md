## Operations with Arrays

There are a number of specific features for operations with arrays:

-   Arrays store pointers to the corresponding interfaces, not the data. This determines some specific features in operations when [adding](/en/docs/mt5/api/reference_subscription/imtsubscriptionarray/imtsubscriptionarray_add), [updating](/en/docs/mt5/api/reference_subscription/imtsubscriptionarray/imtsubscriptionarray_update) and [deleting](/en/docs/mt5/api/reference_subscription/imtsubscriptionarray/imtsubscriptionarray_delete) array elements.
-   Never add a link to one and the same object in an array (when [adding](/en/docs/mt5/api/reference_subscription/imtsubscriptionarray/imtsubscriptionarray_add) or [deleting](/en/docs/mt5/api/reference_subscription/imtsubscriptionarray/imtsubscriptionarray_update) elements), because this will lead to a crash during memory release.
