# IMTGatewayAPI::Allocate

Memory allocation by an application. It is paired to the [IMTGatewayAPI::Free](/en/docs/mt5/api/imtgatewayapi/imtgatewayapi_common/imtgatewayapi_free) method.

Parameters

bytes

\[in\]  Amount of allocated memory in bytes.

Return Value

If successful, it returns a pointer to the allocated memory block, otherwise it returns NULL.
