# Tick Data Functions

The MetaTrader 5 Gateway API features functions for the export and import of tick data to the platform. Unlike [SendTicks\*](/en/docs/mt5/api/imtgatewayapi/imtgatewayapi_send/imtgatewayapi_sendticks), these functions work directly with the history of ticks, rather than the price stream which is broadcast to clients in real time.
