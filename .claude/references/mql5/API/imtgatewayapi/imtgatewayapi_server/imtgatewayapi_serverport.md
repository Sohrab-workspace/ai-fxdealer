# IMTGatewayAPI::ServerPort

Get the number of the port launched for the connection to Gateway API.

Return Value

Server port number. In case several addresses (for example: address1:port1,address2:port2) have been specified via [IMTGatewayAPI::Start](/en/docs/mt5/api/imtgatewayapi/imtgatewayapi_server/imtgatewayapi_start) method, this method will return the port of the first address (port1).
