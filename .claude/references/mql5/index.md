# Welcome to the section of Documentation for the MetaTrader 5 platform

Here you can find the most actual documentation for all components of the platform.

## Managers

-   **[Trading platform](/en/docs/mt5/platform)**  
    The user guide includes the information about the platform [installation](/en/docs/mt5/platform/platform_installation) and [setup](/en/docs/mt5/platform/administration), it's [components](/en/docs/mt5/platform/components) and operation features.
-   [**Administrator terminal**  
    ](/en/docs/mt5/platform/administrator)It is intended for controlling operation of servers and other components of the platform, managing settings of [groups](/en/docs/mt5/platform/administration/admin_groups) of users, [manager](/en/docs/mt5/platform/administration/admin_managers) permissions, financial instruments and data feeds, setting up operation of access servers, managing [quotes](/en/docs/mt5/platform/administration/admin_charts) database, processing logs of server functioning, performing synchronization of history data of charts, restarting data feeds and servers, setting up automatic updates of programs, regulating [working time](/en/docs/mt5/platform/administration/admin_time), controlling function of database [reserving](/en/docs/mt5/platform/administration/admin_network/network_add_edit/network_backup_server).
-   [**Manager terminal**  
    ](/en/docs/mt5/manager)It is intended for managing [accounts](/en/docs/mt5/manager/accounts) and [positions](/en/docs/mt5/manager/positions) of clients, opening new accounts, processing clients' [trade requests](/en/docs/mt5/manager/dealing_risk_management), analyzing aggregate positions of clients and managing company's own positions, receiving full information about clients' operations, sending quotes to the data thread on the server, generating various types of [statements](/en/docs/mt5/manager/reports), sending and receiving messages over the internal mailing system, publishing news in client terminals.

## Clients

-   [**Client terminal**](/en/docs/mt5/client)  
    Allows executing [market deals](/en/docs/mt5/client/trading) and placing 6 types of pending orders. Has 82 built-in [analytical objects](/en/docs/mt5/client/charts_analysis) including technical indicators and objects. Allows creating, testing an using [trade robots](/en/docs/mt5/client/algotrading), indicators and scripts.
-   **[Mobile terminals for iPhone/iPad](/en/docs/mt5/iphone) and [Android](/en/docs/mt5/android)**  
    Allow trading, managing accounts, viewing charts and conducting technical analysis right from a mobile device. They support all types of market and pending orders, 30 indicators and 23 analytical objects. Furthermore, they allow exchanging instant messages with MQL5.community members.
-   [**MetaEditor**](/en/docs/mt5/metaeditor)  
    Intended for developing trade robots and indicators in MQL5 language. Provides a wide range of tools for working with source code: [intellectual control](/en/docs/mt5/metaeditor/development/source_code/intelligent_management), [debugging](/en/docs/mt5/metaeditor/development/debug), [profiling](/en/docs/mt5/metaeditor/development/profiling) and [MQL5 storage](/en/docs/mt5/metaeditor/mql5storage) - a built-in online storage for source codes with version control.
-   [**WebTerminal**](/en/docs/mt5/platform/components/web_terminal)  
    The MetaTrader 5 WebTerminal allows trading in the financial markets via a web browser. It works in all operating systems and browsers requiring no additional software. All transmitted data is securely encrypted.  
    

## API

-   [**Server API**  
    ](/en/docs/mt5/api/serverapi)Allows developing plugins is for extending the functionality of MetaTrader 5 Trade Server and MetaTrader 5 History Server. It is available as C++ interfaces and examples in the source code.
-   [**Manager API**  
    ](/en/docs/mt5/api/managerapi)Intended for expanding the functionality of MetaTrader 5 Manager and MetaTrader 5 Administrator. Allows controlling the platform settings and processing trade requests. It is available as C++ interfaces and examples in the source code.
-   **[Gateway API](/en/docs/mt5/api/gatewayapi)**  
    Intended for developing custom data feeds (quotes, news) and integration with other trading systems. It is available as C++ interfaces, DLL and examples in the source code.
-   **[Report API](/en/docs/mt5/api/reportapi)**  
    Allows creating custom reports for MetaTrader 5 Trade Server in the form of plugins. It is available as C++ interfaces and examples in the source code.
-   **[Web API](/en/docs/mt5/api/webapi)**  
    An open text-based protocol for integrating the MetaTrader 5 platform with web resources and other company's services.
