# Managing Trade Server Settings

With appropriate rights, managers are able to modify client groups and plugins settings on the trade server. For example, a manager can specify new [margin ratios (discounts)](/en/docs/mt5/manager/margin#margin) for trading symbols, [change commission amount](/en/docs/mt5/manager/groups#commission) charged from customers for deals, as well as enable or disable a [plugin](/en/docs/mt5/manager/plugins) running on the server.

![Managing trade server settings](/en/docs/mt5/manager/img/server_management.png "Managing trade server settings")

To control the platform operation, managers are able to [request the trade server journal](/en/docs/mt5/manager/server_journal). It provides data on accounts, trade operations, as well as changes in the platform settings, system and network events and much more.
