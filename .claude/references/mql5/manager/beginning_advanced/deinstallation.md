# Terminal Deinstallation

To remove the Manager terminal from a computer, run the Uninstall.exe file from the platform installation folder or select Uninstall in the appropriate program group in the Start menu.

![To remove the terminal, follow the wizard prompts](/en/docs/mt5/manager/img/uninstall.gif "To remove the terminal, follow the wizard prompts")

Specify the folder the terminal is to be deleted from. Option "Delete user personal data" can be additionally enabled to delete all user data (terminal settings, emails, news etc.) in addition to the [unchangeable files](/en/docs/mt5/manager/beginning_advanced/structure).

If you are sure you want to continue, click Next and wait for the completion of the uninstallation process.
