# For Advanced Users

This section is intended for experienced users. It describes the terminal installation and update features, additional account protection measures and much more.

-   [Installing the terminal](/en/docs/mt5/manager/beginning_advanced/installation) — the terminal is easy to install, you only need to follow the wizard prompts. However, there are some features that should be taken into account, including the system requirements, specific installation over the existing platform, etc.
-   [Files and folders](/en/docs/mt5/manager/beginning_advanced/structure) — find out, in which folders your settings, templates and the terminal journal are located in.
-   [Extended authentication](/en/docs/mt5/manager/beginning_advanced/extended_authorization) — the terminal support certificate based authentication, which greatly increases the safety of the system.
-   [One Time Password - OTP](/en/docs/mt5/manager/beginning_advanced/otp) — another account security layer is using one time passwords generated by your mobile device.
-   [Live Update](/en/docs/mt5/manager/beginning_advanced/liveupdate) — the terminal is updated automatically and usually requires no additional actions from the user.
-   [Exporting data](/en/docs/mt5/manager/beginning_advanced/export) — the terminal allows you to save your account data and trading operations to files to work with them in other applications.
-   [Terminal deinstallation](/en/docs/mt5/manager/beginning_advanced/deinstallation) — you can easily uninstall the terminal if you need: follow the wizard prompts. Some of the features of this process are described in this section.
