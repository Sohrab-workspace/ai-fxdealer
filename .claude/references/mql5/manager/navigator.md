# Navigator

The Navigator window allows you to conveniently switch between various sections of the Manager terminal. You can quickly go to [reports](/en/docs/mt5/manager/reports), [client list](/en/docs/mt5/manager/accounts), [group](/en/docs/mt5/manager/groups) settings, [plugins](/en/docs/mt5/manager/plugins), etc. If you have several manager accounts on different servers, Navigator allows you to conveniently switch between them.

Press Ctrl+D or select the "![Navigator](/en/docs/mt5/manager/img/navigator_icon.png "Navigator") Navigator" command in the [View](/en/docs/mt5/manager/main_menu#view) menu.

![The Navigator window enables fast switching between the Manager terminal sections](/en/docs/mt5/manager/img/navigator.png "The Navigator window enables fast switching between the Manager terminal sections")

The Navigator also provides access to the [technical support](/en/docs/mt5/manager/tech_support) section. There you can find complete documentation on the platform and useful articles, as well as contact developers via the online chat or Service Desk in case of an issue.
