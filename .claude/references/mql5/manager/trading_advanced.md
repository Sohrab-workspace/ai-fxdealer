# For Advanced Users

This section is intended for experienced users. It describes some specific trading features available in the platform:

-   [Margin Calculation: Basic](/en/docs/mt5/manager/trading_advanced/margin_formula) — basic margin calculation formulas for all types of symbols.
-   [Margin Calculation: Retail Forex, CFD, Futures — Netting](/en/docs/mt5/manager/trading_advanced/margin_forex) — margin calculation for the over-the-counter market with the netting position accounting system.
-   [Margin Calculation: Retail Forex, CFD, Futures — Hedging](/en/docs/mt5/manager/trading_advanced/margin_forex_hedge) — margin calculation for the over-the-counter market with the hedging position accounting system.
-   [Margin Calculation: Stock Exchange](/en/docs/mt5/manager/trading_advanced/margin_exchange) — terminology and margin calculation specifics for exchange instruments on a discount basis set by a broker.
-   [Spreads](/en/docs/mt5/manager/trading_advanced/spreads) — margin can be charged on preferential basis in case trading positions are in spread relative to each other. The spread trading is defined as the presence of the oppositely directed positions of correlated symbols.
-   [Collateral Instruments](/en/docs/mt5/manager/trading_advanced/symbols_collateral) — the platform supports a special type of non-tradable instruments, which can be used as client's assets to provide the required margin for open positions of other instruments.
