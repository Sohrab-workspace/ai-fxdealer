# A-Book & B-Book Exposure

The report displays the total volume of clients' current positions routed to external liquidity providers (A-Book) and those executed internally by the broker (B-Book). This enables managers to monitor imbalances and manage risk exposure effectively.

![A-Book & B-Book Exposure](/en/docs/mt5/manager/img/abook_bbook_exposure_report.png "A-Book & B-Book Exposure")

## Filters

Data requested for the report can be filtered. Before generating the report in the Manager terminal, specify the following parameters:

-   Symbols — trading instruments to be included in the report.
-   Groups account groups for which the report should be generated. You can specify one or several accounts separated by commas.
-   Top Symbols — the number of symbols with the highest trading volume to include in the report.
-   Top Accounts — the number of accounts with the highest trading volume to include in the report.

## Report Data

The report provides diagrams showing the distribution of net volume (the difference between buy and sell trades) for A-Book, B-Book, and their combined total. A detailed data table is available below:

-   Symbol — the name of the financial instrument.
-   A-Book Buy — the current total buy volume of client positions executed via A-Book.
-   A-Book Sell — the current total sell volume of client positions executed via A-Book.
-   A-Book Net — the difference between buy and sell volumes for A-Book positions. Additionally, the prevailing imbalance direction is indicated.
-   B-Book Buy — the current total buy volume of client positions executed via B-Book.
-   B-Book Sell — the current total sell volume of client positions executed via B-Book.
-   B-Book Net — the difference between buy and sell volumes for B-Book positions. Additionally, the prevailing imbalance direction is indicated.
-   Total Net — the overall difference between buy and sell volumes across both A-Book and B-Book positions.
