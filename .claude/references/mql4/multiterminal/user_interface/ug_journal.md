# Journal

All operations and events performed are written to special statistics journals that are represented as text files. The "Journal" section of the "Toolbox" window allows viewing these entries. The information is represented in the form of the table containing the following fields:

-   Time — the date and time of an event;
-   Message — the description of an occurred event.

Only the current day entries are represented in the journal. In order to view earlier entries you should execute the "Open" command of the context menu. At the same time the folder containing the text files of the journal will be opened. You can copy the selected entry of the journal to the clipboard using the "Copy" command in the context menu. Also the context menu allows enabling the auto scrolling of messages (it returns you to the first message if the journal is updated), to enable the automatic sizing of columns and the displaying of the grid.
