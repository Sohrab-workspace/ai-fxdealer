# Main Menu

The main menu consists of the following sub-menus:

-   [File](/en/docs/mt4/multiterminal/user_interface/ug_main_menu#file)
-   [Edit](/en/docs/mt4/multiterminal/user_interface/ug_main_menu#edit)
-   [View](/en/docs/mt4/multiterminal/user_interface/ug_main_menu#view)
-   [Tools](/en/docs/mt4/multiterminal/user_interface/ug_main_menu#tools)
-   [Window](/en/docs/mt4/multiterminal/user_interface/ug_main_menu#window)
-   [Help](/en/docs/mt4/multiterminal/user_interface/ug_main_menu#help)

## File Menu [#](ug_main_menu#file)

The File menu contains commands for managing accounts, and for printing and saving reports. The following commands are available in the menu:

-   Connect All — [connect](/en/docs/mt4/multiterminal/accounts/ug_account_connect) all account to the server;
-   Disconnect All — [disconnect](/en/docs/mt4/multiterminal/accounts/ug_account_connect) all accounts from the server;
-   New Account — [add new account](/en/docs/mt4/multiterminal/accounts/ug_accounts) to the list of managed accounts;
-   Save — save the list of orders or accounts history;
-   Print — print the list of orders or accounts history;
-   Print Preview — preview before printing;
-   Print Setup — common print setup;
-   Exit — close the program.

## Edit Menu [#](ug_main_menu#edit)

The Edit menu contains find and copy commands. The following commands are available in the menu:

-   Copy — copy selected data into clipboard;
-   Find — finding a list element according to the given criterion;
-   Find Next — continue the search and find the next list element.

## View Menu [#](ug_main_menu#view)

The View menu contains commands for managing signal windows, toolbar, and the program interface language. Some of these commands are duplicated in the [toolbar](/en/docs/mt4/multiterminal/user_interface/ug_toolbars). The following commands are available in the menu:

-   Languages — call the sub-menu managing languages of the terminal. Having selected a desired language, one can switch the terminal interface to this language. For the changes to take effect, the program must be restarted.
-   Toolbar — show or hide the toolbar of the terminal.
-   Status Bar — show or hide a status bar located in the lower part of the terminal window.
-   Market Watch — open/close the ["Market Watch"](/en/docs/mt4/multiterminal/user_interface/ug_market_watch) signal window where the current quotes are published. The same action can be performed by accelerating buttons of Ctrl+M or by pressing the corresponding button of the [toolbar](/en/docs/mt4/multiterminal/user_interface/ug_toolbars).
-   Accounts — open/close the ["Accounts"](/en/docs/mt4/multiterminal/accounts/ug_accounts) window where the list of managed accounts is located. The same action can be performed by accelerating buttons of Ctrl+A or by pressing the corresponding button of the [toolbar](/en/docs/mt4/multiterminal/user_interface/ug_toolbars).
-   Toolbox — open/close the "Toolbox" window. One can manage signals in this window, look through the news, emails, and journal. The same action can be performed by accelerating buttons of Ctrl+T or by the corresponding button of the [toolbar](/en/docs/mt4/multiterminal/user_interface/ug_toolbars).
-   Full Screen — enable/disable the full screen mode. As soon as this option is enabled, toolbar and status bar will be disabled and all signal windows will be closed. The terminal heading, main menu, workspace remain on the screen. The repeated execution of the command returns the terminal to the initial appearance.  
    The same action can be performed by F11 button or by the corresponding button of the [toolbar](/en/docs/mt4/multiterminal/user_interface/ug_toolbars).

## Tools Menu [#](ug_main_menu#tools)

The Tools menu contains the only command to open terminal settings window.

-   Options — open the [terminal setting window](/en/docs/mt4/multiterminal/getting_started/ug_setup). One can set up parameters for connection, trades, emails, and events in this window. More details can be found in the ["Terminal Settings"](/en/docs/mt4/multiterminal/getting_started/ug_setup) section. The Options window can also be called by accelerating keys of Ctrl+O or by a corresponding button of the [toolbar](/en/docs/mt4/multiterminal/user_interface/ug_toolbars).

## Window Menu [#](ug_main_menu#window)

The Window menu contains commands for switching between order and trade windows.

-   Orders — switch to [open orders](/en/docs/mt4/multiterminal/accounts/ug_account_orders) window;
-   History — switch to [trade history](/en/docs/mt4/multiterminal/accounts/ug_account_history) window;
-   New Order — switch to [new orders](/en/docs/mt4/multiterminal/trading/trade_orders/ug_order_new) window;
-   Close Order — switch to [close orders](/en/docs/mt4/multiterminal/trading/trade_orders/ug_order_close) window;
-   Close By — switch to [close by orders](/en/docs/mt4/multiterminal/trading/trade_orders/ug_order_closeby) window;
-   Multiple Close By — switch to [multiple close by orders](/en/docs/mt4/multiterminal/trading/trade_orders/ug_order_multiple) window;
-   Pending Order — switch to [placing pending orders](/en/docs/mt4/multiterminal/trading/trade_orders/ug_order_pend) window;
-   Modify Order — switch to [modifying orders](/en/docs/mt4/multiterminal/trading/trade_orders/ug_order_modify) window;
-   Delete Order — switch to [deleting pending orders](/en/docs/mt4/multiterminal/trading/trade_orders/ug_order_delete) window;

## Help Menu [#](ug_main_menu#help)

The Help menu is auxiliary. Execution of the menu commands does not influence the terminal operation in any way. One can do the following from this menu:

-   Help Topics — open the built-in help files containing this present Userguide. All settings and functions of the MultiTerminal are described in it. F1 button performs the same action.
-   About... — open the "About..." window where one can find information about the company, its contact details, and the terminal version, as well.
