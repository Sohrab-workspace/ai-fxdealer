# Workspace

The main part of the MetaEditor interface allows working with program source codes. The [Navigator](/en/docs/mt5/metaeditor/navigator) window allows viewing folders and opening files for editing. The [Toolbox](/en/docs/mt5/metaeditor/toolbox) window displays results of various actions in MetaEditor, including search, compilation and profiling. Besides, it provides access to the extensive library of source codes and articles about MQL4/MQL5 programming.

This section additionally lists all [MetaEditor hot keys](/en/docs/mt5/metaeditor/hotkeys) greatly facilitating and speeding up your work.

![MetaEditor workspace](/en/docs/mt5/metaeditor/img/work_area.png)

### Navigator

["Navigator"](/en/docs/mt5/metaeditor/navigator) is designed to work with files and folders inside /MQL4 or /MQL5 directories of the trading platform. Here you can create and delete folders and files, as well as work with projects and [MQL5 Storage](/en/docs/mt5/metaeditor/mql5storage).

### Code editing

Window for writing a source code.

### Status Bar

The [status bar](/en/docs/mt5/metaeditor/status_bar) displays the current cursor position in the program code and command prompts.

### Toolbox

The ["Toolbox"](/en/docs/mt5/metaeditor/toolbox) window displays results of search, compilation and profiling. It also provides access to Code Base, articles about programming, the list of public projects and MetaEditor journal.

### Tabs of open files

Here you can switch between open files. The context menu allows you to go to a file on the disk or start compilation.
