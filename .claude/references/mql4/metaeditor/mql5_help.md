# Built-in help

MetaEditor features the built-in MetaQuotes Language 4 and 5 (MQL4 and MQL5) help providing complete description of all its elements and constructions. The help (or MQL5 Reference) is context-based. Place the cursor over any language element in a source code and press F1 — the required documentation section will be opened immediately:

![MQL4/MQL5 context help](/en/docs/mt5/metaeditor/img/context_help.png "MQL4/MQL5 context help")

You can also open the built-in Help using the "MQL4/MQL5 Reference" in the [Help](/en/docs/mt5/metaeditor/main_menu_help) menu.

In addition to the built-in Help, MetaEditor also features the context-based search on [MQL5.community](https://www.mql5.com/ "MQL5.community"). Place the cursor over any language element in a source code and press Ctrl+F1 — all detected website materials will be displayed on the [Search](/en/docs/mt5/metaeditor/toolbar_search) tab of the Toolbox window.

![Context-based search on MQL5.community](/en/docs/mt5/metaeditor/img/online_help.png "Context-based search on MQL5.community")

You can also perform a search on MQL5.community via the [Search toolbar](/en/docs/mt5/metaeditor/toolbar_search) .
