# Terminal Settings

The window of the administrator terminal settings can be opened by pressing the "![Options](/en/docs/mt4/administrator/img/options_icon.png "Options") Options" button in the ["Services"](/en/docs/mt4/administrator/getting_started/ug_main_menu) menu or using the "Ctrl+O" key combination. All the settings are grouped in three tabs:

-   [Common](/en/docs/mt4/administrator/settings/settings_common)  
    General settings of the administrator terminal.
-   [Support](/en/docs/mt4/administrator/settings/settings_support)  
    The authorization details for contacting the technical support team.
