# Journal

"Journal" tab contains the information about all actions of administrator terminal for the current session. In the journal is fixed all information about terminal start and about the events during its operation, including all trading operations. In the window are placed only the last messages. To check the previous messages it is necessary to run the command "Open" and choose the required file.
