# Articles

This tab displays news and articles published at the [MetaQuotes Software Corp. Support Center](https://support.metaquotes.net/ru/articles "MetaQuotes Software Corp. Technical Support WebSite") website. These articles describe how to work with the trading platform.

![Articles](/en/docs/mt4/administrator/img/toolbox_articles.png "Articles")

The articles are displayed as a list:

-   Name — name of an article;
-   Category — category an article belongs to. To show/hide a category, use the context menu.
-   Date — date when an article was published.

To view a short description of an article, put the mouse cursor over its name. To read a full article, double-click on its name. Once you do it, the corresponding page of the Support Center is opened in your web browser.

The context menu of this tab allows to execute the following commands:

-   ![View](/en/docs/mt4/administrator/img/article_view_icon.png "View") View — open a selected article in a web browser;
-   ![Refresh](/en/docs/mt4/administrator/img/refresh_icon.png "Refresh") Refresh — refresh the list of articles;
-   Categories — open the submenu of working with the categories of articles;

-   ![Select All](/en/docs/mt4/administrator/img/article_select_all_icon.png "Select All") Select All — select all categories for displaying in the list;
-   ![Unselect All](/en/docs/mt4/administrator/img/article_unselect_icon.png "Unselect All") Unselect All —  hide all categories;
-   List of categories — the list of all existing categories is shown below. In order to select a category, put a checkmark against it by a single mouse click;

-   Category — show/hide the column of article categories. The same action can be performed using the "R" key;
-   Auto Arrange — enable/disable the automatic setup of the column size. The same action can be performed by pressing the "A" key;
-   Grid — show/hide grid to separate fields. The same action can be performed by pressing the "G" key.
