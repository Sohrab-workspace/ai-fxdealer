# Toolbars

A toolbar is an adjustable window having a set of buttons assigned to the often used commands. Three types of tools are represented in the program: ["Standard"](/en/docs/mt4/manager/user_interface/ug_toolbars#standard), ["Dealer"](/en/docs/mt4/manager/user_interface/ug_toolbars#dealer), and ["Coverage"](/en/docs/mt4/manager/user_interface/ug_toolbars#coverage).

All toolbars are made in form of floating windows and can have any location in the terminal working space. Toolbars are usually located below the main menu. Toolbars can be shown or hidden through ["View — Toolbars"](/en/docs/mt4/manager/user_interface/ug_main_menu#view) menu command. To setup the toolbars, it is necessary to execute the "Customize" command of the context menu of the corresponding toolbar. All buttons of toolbars are provided with popup prompts to appear as soon as the mouse cursor is placed on the button.

## "Standard" Toolbar [#](ug_toolbars#standard)

-   ![Connect](/en/docs/mt4/manager/img/ic_server_login.png "Connect") Connect — connect to the server;
-   ![Disconnect](/en/docs/mt4/manager/img/ic_server_disconnect.png "Disconnect") Disconnect — disconnect from the server;
-   ![Account](/en/docs/mt4/manager/img/ic_account_manager.png "Account") Account — reconnect manager with different login;
-   ![Save](/en/docs/mt4/manager/img/ic_save_as.png "Save") Save — save list;
-   ![Print](/en/docs/mt4/manager/img/ic_print.png "Print") Print — print out the document;
-   ![Print Preview](/en/docs/mt4/manager/img/ic_print_preview.png "Print Preview") Print Preview — previewing before printing;
-   !["Market Watch" Window](/en/docs/mt4/manager/img/ic_market_watch.png ""Market Watch" Window") Market Watch — open/close ["Market Watch"](/en/docs/mt4/manager/user_interface/ug_market_watch) window. The ["View — Market Watch"](/en/docs/mt4/manager/user_interface/ug_main_menu#file) menu command and accelerating buttons Ctrl+M perform the same action;
-   ![Margin Calls](/en/docs/mt4/manager/img/ic_margin_call.png "Margin Calls") Margin Calls — open/close ["Margin Call"](/en/docs/mt4/manager/request_processing/ug_win_queue#margin_call) window;
-   ![Queue](/en/docs/mt4/manager/img/ic_queue.png "Queue") Queue — open/close ["Queue"](/en/docs/mt4/manager/request_processing/ug_win_queue#queue) window;
-   ![Toolbox](/en/docs/mt4/manager/img/ic_toolbox.png "Toolbox") Toolbox — open/close "Toolbox" window;
-   ![New Account](/en/docs/mt4/manager/img/ic_new_account.png "New Account") New Account — add [a new account](/en/docs/mt4/manager/client_accounts/ug_new_accounts);
-   ![Options](/en/docs/mt4/manager/img/ic_options.png "Options") Options — [settings](/en/docs/mt4/manager/getting_started/ug_setup) of the manager terminal;
-   ![Help](/en/docs/mt4/manager/img/ic_help.png "Help") Help — show help content;
-   ![About](/en/docs/mt4/manager/img/ic_about.png "About") About — "About" window call.

You can quickly open the window of account trades management or the window of a certain trade from the "Standard" toolbar. To do it, specify an account number or a trade number after the "#" symbol in the "Find Account or Order" field and press the "Enter" button.

## "Dealer" Toolbar [#](ug_toolbars#dealer)

-   ![Connect](/en/docs/mt4/manager/img/ic_dealer_connect.png "Connect") Connect — connect the dealer to the server;
-   ![Disconnect](/en/docs/mt4/manager/img/ic_dealer_disconnect.png "Disconnect") Disconnect — disconnect the dealer from the server;
-   ![Automation](/en/docs/mt4/manager/img/ic_automation.png "Automation") Automation — enable/disable [automatic processing](/en/docs/mt4/manager/request_processing/ug_automation) of requests;
-   ![Multiple Execution](/en/docs/mt4/manager/img/ic_multiple.png "Multiple Execution") Multiple Execution — [multiple execution](/en/docs/mt4/manager/request_processing/ug_dealing) of queued requests of the same type;
-   ![Throw in Prices](/en/docs/mt4/manager/img/ic_throw.png "Throw in Prices") Throw in Prices — [push new quotations](/en/docs/mt4/manager/request_processing/ug_dealing) into the quotation flow from data feeders when replying a request.

## "Coverage" Toolbar [#](ug_toolbars#coverage)

-   ![Connect](/en/docs/mt4/manager/img/ic_connect_coverage.png "Connect") Connect — connect hedging account.
-   ![Disconnect](/en/docs/mt4/manager/img/ic_disconnect_coverage.png "Disconnect") Disconnect — disconnect hedging account.
-   ![New Order](/en/docs/mt4/manager/img/ic_coverage_order.png "New Order") New Order — open new hedging order.
