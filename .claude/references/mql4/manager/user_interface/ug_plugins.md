# Plugins Window

The "Plugins" window is intended for controlling of plugins-the server functionality extension modules.

![Plugins Window](/en/docs/mt4/manager/img/br_plugins.png "Plugins Window")

<table class="attentable" cellspacing="0" cellpadding="8" border="1"><tbody><tr class="attentable"><td class="attentable"><p class="p_tableatten"><span class="f_tableatten">For a manager to be able to configure a plugin, the "Configurable by managers" option must be enabled in the plugin configuration on the server.</span></p></td></tr></tbody></table>

## Plugin settings [#](ug_plugins#plugin_setup)

To change plugin settings, it is necessary to execute the "Settings" context menu command. The window with detailed settings of the plugin will appear:

![Plugins](/en/docs/mt4/manager/img/win_plugins.png "Plugins")

Each plugin can contain a set of some parameters serving for adjustment of its operation. Current parameters are displayed in "Parameters:" of plugin. For changing value or name of the parameter, it is necessary to select a necessary field in the list using doubleclick. After changing the edited field, it is necessary to press Enter key. To add the new parameter, it is necessary to press "Add" button. To delete the selected parameter, it is necessary to press "Delete" button.
