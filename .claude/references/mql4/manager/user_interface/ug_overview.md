# Overview

![MetaTrader 4 Manager](/en/docs/mt4/manager/img/manager.png "MetaTrader 4 Manager")

The interface of the software includes the following elements:

-   The Main Window Heading  
    The heading of the program includes the number of the current account, the name of application.
-   Main Menu  
    All commands and functions to be executed in the client terminal are collected in the main menu. The main menu of the program includes: ["File"](/en/docs/mt4/manager/user_interface/ug_main_menu#file), ["Edit"](/en/docs/mt4/manager/user_interface/ug_main_menu#edit), ["View"](/en/docs/mt4/manager/user_interface/ug_main_menu#view), ["Tools"](/en/docs/mt4/manager/user_interface/ug_main_menu#tools), ["Window"](/en/docs/mt4/manager/user_interface/ug_main_menu#window), ["Help"](/en/docs/mt4/manager/user_interface/ug_main_menu#help). More details can be found in the ["Main Menu"](/en/docs/mt4/manager/user_interface/ug_main_menu) section.
-   Toolbars  
    Three toolbars ["Standard"](/en/docs/mt4/manager/user_interface/ug_toolbars#standard), ["Dealing"](/en/docs/mt4/manager/user_interface/ug_toolbars#dealer), and ["Coverage"](/en/docs/mt4/manager/user_interface/ug_toolbars#coverage) are built in the terminal. The basic commands and functions of the main menu are duplicated in the toolbars. However, these toolbars are adjustable, and it is possible to collect only the most frequently used elements of management in them. More details can be found in the ["Toolbars"](/en/docs/mt4/manager/user_interface/ug_toolbars) section.
-   "Market Watch" Window  
    The list of financial instruments (symbols) is given in this window. More details can be found in the ["Market Watch" Window](/en/docs/mt4/manager/user_interface/ug_market_watch) section.
-   Status Bar  
    The status bar is also used in the terminal for visualization of additional information. The indicator of server connection is located in the status bar, command prompts and price data are displayed there, as well.
