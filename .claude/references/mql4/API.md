# MetaTrader 4 API — Reference

The MetaTrader 4 trading platform provides public interfaces (Application Programming Interface, API) for extending its functionality: adding new functions into the platform, integrating it with other solutions and customizing it for a variety of unique tasks.

![MetaTrader 4 API](/en/docs/mt4/api/img/api.jpg "MetaTrader 4 API")

Five APIs are available:

-   [Server API](/en/docs/mt4/api/server_api) is a set of tools for extending the functionality and configuring the operation of the MetaTrader 4 server for specific needs.
-   [Manager API](/en/docs/mt4/api/manager_api) is responsible for the development of additional functionality, platform integration and customization of its functions. Actually, using this interface, you can create your own manager terminal to work with the platform.
-   [DataFeed API](/en/docs/mt4/api/datafeed_api) is designed for the development of news and quote data feeds.
-   [Report API](/en/docs/mt4/api/report_api) is a separate interface for creating custom reports on financial operations and accounts of traders.
-   [WebServices API](/en/docs/mt4/api/webservices_api) enables integration of the platform with websites. It can be used to create a trader's room, to feed quotes, and much more.
