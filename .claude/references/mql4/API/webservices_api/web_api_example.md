# Example: Quotes on a Website

The section features an example of a simple informer with quotes. You can download a zip with the example files at the [link](https://support.metaquotes.net/spfiles/webservices/webservices-quotes.zip). To get started, you only need to unzip the files to the website directory on the server.

## File Descriptions

-   quotes.php - an informer of quotes of symbols EURUSD, GBPUSD, USDJPY, AUDUSD, USDCAD. The file contains:

-   configuration parameters for connecting to the server (IP address, port)
-   request caching parameters
-   The function of sending requests to the server, with an implemented support for result catching

-   example.php - an example of a web page containing the informer quotes.php.
-   cache - directory for storing the cache of results
-   img - a directory with images that are used for displaying quotes.

Request parameters are set using the PHP define function:

The values of the following parameters are defined at the beginning of the quotes.php file:

-   T\_HOST - MetaTrader server address. Example: "www.metaquotes.ru".
-   T\_PORT - MetaTrader server port. Example: 443.
-   T\_TIMEOUT - MetaTrader server connection timeout in seconds. Example: 5.
-   T\_QUOTES - list of symbols to be published. Example: "EURUSD,GBPUSD,USDJPY,AUDUSD,USDCAD,".
-   T\_CACHEDIR - directory to store cache files. Example: "c:/temp/" or "cache/".
-   T\_CACHETIME - cache expiry time in seconds. Example: 5.
-   T\_CLEAR\_DELNUMBER - the number of files closed at a time, reaching which the process of cache clearing should be stopped. Example: 15.

The values ​​of these parameters can be changed at your discretion.
