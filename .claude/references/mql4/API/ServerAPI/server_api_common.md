# Common Functions

The common functions of MetaTrader 4 Server API include general purpose functions, which do not control platform settings or information on the server, but are required for plugin development. This group of functions includes:

-   [Memory management functions](/en/docs/mt4/api/server_api/server_api_common/server_api_common_memory)
-   [Functions for working with logs](/en/docs/mt4/api/server_api/server_api_common/server_api_common_journal)
-   [Service Functions](/en/docs/mt4/api/server_api/server_api_common/server_api_common_service)
