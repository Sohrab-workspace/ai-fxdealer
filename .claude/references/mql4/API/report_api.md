# MetaTrader 4 Report API

The MetaTrader 4 Manager Terminal includes standard reports on financial operations and traders' accounts. MetaTrader Report API is a separate interface allowing you to create custom reports for the manager terminal.

MetaTrader Report API consists of a set of exported functions required for report generation. Also API includes standard reports of the manager terminal in the form of the source code, which can be used as an example and a basis for developing your own reports.
