# Exported Functions and Factory

The mtmanapi.dll library (and its 64-bit version), which is used for enabling the interaction of the application and the MetaTrader 4 trading platform, can export several functions:

In order to simplify access to the CManagerInterface interface, the "MT4ManagerAPI.h" file contains [a factory of interfaces](/en/docs/mt4/api/manager_api/manager_api_exported_factory/manager_api_cmanagerfactory), which automatically loads the correct library (32/64-bit) of the Manager API and provides access to exported functions.
