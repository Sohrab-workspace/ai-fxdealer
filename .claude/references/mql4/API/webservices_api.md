# MetaTrader 4 WebServices API

MetaTrader 4 WebServices API enables integration of the platform with websites. It can be used to create a trader's room, to feed quotes, and much more.

The API is a set of commands that make up a protocol for communication with the server via a TCP connection. The communication is realized in a text form, so access to the API can be implemented using a wide range of web-based technologies, such as CGI, PHP, ASP, ASP.NET, JSP, and others.
