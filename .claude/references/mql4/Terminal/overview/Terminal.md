# Terminal

"Terminal" is a multifunctional window allowing to access to various terminal features. This window allows to control over [trading activities](/en/docs/mt4/terminal/overview/terminal/terminal_trade), view [news](/en/docs/mt4/terminal/overview/terminal/terminal_news) and [account history](/en/docs/mt4/terminal/overview/terminal/terminal_account_history), set up [alerts](/en/docs/mt4/terminal/overview/terminal/terminal_alerts), and work with the [internal mail](/en/docs/mt4/terminal/overview/terminal/terminal_mailbox) and [system journal](/en/docs/mt4/terminal/overview/terminal/terminal_journal). The window can be opened by the ["View — Terminals" menu](/en/docs/mt4/terminal/overview/main_menu/main_menu_view) command, by pressing of accelerating keys of Ctrl+T, or the !["Terminal" window](/en/docs/mt4/terminal/img/tb_standard_terminal.png ""Terminal" window") button of the ["Standard" toolbar](/en/docs/mt4/terminal/overview/toolbars/toolbars_standard). There are several tabs in the window:

-   [Trade](/en/docs/mt4/terminal/overview/terminal/terminal_trade) — in this tab, one can view the status of open [positions](/en/docs/mt4/terminal/positions) and [pending orders](/en/docs/mt4/terminal/positions/orders), as well as manage all trading activities. Besides, the total financial result for all open positions is published in this tab;

-   [Exposure](/en/docs/mt4/terminal/overview/terminal/toolbox_exposure) — this tab contains the summary information about the state of assets by all open positions;

-   [Account History](/en/docs/mt4/terminal/overview/terminal/terminal_account_history) — history for all performed trade operations and balance without taking open positions into consideration are published here. The efficiency of all trade activities can be estimated with the results given in this tab;
-   [News](/en/docs/mt4/terminal/overview/terminal/terminal_news) — all financial news incoming in the terminal can be viewed from this tab. This prompt information is indispensable for fundamental analysis;
-   [Alerts](/en/docs/mt4/terminal/overview/terminal/terminal_alerts) — various alerts can be viewed and set here. Any files executable in the operational environment (including wave files) and messages to be sent by email can be used as alerts;
-   [Mailbox](/en/docs/mt4/terminal/overview/terminal/terminal_mailbox) — all message income in the terminal by internal email are stored here. Electronic messages are sent from here, as well;
-   [Company](/en/docs/mt4/terminal/overview/terminal/toolbox_company) — this tab displays a useful web page for a trader;
-   [Market](/en/docs/mt4/terminal/overview/terminal/toolbox_market) — buying or downloading applications from the market of MQL4 applications at [MQL5.community](https://www.mql5.com/en/market).
-   [Signals](/en/docs/mt4/terminal/overview/terminal/toolbox_signals) — this tab displays trading signals of the ["Signals"](/en/docs/mt4/terminal/signals) service, which are available for subscription;
-   [Code Base](/en/docs/mt4/terminal/overview/terminal/toolbox_codebase) — here you can download any application published in the "Code Base" section of the [MQL5.community](https://www.mql5.com/en/code) website;
-   [Search](/en/docs/mt4/terminal/overview/terminal/toolbox_search) — this tab displays the result of [searching](/en/docs/mt4/terminal/overview/search) through the client terminal;
-   [Expert Advisors](/en/docs/mt4/terminal/overview/terminal/terminal_experts) — information about functioning of the attached expert, including opening/closing of positions, order modifying, the expert's own messages, etc., are published in this tab;
-   [Journal](/en/docs/mt4/terminal/overview/terminal/terminal_journal) — information about terminal launching and about events during its operation, including all trade operations performed, is stored in the journal.
