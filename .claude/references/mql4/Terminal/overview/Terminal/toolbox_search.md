# Search

The trading platform provides the smart and powerful system for searching on the websites of MQL4.community and [MQL5.community](https://www.mql5.com "MQL5.community") — the communities of traders and [MQL4/MQL5](/en/docs/mt4/terminal/autotrading/mql4) developers. The sites contain plenty of useful information: [blogs](https://www.mql5.com/en/blogs "MQL5.community blogs"), [documentation](https://docs.mql4.com/ "MQL4.community Documentation"), [forum](https://www.mql5.com/en/forum/mql4 "MQL4.community Forum"), [articles](https://www.mql5.com/en/articles/mt4 "MQL4.community Articles") on programming and using the platform. The community provides access to the huge [source code database](/en/docs/mt4/terminal/overview/terminal/toolbox_codebase "MQL5.community source code database") and [the application store](/en/docs/mt4/terminal/overview/terminal/toolbox_market "Trading platform application store at MQL5.community") for the platform. You can also copy deals of professional traders via [Signals service](/en/docs/mt4/terminal/signals "Copying deals via Signals service at MQL5.community").

Results of the [search on MQL5.community](/en/docs/mt4/terminal/overview/search) are displayed on the tab.

![Search results](/en/docs/mt4/terminal/img/toolbox_search.png "Search results")

To open the found element, left-click on the header. Use the top panel to view the search results by Market Products, Code Base, Signals, MQL4.community Forum and Documentation.
