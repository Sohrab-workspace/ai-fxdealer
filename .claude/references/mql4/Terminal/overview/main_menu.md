# Main Menu

The main menu of the terminal is located under the program heading and represents a set of sub-menus.

![main_menu](/en/docs/mt4/terminal/img/main_menu.png)

Commands and functions to be executed in the client terminal are collected in it. The main menu consists of:

-   [File](/en/docs/mt4/terminal/overview/main_menu/main_menu_file)  
    Working with charts, storing of history data and profiles, and print.
-   [View](/en/docs/mt4/terminal/overview/main_menu/main_menu_view)  
    Setting toolbar, managing command windows (["Market Watch"](/en/docs/mt4/terminal/overview/market_watch), ["Data Window"](/en/docs/mt4/terminal/overview/data_window), ["Navigator"](/en/docs/mt4/terminal/overview/navigator), ["Terminal"](/en/docs/mt4/terminal/overview/terminal), ["Tester"](/en/docs/mt4/terminal/autotrading/tester)) and the program interface language.
-   [Insert](/en/docs/mt4/terminal/overview/main_menu/main_menu_insert)  
    Managing technical indicators, line studies, and other objects.
-   [Charts](/en/docs/mt4/terminal/overview/main_menu/main_menu_charts)  
    Setting of bars displaying, scale, managing the imposed objects, presence of grid, chart properties.
-   [Tools](/en/docs/mt4/terminal/overview/main_menu/main_menu_tools)  
    General settings of the client terminal, History Center, global variables, and [MetaEditor (MetaQuotes Language 4 editor)](/en/docs/mt4/terminal/autotrading/metaeditor).
-   [Window](/en/docs/mt4/terminal/overview/main_menu/main_menu_window)  
    The list of open charts and their locations.
-   [Help](/en/docs/mt4/terminal/overview/main_menu/main_menu_help)  
    Userguide and information about the program.
