# Help

The "Help" menu is auxiliary.

![help_menu](/en/docs/mt4/terminal/img/help_menu.png)

Execution of this menu commands does not influence the terminal operation in any way. One can do the following from this menu:

-   Help Topics — open the in-built help files containing this present "Userguide". All settings and functions of the Client Terminal are described in it.  
    F1 button performs the same action.

-   Online Search — go to [searching on MQL5.community](/en/docs/mt4/terminal/overview/search).
-   Video Guides — links to educational videos about Signals, Hosting and Market.
-   Links to MQL5.community sections — using the menu you can quickly go to the main sections of mql5.com: ["Documentation"](https://www.mql5.com/en/docs), ["Articles"](https://www.mql5.com/en/articles), ["Code Base"](https://www.mql5.com/en/code), ["Jobs"](https://www.mql5.com/en/job), ["Forum"](https://www.mql5.com/en/forum), ["Market"](https://www.mql5.com/en/market) and ["Signals"](https://www.mql5.com/en/signals "Signals").

-   iPhone - AppStore — a link to download the mobile terminal for the [iOS](https://download.terminal.free/cdn/mobile/mt4/ios?hl=en&utm_campaign=metaquotes.download&utm_source=www.metatrader4.com "iOS") powered devices.
-   Android - Google Play — a link to download the mobile terminal for the [Android](https://download.terminal.free/cdn/mobile/mt4/android?hl=en&utm_campaign=metaquotes.download&utm_source=www.metatrader4.com "Android") powered devices.

-   About... — open the "About..." window where one can find information about the company, its contact details, and the terminal version, as well.
