# Standard

General commands managing the terminal are collected in this toolbar:

-   ![New Chart](/en/docs/mt4/terminal/img/tb_standard_new_chart.png "New Chart") — create a new symbol chart window. The command opens the list of financial instruments (symbols). One can open a new chart having selected a necessary name in this list.  
    The same actions are performed after the ["File — New Chart"](/en/docs/mt4/terminal/overview/main_menu/main_menu_file) and ["Window — New Window"](/en/docs/mt4/terminal/overview/main_menu/main_menu_file) menu commands have been executed , as well as the context menu commands of the ["Market Watch — Chart Window"](/en/docs/mt4/terminal/overview/market_watch) window;
-   ![Profiles](/en/docs/mt4/terminal/img/tb_standard_profile.png "Profiles") — menu used to manage profiles. One can activate a profile having selected it in this menu. A profile can be deleted in this menu, and a new profile can be stored based on the current chart windows configuration. More details can be found in the ["Profiles and Templates" section](/en/docs/mt4/terminal/chart_management/templates).  
    To work with profiles, one can also execute the ["File — Profiles" menu](/en/docs/mt4/terminal/overview/main_menu/main_menu_file) command;

-   !["Market Watch" window](/en/docs/mt4/terminal/img/tb_standard_market_watch.png ""Market Watch" window") — open/close the ["Market Watch"](/en/docs/mt4/terminal/overview/market_watch) window. The current quotes for all available symbols are published in this window. This window allows to open symbol charts and place orders. More details about working with this window are given in the ["Market Watch" section](/en/docs/mt4/terminal/overview/market_watch).  
    The ["View — Market Watch" menu](/en/docs/mt4/terminal/overview/main_menu/main_menu_view) command and accelerating keys of Ctrl+M allow to open or close this window, as well;
-   !["Market Watch" window](/en/docs/mt4/terminal/img/tb_standard_data_window.png ""Market Watch" window") — open/close data window named ["Market Watch"](/en/docs/mt4/terminal/overview/data_window). Information about prices and indicators imposed is published in this window. More details about working with this window can be found in the ["Data Window" section](/en/docs/mt4/terminal/overview/data_window).  
    The ["View — Data Window" menu](/en/docs/mt4/terminal/overview/main_menu/main_menu_view) command and accelerating keys of Ctrl+D allow to open or close this window, as well;
-   !["Navigator" window](/en/docs/mt4/terminal/img/tb_standard_navigator.png ""Navigator" window") — open/close the ["Navigator" window](/en/docs/mt4/terminal/overview/navigator). All open accounts, available experts, technical indicators, and scripts are placed in this window. More details about working with this window can be found in the ["Navigator" section](/en/docs/mt4/terminal/overview/navigator).  
    The ["View — Navigator" menu](/en/docs/mt4/terminal/overview/main_menu/main_menu_view) command and accelerating keys of Ctrl+N allow to open or close this window, as well;
-   !["Terminal" window](/en/docs/mt4/terminal/img/tb_standard_terminal.png ""Terminal" window") — open/close the ["Terminal" window](/en/docs/mt4/terminal/overview/terminal). This window allows to place trading orders, view the account history, get news, and many other things. More details about working with this window can be found in the ["Terminal" section](/en/docs/mt4/terminal/overview/terminal).  
    The ["View — Terminal" menu](/en/docs/mt4/terminal/overview/main_menu/main_menu_view) command and accelerating keys of Ctrl+T allow to open or close this window, as well;
-   !["Tester" window](/en/docs/mt4/terminal/img/tb_standard_strategy_tester.png ""Tester" window") — open/close the ["Tester" window](/en/docs/mt4/terminal/overview/strategy_tester). This window is intended for testing and optimization of expert advisors. More details about working with this window can be found in the ["Strategy Testing" section](/en/docs/mt4/terminal/overview/strategy_tester).  
    The ["View — Strategy Tester" menu](/en/docs/mt4/terminal/overview/main_menu/main_menu_view) command and accelerating keys of Ctrl+R allow to open or close this window, as well;
-   ![New Order](/en/docs/mt4/terminal/img/tb_standard_order_new.png "New Order") — place a new order. The command opens the window that allows to place a market or a pending order. More details can be found in the ["Trading" section](/en/docs/mt4/terminal/positions/orders).  
    This window can also be opened by the ["Tools — New Order" menu](/en/docs/mt4/terminal/overview/main_menu/main_menu_tools) command, the ["Market Watch — New Order" window](/en/docs/mt4/terminal/overview/market_watch) or ["Terminal — Trading — New Order" window](/en/docs/mt4/terminal/overview/terminal) context menu commands, as well as by pressing of F9;
-   ![MetaEditor](/en/docs/mt4/terminal/img/tb_standard_metaeditor.png "MetaEditor") — launch MetaEditor. This editing program allows to create and edit [expert advisors](/en/docs/mt4/terminal/autotrading/experts), [custom indicators](/en/docs/mt4/terminal/autotrading/custom_indicators), [scripts](/en/docs/mt4/terminal/autotrading/scripts), and MQL4 libraries. More details about working with this editing program can be found in the ["MetaEditor" section](/en/docs/mt4/terminal/autotrading/metaeditor) and in its own help files.  
    This action can also be performed by the ["Tools — MetaQuotes Language Editor" menu](/en/docs/mt4/terminal/overview/main_menu/main_menu_tools) command or by pressing of F4;
-   ![Enable/disable expert advisors](/en/docs/mt4/terminal/img/tb_standard_expert_enable.png "Enable/disable expert advisors") — enable/disable performing of trade operations by [Expert Advisors](/en/docs/mt4/terminal/autotrading/experts) and [scripts](/en/docs/mt4/terminal/autotrading/scripts). The function of this button is analogous to the option in the ["Expert Advisors"](/en/docs/mt4/terminal/setup/setup_experts) tab.
-   ![Terminal Settings](/en/docs/mt4/terminal/img/tb_standard_options.png "Terminal Settings") — call the window of [terminal settings](/en/docs/mt4/terminal/setup). All basic settings of the terminal are made in this window. More details can be found in the [section of the same name](/en/docs/mt4/terminal/setup).  
    The window of terminal settings can also be opened by the ["Tools — Options" menu](/en/docs/mt4/terminal/overview/main_menu/main_menu_tools) command or by pressing accelerating keys of Ctrl+O;
-   ![Full screen mode](/en/docs/mt4/terminal/img/tb_standard_full_screen.png "Full screen mode") — enable/disable the full-screen mode. When this option is enabled, toolbars and status bar are hidden, and all service windows are closed. The client terminal name, main menu, workspace (charts), and chart windows tabs remain in the screen. The repeated execution of this command will result in return to the initial status.  
    The same actions can be performed by pressing F11 or by the ["View — Full Screen" menu](/en/docs/mt4/terminal/overview/main_menu/main_menu_view) command;
-   ![Chart Print](/en/docs/mt4/terminal/img/tb_standard_print.png "Chart Print") — print the chart. If the "Color Printing" option is enabled in the [program settings](/en/docs/mt4/terminal/setup/setup_charts), the chart can be printed out in color, not as a black-and-white one.  
    The same action can be performed by the ["File — Print..." menu](/en/docs/mt4/terminal/overview/main_menu/main_menu_file) command or by the [chart context menu](/en/docs/mt4/terminal/chart_management/charts_control#context) command of the same name;
-   ![Print Preview](/en/docs/mt4/terminal/img/tb_standard_print_prev.png "Print Preview") — chart preview before it is printed out. The settings of the selected printer will be used at that. Thus, one can check before printing whether all desired data come onto the paper.  
    The same action can be performed by the ["File — Print Preview" menu](/en/docs/mt4/terminal/overview/main_menu/main_menu_file) command or by the [chart context menu](/en/docs/mt4/terminal/chart_management/charts_control#context) command of the same name;
-   ![Help](/en/docs/mt4/terminal/img/tb_standard_help.png "Help") — call the help files. After this button has been pressed, one should click on the desired interface element to call the corresponding help section.  
    The help can also be called by the ["Help — Help Topics" menu](/en/docs/mt4/terminal/overview/main_menu/main_menu_help) command or by pressing of F1.
