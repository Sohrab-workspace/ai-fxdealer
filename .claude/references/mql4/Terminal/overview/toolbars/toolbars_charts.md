# Charts

Commands for managing charts and technical indicators imposed in the charts are grouped in this toolbar. They include:

-   ![Bar Chart](/en/docs/mt4/terminal/img/tb_charts_bar_chart.png "Bar Chart") — display the chart as a sequence of bars (transform into the bar chart).  
    The same action can be performed by the ["Charts — Bar Chart" menu](/en/docs/mt4/terminal/overview/main_menu/main_menu_charts) command or by pressing accelerating keys of Alt+1;
-   ![Candlesticks](/en/docs/mt4/terminal/img/tb_charts_candlestick.png "Candlesticks") — display the chart as a sequence of candlesticks (transform into the candlestick chart).  
    The same action can be performed by the ["Charts — Candlesticks" menu](/en/docs/mt4/terminal/overview/main_menu/main_menu_charts) command or by pressing accelerating keys of Alt+2;
-   ![Line Chart](/en/docs/mt4/terminal/img/tb_charts_line_chart.png "Line Chart") — display the chart as a broken line (transform into the line chart).  
    The same action can be performed by the ["Charts — Line Chart" menu](/en/docs/mt4/terminal/overview/main_menu/main_menu_charts) command or by pressing accelerating keys of Alt+3;
-   ![Zoom In](/en/docs/mt4/terminal/img/tb_charts_zoom_in.png "Zoom In") — zoom in the chart.  
    The same action can be performed by the ["Charts — Zoom In" menu](/en/docs/mt4/terminal/overview/main_menu/main_menu_charts) command, the corresponding [chart context menu](/en/docs/mt4/terminal/chart_management/charts_control#context) command, or by pressing the "+" key;
-   ![Zoom Out](/en/docs/mt4/terminal/img/tb_charts_zoom_out.png "Zoom Out") — zoom out the chart.  
    The same action can be performed by the ["Charts — Zoom Out" menu](/en/docs/mt4/terminal/overview/main_menu/main_menu_charts) command, the corresponding [chart context menu](/en/docs/mt4/terminal/chart_management/charts_control#context) command, or by pressing the "-" key;

-   ![Tile Windows](/en/docs/mt4/terminal/img/tile_windows_button.png "Tile Windows") — arrange chart windows as a tile.  
    The same action can be performed by the ["Window — Tile Windows" menu](/en/docs/mt4/terminal/overview/main_menu/main_menu_view) command or by pressing "Alt+R" key combination;

-   ![Auto Scroll](/en/docs/mt4/terminal/img/tb_charts_auto_scroll.png "Auto Scroll") — automatic scrolling of the chart to the latest bars. If this option is enabled, the latest bars will always be displayed in the screen.  
    The same action can be performed by the ["Charts — Auto Scroll" menu](/en/docs/mt4/terminal/overview/main_menu/main_menu_charts) command;
-   ![Chart Shift](/en/docs/mt4/terminal/img/tb_charts_shift.png "Chart Shift") — shift the chart from the window right border to the shift label of the chart. The chart shift label (a grey triangle in the upper part of the window) can be moved horizontally with the mouse within 10 to 50% of the window size.  
    The same action can be performed by the ["Charts — Chart Shift" menu](/en/docs/mt4/terminal/overview/main_menu/main_menu_charts) command;
-   ![Indicators](/en/docs/mt4/terminal/img/tb_charts_indicators.png "Indicators") — call the sub-menu to manage indicators. All built-in [technical indicators](/en/docs/mt4/terminal/analytics/tech_indicators) are grouped in it. Having chosen a name in the list, one can impose the indicator into the active chart. More details can be found in the ["Technical Indicators" section](/en/docs/mt4/terminal/analytics/tech_indicators).  
    The same actions can be performed by the ["Insert — Indicators" menu](/en/docs/mt4/terminal/overview/main_menu/main_menu_insert) command;
-   ![Periods](/en/docs/mt4/terminal/img/tb_charts_period.png "Periods") — sub-menu to manage the chart periods. When a period is selected in this sub-menu, the corresponding changes will appear in the active chart.  
    Switching among chart periods can be performed by the ["Charts — Timeframes" menu](/en/docs/mt4/terminal/overview/main_menu/main_menu_charts) command, the corresponding [chart context menu](/en/docs/mt4/terminal/chart_management/charts_control#context) command, or the corresponding buttons of the ["Timeframes" toolbar](/en/docs/mt4/terminal/overview/toolbars/toolbars_periodicity);
-   ![Template](/en/docs/mt4/terminal/img/tb_charts_template.png "Template") — call the sub-menu to manage templates. Any template can be imposed into the active chart from this sub-menu. Moreover, a new template can be stored based on the active chart or an existing template can be deleted. More details about working with templates can be found in the ["Templates and Profiles" section](/en/docs/mt4/terminal/chart_management/templates).  
    The same actions can be performed by the ["Charts — Template" menu](/en/docs/mt4/terminal/overview/main_menu/main_menu_charts) command or the corresponding [chart context menu](/en/docs/mt4/terminal/chart_management/charts_control#context) command.
