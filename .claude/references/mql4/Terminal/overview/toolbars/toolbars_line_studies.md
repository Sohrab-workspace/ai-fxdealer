# Line Studies

[Line Studies](/en/docs/mt4/terminal/analytics/objects_control/line_studies) and graphical objects (geometrical shapes, signs, and texts) to be imposed into the chart are collected in the "Line Studies" toolbar. Commands of this toolbar (excluding the "Crosshair" and "Cursor") are duplicated in the ["Insert" menu](/en/docs/mt4/terminal/overview/main_menu/main_menu_insert). Working with line studies is described in details in the [corresponding section](/en/docs/mt4/terminal/analytics/objects_control/line_studies). Other icons in this toolbar are intended for working with graphical objects and for changing the cursor operation mode:

-   ![Ellipse](/en/docs/mt4/terminal/img/tb_ls_ellips.png "Ellipse") — draw an ellipse. This shape helps to select various areas in the chart. More details about working with objects can be found in the [section of the same name](/en/docs/mt4/terminal/analytics/objects_control).  
    The ["Insert — Shapes — Ellipse" menu](/en/docs/mt4/terminal/overview/main_menu/main_menu_insert) command allows to draw this figure, as well;
-   ![Triangle](/en/docs/mt4/terminal/img/tb_ls_triangle.png "Triangle") — draw a triangle. This shape helps to select various areas in the chart. More details about working with objects can be found in the [section of the same name](/en/docs/mt4/terminal/analytics/objects_control).  
    The ["Insert — Shapes — Triangle" menu](/en/docs/mt4/terminal/overview/main_menu/main_menu_insert) command allows to draw this figure, as well;
-   ![Rectangle](/en/docs/mt4/terminal/img/tb_ls_rectangle.png "Rectangle") — draw a rectangle. This shape helps to select various areas in the chart. More details about working with objects can be found in the [section of the same name](/en/docs/mt4/terminal/analytics/objects_control).  
    The ["Insert — Shapes — Rectangle" menu](/en/docs/mt4/terminal/overview/main_menu/main_menu_insert) command allows to draw this figure, as well;
-   ![Text Label](/en/docs/mt4/terminal/img/tb_ls_text_label.png "Text Label") — add a text label. After this button has been pressed, one should select a point in the chart where the label should be placed. Unlike the text (described below), the text label is attached not to the chart, but to the chart window. When the chart is scrolled, the text label does not move.  
    The ["Insert — Text Label" menu](/en/docs/mt4/terminal/overview/main_menu/main_menu_insert) command will also add comment into the chart;
-   ![Text](/en/docs/mt4/terminal/img/tb_ls_text.png "Text") — add a comment into the chart. It will be anchored directly to the chart. When the chart is scrolled, the text will move together with it.  
    The ["Insert — Text" menu](/en/docs/mt4/terminal/overview/main_menu/main_menu_insert) command will also add comment into the chart;
-   ![Symbols](/en/docs/mt4/terminal/img/tb_ls_symbols.png "Symbols") — menu to manage graphical symbols. One can add graphical signs into the chart or delete all the imposed signs from it using this menu.  
    These objects can also be imposed into the chart by the ["Insert — Arrows" menu](/en/docs/mt4/terminal/overview/main_menu/main_menu_insert) command;
-   ![Crosshair](/en/docs/mt4/terminal/img/tb_ls_crosshair.png "Crosshair") — a crosshair. Cursor in this mode allows to highlight coordinates of the selected point in the chart or indicator window. This mode can only work within the active chart window. If, being in the "Crosshair" mode, one keeps the left mouse button pressed and moves the cursor, a line connecting these two points will be drawn. The distance between the points and the current price value will be highlighted to the right of the second point of the line. This will look as \[amount of bars between two points\] / \[amount of points between two points\] / \[the current price value in the second point\]. This tool is named electronic ruler.  
    Cursor can also be switched to the crosshair mode by pressing accelerating keys of Ctrl+F or the third mouse button;
-   ![Cursor](/en/docs/mt4/terminal/img/tb_ls_cursor.png "Cursor") — a standard cursor (selection mode). The cursor in this mode allows to select various objects in the chart.
