# Deleted Charts

Some charts become useless at some stages of work, and they need to be deleted. This can be done by the ["File — Close" menu](/en/docs/mt4/terminal/overview/main_menu/main_menu_file) command or by pressing of accelerating keys of Ctrl+F4. At that, the templates of deleted windows will be stored if "Save deleted charts to reopen" is enabled in the [client terminal settings window](/en/docs/mt4/terminal/setup/setup_charts). These templates can be used in future. So, if a chart has been deleted occasionally, it can be restored. To do so, one has to execute the ["File — Open Deleted" menu](/en/docs/mt4/terminal/overview/main_menu/main_menu_file) command and select a symbol from the list. After that, the chart window of the selected symbol will be fully restored with all its settings and objects imposed into it.
