# User Interface

Interface of the terminal includes the following components:

-   Common Window Heading  
    The program heading includes the current account number, application name, as well as the active chart window name and its period (timeframe).
-   Main Menu  
    All commands and functions that can be executed in the client terminal are collected in its main menu. It contains: ["File"](/en/docs/mt4/terminal/overview/main_menu/main_menu_file), ["View"](/en/docs/mt4/terminal/overview/main_menu/main_menu_view), ["Insert"](/en/docs/mt4/terminal/overview/main_menu/main_menu_insert), ["Charts"](/en/docs/mt4/terminal/overview/main_menu/main_menu_charts), ["Tools"](/en/docs/mt4/terminal/overview/main_menu/main_menu_tools), ["Window"](/en/docs/mt4/terminal/overview/main_menu/main_menu_window), ["Help"](/en/docs/mt4/terminal/overview/main_menu/main_menu_help).  
    More details can be found in the ["Main Menu" section](/en/docs/mt4/terminal/overview/main_menu).
-   Toolbars  
    Four toolbars are built into the terminal: ["Standard"](/en/docs/mt4/terminal/overview/toolbars/toolbars_standard), ["Charts"](/en/docs/mt4/terminal/overview/toolbars/toolbars_charts), ["Line Studies"](/en/docs/mt4/terminal/overview/toolbars/toolbars_line_studies), and ["Timeframes"](/en/docs/mt4/terminal/overview/toolbars/toolbars_periodicity). The toolbars contain duplicated commands and functions of the main menu. However, these toolbars are customizable, so one can collect only mostly used widgets in them.  
    More details can be found in the ["Toolbars" section](/en/docs/mt4/terminal/overview/toolbars).
-   Market Watch  
    This window contains the symbol list. Commands of this window allow to control trade positions and charts.  
    More details can be found in the ["Market Watch" section](/en/docs/mt4/terminal/overview/market_watch).
-   Depth of Market  
    The depth of market displays the current market for a trading symbol. This tool provides the ability of quick and easy order management.  
    More details can be found in the ["Depth of Market" section](/en/docs/mt4/terminal/overview/depth_of_market).
-   Data Window  
    This window shows information about prices, as well as indicators and expert advisors imposed. This data window does not allow to perform any actions.  
    More details can be found in the ["Data Window" section](/en/docs/mt4/terminal/overview/data_window).
-   Navigator  
    Open accounts, expert advisors, indicators, and scripts are grouped in the navigation window. The "Navigator" can help to control them promptly.  
    More details can be found in the ["Navigator" section](/en/docs/mt4/terminal/overview/navigator).
-   Terminal  
    Terminal is a multifunctional window intended for working with trade positions, news, account history, alerts, internal mail, and logs of the program and of expert advisors, as well. Moreover, the "Terminal" window allows to open and modify orders of different kinds and manage trade positions.  
    More details can be found in the ["Terminal" section](/en/docs/mt4/terminal/overview/terminal).
-   Tester  
    This window is intended for testing expert advisors. Besides testing and viewing of various reports, the expert parameters can be optimized in this window, as well.  
    More details can be found in the ["Tester" section](/en/docs/mt4/terminal/autotrading/tester).
-   Tick Charts  
    A tick chart is the basis of data analysis. Besides price changes, various analytical objects can be located in the chart: line studies, technical and custom indicators, texts, and geometrical figures.  
    More details can be found in the ["Working with Charts" section](/en/docs/mt4/terminal/chart_management/charts_setup).
-   Status Bar  
    For additional information to be visualized, the status bar is used in the terminal. The status bar contains indicator of the server connection, amount of incoming and outgoing traffic, names of active [template and profile](/en/docs/mt4/terminal/chart_management/templates), as well as command prompts and price data.
-   Fast Navigation Tools  
    Hot keys, acceleration keys, and fast navigation bar are intended for quick work with the terminal. Using of keys allows to execute various commands of the program promptly, and the fast navigation bar makes it possible to move the chart along its time base precisely, change its timeframe, and even the symbol itself.  
    More details can be found in the ["Fast Navigation Tools"](/en/docs/mt4/terminal/overview/fast_nav).
