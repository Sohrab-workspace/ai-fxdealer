# Shutdown

Normally, a script shuts down by itself. But it can be removed manually. At that, it must be deinitialized first and then rolled out from the memory. To remove a script from the chart, one has to execute the [chart contest menu](/en/docs/mt4/terminal/chart_management/charts_control) "Remove Script" command or attach another script to the chart. Besides, a script will be removed from the chart after symbol or timeframe has been changed.
