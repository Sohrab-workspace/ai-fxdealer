# Daily Reports

> **Server:** `88.218.200.140:443` | **Login:** `52`
> **Source:** [MetaQuotes MT4 Manager API — Reports](https://support.metaquotes.net/en/docs/mt4/api/manager_api/manager_api_report)

---

## Overview

Reports provide end-of-day summaries for accounts: balance history, P&L breakdown, trades count,
deposits/withdrawals, commissions, and swap. Useful for analytics and daily book reconciliation.

Two report types:
- **`DailyReport`** — end-of-day snapshot generated by the server each day per account
- **`ReportGroupRequest` + `TradeRecord[]`** — closed trades report for a group + date range

---

## Methods

| Method | Purpose |
|--------|---------|
| `ReportsRequest` | Get closed trade records for a group and date range |
| `DailyReportsRequest` | Get daily report records for a group and date range |

## Example Code

```python
import ctypes
import time

# ─── DailyReportsRequest ──────────────────────────────────────────────────────
class DailyGroupRequest(ctypes.Structure):
    _fields_ = [
        ("group",  ctypes.c_char * 16),
        ("from",   ctypes.c_int),    # start of day (time_t)
        ("to",     ctypes.c_int),    # end of day (time_t)
    ]

# Request: last 30 days for all real accounts
req = DailyGroupRequest()
req.group = b"real"
req.from_time = int(time.time()) - (30 * 86400)
req.to_time   = int(time.time())

total = ctypes.c_int(0)
rep_ptr = manager.DailyReportsRequest(ctypes.byref(req), ctypes.byref(total))

if rep_ptr:
    print(f"Daily reports: {total.value}")
    DailyReportArray = DailyReport * total.value
    reports = ctypes.cast(rep_ptr, ctypes.POINTER(DailyReportArray)).contents
    records = [struct_to_dict(r) for r in reports]
    manager.MemFree(rep_ptr)
    print(records[0])
```

## DailyReport Structure

```python
class DailyReport(ctypes.Structure):
    _fields_ = [
        ("login",            ctypes.c_int),
        ("group",            ctypes.c_char * 16),
        ("currency",         ctypes.c_char * 16),
        ("time",             ctypes.c_int),        # report date (time_t)
        ("value",            ctypes.c_double),     # account value at EOD
        ("profit",           ctypes.c_double),     # P&L for the day
        ("balance_prev",     ctypes.c_double),     # balance at start of day
        ("balance",          ctypes.c_double),     # balance at end of day
        ("equity",           ctypes.c_double),     # equity at end of day
        ("margin",           ctypes.c_double),     # margin at end of day
        ("margin_free",      ctypes.c_double),     # free margin at EOD
        ("deposit",          ctypes.c_double),     # total deposits on this day
        ("credit",           ctypes.c_double),     # credit changes on this day
        ("commission",       ctypes.c_double),     # total commission on this day
        ("storage",          ctypes.c_double),     # total swap on this day
        ("closed_profit",    ctypes.c_double),     # realized P&L (closed trades)
        ("floating_profit",  ctypes.c_double),     # floating P&L (open trades)
        ("correction",       ctypes.c_double),     # balance correction
        ("prev_month_balance", ctypes.c_double),   # balance at start of prev month
        ("reserved",         ctypes.c_int * 8),
    ]
```

## Notes

- The `time` field marks the **start of the trading day** (not end). Trading days are defined by `ConCommon::endhour` and `ConCommon::endminute`.
- `deposit` includes both cash deposits and credit additions for that day.
- `commission` is typically negative (charged to client), `storage` may be positive or negative depending on swap direction.
- For a complete P&L reconciliation: `profit = closed_profit + floating_profit - commission - storage + deposit + correction + credit`.
